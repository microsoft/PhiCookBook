## **Phi-4-mini-mm-i kasutamine koodi genereerimiseks**

Phi-4-mini jätkab Phi perekonna tugevaid kodeerimisvõimeid. Saate kasutada Prompti, et esitada kodeerimisega seotud küsimusi. Loomulikult, tänu tugevale põhjendusvõimele, on sellel veelgi paremad kodeerimisvõimed, näiteks projektide loomine vastavalt nõuetele. Näiteks projektide loomine vastavalt nõuetele, nagu:

### **Nõue**

Loo ostukorvi rakendus

- Loo REST API järgmiste meetoditega:
    - Hankige õllede nimekiri, kasutades lehe nihket ja limiiti.
    - Hankige õlle üksikasjad ID järgi.
    - Otsige õlut nime, kirjelduse, loosungi, toiduga sobivuse ja hinna järgi.
- Loo toodete nimekiri avalehel.
    - Loo otsinguriba toodete filtreerimiseks.
    - Liigu kirjelduse lehele, kui kasutaja klikib tootel.
- (Valikuline) Loo hinnafiltri liugur toodete filtreerimiseks.
- Loo ostukorv.
    - Lisa tooteid ostukorvi.
    - Eemalda tooteid ostukorvist.
    - Arvuta ostukorvis olevate toodete koguhind.

    model_path,
    trust_remote_code=True,
    torch_dtype='auto',
    _attn_implementation='flash_attention_2',
).cuda()

               ## Backend
                 
                   #### `models/user.py`
                   ```python

                   ```
                   .......
               

               ## Frontend
                 
                   #### `templates/index.html`
                   ```html

                   ```
                   ......."""

prompt = f'{user_prompt}Please create a project with Python and Flask according to the following requirements：\n{requirement}{note}{prompt_suffix}{assistant_prompt}'
