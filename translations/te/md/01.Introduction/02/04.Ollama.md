## Ollama లో Phi కుటుంబం


[Ollama](https://ollama.com) సరళమైన స్క్రిప్ట్‌ల ద్వారా ఎక్కువ మందికి ఓపెన్ సోర్స్ LLM లేదా SLM ను నేరుగా నియామకం చేయడానికి అనుమతిస్తారు, అలాగే స్థానిక Copilot అప్లికేషన్ దృశ్యాలకు సహాయపడటానికి APIలను కూడా నిర్మించగలదు.

## **1. సంస్థాపన**

Ollama Windows, macOS, మరియు Linux పైన నడవడానికి మద్దతు ఇస్తుంది. మీరు Ollama ను ఈ లింక్ ద్వారా ఇన్‌స్టాల్ చేయవచ్చు ([https://ollama.com/download](https://ollama.com/download)). విజయవంతంగా సంస్థాపన అయిన తర్వాత, మీరు టెర్మినల్ విండో ద్వారా Ollama స్క్రిప్ట్ ఉపయోగించి నేరుగా Phi-3 ను కాల్ చేయవచ్చు. మీరు Ollama లోని [available libaries in Ollama](https://ollama.com/library) ను చూడవచ్చు. మీరు ఈ రిపోజిటరీని Codespace లో తెరిస్తే, దానిలో ఇప్పటికే Ollama ఇన్‌స్టాల్ చేయబడింది.

```bash

ollama run phi4

```

> [!NOTE]
> మోడల్‌ను మీరు మొదటిసారిగా 실행ించినప్పుడు ముందుగా డౌన్‌లోడ్ చేయబడుతుంది. ఖచ్చితంగా, మీరు ఇప్పటికే డౌన్‌లోడ్ చేసిన Phi-4 మోడల్‌ను నేరుగా నిర్దేశించవచ్చు. ఆ ఆజ్ఞను నడపడానికి మేము WSL ను ఉదాహరణగా తీసుకుంటాము. మోడల్ విజయవంతంగా డౌన్‌లోడ్ అయిన తర్వాత, మీరు నేరుగా టెర్మినల్‌లో పరస్పర చర్య చేయవచ్చు.

## **2. Ollama నుంచి phi-4 API ని కాల్ చేయడం**

Ollama ద్వారా జనరేట్ చేసిన Phi-4 API ని కాల్ చేయాలనుకుంటే, Ollama సర్వర్‌ను ప్రారంభించడానికి మీరు టెర్మినల్‌లో ఈ ఆజ్ఞను ఉపయోగించవచ్చు.

```bash

ollama serve

```

> [!NOTE]
> MacOS లేదా Linux నడుపుతున్నట్లయితే, క్రింది తప్పు సందేశాన్ని మీరు ఎదుర్కోవచ్చు **"Error: listen tcp 127.0.0.1:11434: bind: address already in use"**. మీరు ఆ ఆజ్ఞను రన్ చేసినప్పుడు ఈ పొరపాటు కనిపించవచ్చు. ఇది సాధారణంగా సర్వర్ ఇప్పటికే నడుస్తున్నదని సూచిస్తుండడంతో, మీరు ఆ పొరపాటును నిర్లక్షించవచ్చు, లేదా Ollama ను ఆపి మళ్లీ ప్రారంభించవచ్చు:

**macOS**

```bash

brew services restart ollama

```

**Linux**

```bash

sudo systemctl stop ollama

```

Ollama రెండు API లను మద్దతు చేస్తుంది: generate మరియు chat. మీరు మీ అవసరాల ప్రకారం Ollama అందించిన మోడల్ API ను కాల్ చేయవచ్చు, స్థానిక సేవకు పోర్ట్ 11434 పైన నడుస్తున్నదానికి అభ్యర్థనలు పంపడం ద్వారా.

**Chat**

```bash

curl http://127.0.0.1:11434/api/chat -d '{
  "model": "phi3",
  "messages": [
    {
      "role": "system",
      "content": "Your are a python developer."
    },
    {
      "role": "user",
      "content": "Help me generate a bubble algorithm"
    }
  ],
  "stream": false
  
}'
```

## అదనపు వనరులు

Ollama లో అందుబాటులో ఉన్న మోడల్స్ జాబితాను [వారి లైబ్రరీ](https://ollama.com/library)లో తనిఖీ చేయండి.

Ollama సర్వర్ నుంచి మీ మోడల్‌ను ఈ ఆజ్ఞను ఉపయోగించి పుల్ చేయండి

```bash
ollama pull phi4
```

ఈ ఆజ్ఞను ఉపయోగించి మోడల్‌ను నడపండి

```bash
ollama run phi4
```

***Note:*** మరింత తెలుసుకోడానికి ఈ లింక్‌ను చూడండి [https://github.com/ollama/ollama/blob/main/docs/api.md](https://github.com/ollama/ollama/blob/main/docs/api.md)

## Python నుండి Ollama ను కాల్ చేయడం

మీరు పైగా ఉపయోగించిన స్థానిక సర్వర్ ఎండ్పాయింట్లకు అభ్యర్థనలు చేయడానికి `requests` లేదా `urllib3` ఉపయోగించవచ్చు. అయినప్పటికీ, Pythonలో Ollama ను ఉపయోగించే ఒక ప్రజాదరణ పొందిన విధానం [openai](https://pypi.org/project/openai/) SDK ద్వారా చేయడం, ఎందుకంటే Ollama OpenAI-అనుకూల సర్వర్ ఎండ్పాయింట్లను కూడా అందిస్తుంది.

ఇది phi3-mini కోసం ఒక ఉదాహరణ:

```python
import openai

client = openai.OpenAI(
    base_url="http://localhost:11434/v1",
    api_key="nokeyneeded",
)

response = client.chat.completions.create(
    model="phi4",
    temperature=0.7,
    n=1,
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Write a haiku about a hungry cat"},
    ],
)

print("Response:")
print(response.choices[0].message.content)
```

## JavaScript నుండి Ollama ను కాల్ చేయడం

```javascript
// Phi-4 తో ఫైల్‌ను సారాంశం చేయడానికి ఉదాహరణ
script({
    model: "ollama:phi4",
    title: "Summarize with Phi-4",
    system: ["system"],
})

// సారాంశం యొక్క ఉదాహరణ
const file = def("FILE", env.files)
$`Summarize ${file} in a single paragraph.`
```

## C# నుండి Ollama ను కాల్ చేయడం

కొత్త C# Console అప్లికేషన్‌ను సృష్టించి క్రింది NuGet ప్యాకేజీని జోడించండి:

```bash
dotnet add package Microsoft.SemanticKernel --version 1.34.0
```

తర్వాత `Program.cs` ఫైల్‌లో ఈ కోడ్‌ను మార్చండి

```csharp
using Microsoft.SemanticKernel;
using Microsoft.SemanticKernel.ChatCompletion;

// add chat completion service using the local ollama server endpoint
#pragma warning disable SKEXP0001, SKEXP0003, SKEXP0010, SKEXP0011, SKEXP0050, SKEXP0052
builder.AddOpenAIChatCompletion(
    modelId: "phi4",
    endpoint: new Uri("http://localhost:11434/"),
    apiKey: "non required");

// invoke a simple prompt to the chat service
string prompt = "Write a joke about kittens";
var response = await kernel.InvokePromptAsync(prompt);
Console.WriteLine(response.GetValue<string>());
```

యాప్‌ను ఈ ఆజ్ఞతో నడపండి:

```bash
dotnet run
```
