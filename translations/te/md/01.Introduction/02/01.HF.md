# **Hugging Faceలో Phi ఫ్యామిలీ ఉపయోగించడం**

[Hugging Face](https://huggingface.co/) చాలా సమృద్ధి డేటా మరియు ఓపెన్ సోర్స్ మోడల్ వనరులతో కూడిన బాగా ప్రజాదరణ పొందిన AI కమ్యూనిటీ. Microsoft, Meta, Mistral, Apple, Google వంటి వివిధ సంస్థలు Hugging Face ద్వారా ఓపెన్ సోర్సు LLM మరియు SLM విడుదల చేస్తుంటాయి.

Microsoft Phi ఫ్యామిలీ Hugging Faceలో విడుదల చేయబడింది. డెవలపర్‌లు సందర్భాలు మరియు వ్యాపార అవసరాల ఆధారంగా సంబంధిత Phi ఫ్యామిలీ మోడల్‌ను డౌన్లోడ్ చేసుకోవచ్చు。Hugging Faceపై Phi Pytorch మోడళ్లను ప్రమాణంగా అమర్చడమే కాకుండా, మేము క్వాంటైజ్డ్ మోడళ్లను కూడా విడుదల చేశాం — GGUF మరియు ONNX ఫార్మాట్‌లను ఉపయోగించడంతో ఎండ్-యూజర్లు ఎంపిక పొందగలుగుతారు.

## **Hugging faceలో మోడళ్లు డౌన్లోడ్ చేయడం**

మీరు ఈ లింక్ ద్వారా Phi ఫ్యామిలీ మోడల్‌ను డౌన్లోడ్ చేయవచ్చు

[Hugging Faceలో Microsoft మోడల్స్](https://huggingface.co/microsoft)

-  **Phi-1 / 1.5** https://huggingface.co/collections/microsoft/phi-1-6626e29134744e94e222d572

-  **Phi-3 / 3.5** https://huggingface.co/collections/microsoft/phi-3-6626e15e9585a200d2d761e3

-  **Phi-4** https://huggingface.co/collections/microsoft/phi-4-677e9380e514feb5577a40e4

- **Phi-4-reasoning** https://huggingface.co/microsoft/Phi-4-reasoning

- **Phi-4-reasoning Plus** https://huggingface.co/microsoft/Phi-4-reasoning-plus 

- **Phi-4-mini-reasoning** https://huggingface.co/microsoft/Phi-4-mini-reasoning

మీరు మోడల్‌ను విభిన్న మార్గాలలో డౌన్లోడ్ చేయవచ్చు, ఉదాహరణకు ***Hugging face CLI SDK***ను ఇన్స్టాల్ చేయడం లేదా ***git clone*** ఉపయోగించడం.

### **Phi ఫ్యామిలీ మోడల్ డౌన్లోడ్ చేయడానికి Hugging face CLI ఉపయోగించడం**

- Hugging face CLIని ఇన్స్టాల్ చేయండి

```bash

pip install -U "huggingface_hub[cli]"

```

- huggingface-cli ఉపయోగించి లాగిన్ చేయడం

మీ [User Access Token](https://huggingface.co/docs/hub/security-tokens)ని మీ [Settings page](https://huggingface.co/settings/tokens) నుంచి తీసుకుని Hugging Faceలో లాగిన్ అవ్వండి


```bash

huggingface-cli login --token $HF_TOKEN --add-to-git-credential

```

- డౌన్లోడ్ చేయండి 


మీరు మోడల్‌ను డౌన్లోడ్ చేసి cacheలో సేవ్ చేయవచ్చు 

```bash

huggingface-cli download microsoft/phi-4

```

మీరు ప్రత్యేకంగా కోరుకున్న చోట్లో నిల్వ స్థలాన్ని సెట్ చేయవచ్చు


```bash

huggingface-cli download microsoft/phi-4 --local-dir $YOUR_PATH

```


### **Phi ఫ్యామిలీ మోడల్ డౌన్లోడ్ చేసేందుకు git clone ఉపయోగించడం**

మీరు మోడల్‌ను డౌన్లోడ్ చేయడానికి మీరు ***git clone*** కూడా ఉపయోగించవచ్చు

```bash

git lfs install

git clone https://huggingface.co/microsoft/phi-4

```

## **నమూనాలు - Microsoft Phi-4 ఇన్ఫరెన్స్**

- **transformers లైబ్రరీని ఇన్స్టాల్ చేయడం**

```bash

pip install transformers -U

```

- **ఈ కోడ్‌ను VSCodeలో నడపడం**

```python

import transformers

pipeline = transformers.pipeline(
    "text-generation",
    model="microsoft/phi-4",
    model_kwargs={"torch_dtype": "auto"},
    device_map="auto",
)

messages = [
    {"role": "user", "content": "I have $20,000 in my savings account, where I receive a 4% profit per year and payments twice a year. Can you please tell me how long it will take for me to become a millionaire? Also, can you please explain the math step by step as if you were explaining it to an uneducated person?"},
]

outputs = pipeline(messages, max_new_tokens=2048)
print(outputs[0]["generated_text"][-1])

```
