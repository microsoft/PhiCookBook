# Phi-3.5-Instruct WebGPU RAG చాట్బాట్

## WebGPU మరియు RAG ప్యాటర్న్ చూపించడానికి డెమో

Phi-3.5 Onnx Hosted మోడల్‌తో RAG ప్యాటర్న్ Retrieval-Augmented Generation విధానాన్ని ఉపయోగించుకుంటుంది, Phi-3.5 మోడల్స్ శక్తిని ONNX హోస్టింగ్‌తో కలిపి సమర్థవంతమైన AI డిప్లాయ్‌మెంట్‌లను సాధిస్తుంది. ఈ ప్యాటర్న్ డొమెయిన్-నిర్దిష్ట పనుల కోసం మోడళ్లను ఫైన్-ట్యూన్ చేయడంలో కీలకంగా ఉంటుంది, నాణ్యత, ఖర్చు-ప్రత్యక్షత మరియు దీర్ఘ-కాంటెక్స్ట్ అవగాహనను కలపడం జరుగుతుంది. ఇది Azure AI యొక్క సూట్ భాగంగా ఉంటుంది, వివిధ పరిశ్రమల అనుకూలీకరణ అవసరాలకు సరిపోయే, కనుగొనడం, ప్రయత్నించడం మరియు ఉపయోగించడం సులభమైన విస్తృత మోడల్ ఎంపికను అందిస్తుంది. 

## WebGPU అంటే ఏమిటి 
WebGPU అనేది ఆధునిక వెబ్ గ్రాఫిక్స్ API, ఇది వెబ్ బ్రౌజర్‌ల నుండి నేరుగా డివైస్ గ్రాఫిక్స్ ప్రాసెసింగ్ యూనిట్ (GPU) కు సమర్థవంతమైన ప్రాప్యతను ఇవ్వడానికి రూపొందించబడింది. ఇది WebGL కు ప్రతిసారి వస్తున్న వారసుడిగా ఉద్దేశించబడింది, మరియు కొన్ని కీలక మెరుగుదలలను అందిస్తుంది:

1. **ఆధునిక GPUలతో అనుకూలత**: WebGPU సమకాలీన GPU నిర్మాణాలతో సజావుగా పని చేయడానికి రూపొందించబడింది, Vulkan, Metal, మరియు Direct3D 12 వంటి సిస్టమ్ APIలను ఉపయోగిస్తుంది.
2. **పెరిగిన పనితీరు**: ఇది జనరల్-పర్పస్ GPU గణనలను మరియు వేగవంతమైన ఆపరేషన్లను మద్దతిస్తుంది, గ్రాఫిక్స్ రెండరింగ్ మరియు మెషీన్ లెర్నింగ్ పనుల కోసం అనుకూలంగా ఉంటుంది.
3. **అత్యాధునిక లక్షణాలు**: WebGPU మరింత అధునాతన GPU సామర్థ్యాలకు ప్రాప్యతను అందిస్తుంది, ఆలోచనాత్మక మరియు డైనమిక్ గ్రాఫిక్స్ మరియు గణన పనుల్ని సాధ్యమైనలా చేస్తుంది.
4. **JavaScript పని భారాన్ని తగ్గింపు**: మరిన్ని పనులను GPU కు ఆప్లోడ్ చేయడం ద్వారా, WebGPU JavaScript పైని పని భారాన్ని గణనీయంగా తగ్గిస్తుంది, అందువలన మెరుగైన పనితీరు మరియు స్మూత్ అనుభవాలు వస్తాయి.

WebGPU ప్రస్తుతం Google Chrome వంటి బ్రౌజర్లలో మద్దతు పొందుతోంది, మరియు ఇతర ప్లాట్‌ఫారమ్‌లకు మద్దతును విస్తరించడానికి పని కొనసాగుతోంది.

### 03.WebGPU
Required Environment:

**మద్దతు ఉన్న బ్రౌజర్లు:** 
- Google Chrome 113+
- Microsoft Edge 113+
- Safari 18 (macOS 15)
- Firefox Nightly.

### WebGPU ను ఎనేబుల్ చేయడం:

- Chrome/ Microsoft Edge లో 

ఈ `chrome://flags/#enable-unsafe-webgpu` ఫ్లాగ్‌ను ఎనేబుల్ చేయండి.

#### మీ బ్రౌజర్ తెరవండి:
Google Chrome లేదా Microsoft Edge ప్రారంభించండి.

#### ఫ్లాగ్ పేజీకి ప్రవేశించండి:
అడ్రస్ బార్‌లో `chrome://flags` టైప్ చేసి Enter నొక్కండి.

#### ఫ్లాగ్ కోసం శోధించండి:
పేజీ మెడలోని శోధన బాక్స్‌లో 'enable-unsafe-webgpu' టైప్ చేయండి

#### ఫ్లాగ్‌ను ఎనేబుల్ చేయండి:
ఫలితాల జాబితాలో #enable-unsafe-webgpu ఫ్లాగ్‌ను కనుగొనండి.

దానికి పక్కన ఉన్న డ్రాప్డౌన్ మెనును క్లిక్ చేసి Enabled ఎంచుకోండి.

#### మీ బ్రౌజర్‌ని రీస్టార్ట్ చేయండి:

ఫ్లాగ్‌ను ఎనేబుల్ చేసిన తర్వాత, మార్పులు ప్రభావితం కావడానికి మీ బ్రౌజర్‌ను రీస్టార్ట్ చేయాలి. పేజీ దిగువకు కనిపించే Relaunch బటన్‌పై క్లిక్ చేయండి.

- Linux కోసం, బ్రౌజర్‌ను `--enable-features=Vulkan` తో ప్రారంభించండి.
- Safari 18 (macOS 15)లో WebGPU డిఫాల్ట్‌గా సక్రియంగా ఉంటుంది.
- Firefox Nightly లో, అడ్రస్ బార్‌లో about:config టైప్ చేసి `set dom.webgpu.enabled to true`.

### Microsoft Edge కోసం GPU సెటప్ చేయడం 

Windows లో Microsoft Edge కోసం హై-పర్ఫార్మెన్స్ GPU సెటప్ చేయడానికి ఈ క్రింది దశలు ఉన్నాయి:

- **Open Settings:** Start menu ను క్లిక్ చేసి Settings ఎంచుకోండి.
- **System Settings:** System కి వెళ్లి తరువాత Display ఎంచుకోండి.
- **Graphics Settings:** స్క్రోల్ చేయండి మరియు Graphics settings పై క్లిక్ చేయండి.
- **Choose App:** “Choose an app to set preference” కింద, Desktop app ఎంచుకుని తరువాత Browse క్లిక్ చేయండి.
- **Select Edge:** Edge ఇన్స్టాలేషన్ ఫోల్డర్‌కు నావిగేట్ చేయండి (సాధారణంగా `C:\Program Files (x86)\Microsoft\Edge\Application`) మరియు `msedge.exe` ను ఎంచుకోండి.
- **Set Preference:** Options క్లిక్ చేసి, High performance ఎంచుకుని, తరువాత Save క్లిక్ చేయండి.
ఈ చర్య Microsoft Edge మీ హై-పర్ఫార్మెన్స్ GPU ను ఉపయోగించి మంచి పనితీరు అందించడానికి నిర్ధారిస్తుంది. 
- **Restart** మీ మెషిన్‌ను రీస్టార్ట్ చేయండి ఈ సెట్టింగులు ప్రభావితం కావడానికి
