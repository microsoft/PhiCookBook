# ఇంటరాక్టివ్ ఫై 3 మినీ 4K ఇన్‌స్ట్రక్ట్ చాట్‌బాట్ విత్ విస్పర్

## అవలోకనం

ఇంటరాక్టివ్ ఫై 3 మినీ 4K ఇన్‌స్ట్రక్ట్ చాట్‌బాట్ ఒక సాధనం, ఇది వినియోగదారులు మైక్రోసాఫ్ట్ ఫై 3 మినీ 4K ఇన్‌స్ట్రక్ట్ డెమోతో టెక్స్ట్ లేదా ఆడియో ఇన్‌పుట్ ద్వారా ఇంటరాక్ట్ అయ్యేందుకు అనుమతిస్తుంది. ఈ చాట్‌బాట్ అనువాదం, వాతావరణం నవీకరణలు, మరియు సాధారణ సమాచార సేకరణ వంటి వివిధ పనుల కోసం ఉపయోగించబడుతుంది.

### ప్రారంభించటం

ఈ చాట్‌బాట్ ఉపయోగించడానికి, ఈ సూచనలను అనుసరించండి:

1. కొత్త [E2E_Phi-3-mini-4k-instruct-Whispser_Demo.ipynb](https://github.com/microsoft/Phi-3CookBook/blob/main/code/06.E2E/E2E_Phi-3-mini-4k-instruct-Whispser_Demo.ipynb) ను ఓపెన్ చేయండి  
2. నోట్బుక్ ప్రధాన విండోలో, మీరు ఒక చాట్‌బాక్స్ ఇంటర్‌ఫేస్‌ను, టెక్స్ట్ ఇన్‌పుట్ బాక్స్ మరియు "Send" బటన్ తో చూడవచ్చు.  
3. టెక్స్ట్-ఆధారిత చాట్‌బాట్ ఉపయోగించాలంటే, టెక్స్ట్ ఇన్‌పుట్ బాక్స్‌లో మీ సందేశం టైప్ చేసి "Send" బటన్ క్లిక్ చేయండి. చాట్‌బాట్ నోట్బుక్‌లోనుండి ప్రత్యక్షంగా ప్లే చేయగల ఆడియో ఫైల్‌తో స్పందిస్తుంది.

**గమనిక**: ఈ సాధనాన్ని ఉపయోగించడానికి GPU మరియు మైక్రోసాఫ్ట్ ఫై-3 మరియు OpenAI విస్పర్ మోడల్స్ పై యాక్సెస్ అవసరం, ఇది స్పీచ్ గుర్తింపు మరియు అనువాదానికి ఉపయోగించబడుతుంది.

### GPU అవసరాలు

ఈ డెమోను నడపడానికి 12Gb GPU మెమరీ అవసరం.

**Microsoft-Phi-3-Mini-4K instruct** డెమోను GPU పై నడిపేటప్పుడు మెమరీ అవసరాలు అనేక అంశాలపై منحصر అవుతాయి, ఉదాహరణకు ఇన్‌పుట్ డేటా పరిమాణం (ఆడియో లేదా టెక్స్ట్), అనువాదానికి ఉపయోగించే భాష, మోడల్ స్పీడ్ మరియు GPUలో అందుబాటులో ఉన్న మెమరీ.

సాధారణంగా, విస్పర్ మోడల్ GPUలపై నడవడానికి రూపొందించబడింది. విస్పర్ మోడల్ నడిపించేందుకు కనీసం 8 GB GPU మెమరీని సిఫారసు చేస్తారు, కానీ అవసరమైతే పెద్ద మెమరీ వ్యవహరించగలదు.

పరిమితి ఎక్కువ డేటా లేదా ఎక్కువ მოთხოვనలతో మోడల్ నడపడం GPU మెమరీ ఎక్కువ అవసరం అవ్వచ్చు లేదా పనితీరులో సమస్యలు రావచ్చు. మీ అవసరాలకు సరిపోయే సెట్టింగును కనుగొనేందుకు వేరే వేరే కాన్ఫిగరేషన్‌లతో పరీక్షించాలి మరియు మెమరీ వినియోగాన్ని పర్యవేక్షించాలి.

## ఇంటరాక్టివ్ ఫై 3 మినీ 4K ఇన్‌స్ట్రక్ట్ చాట్‌బాట్ విత్ విస్పర్ కోసం E2E నమూనా

[Interactive Phi 3 Mini 4K Instruct Chatbot with Whisper](https://github.com/microsoft/Phi-3CookBook/blob/main/code/06.E2E/E2E_Phi-3-mini-4k-instruct-Whispser_Demo.ipynb) పేరుగల జుపైటర్ నోట్‌బుక్ Microsoft Phi 3 Mini 4K ఇన్‌స్ట్రక్ట్ డెమోను ఆడియో లేదా వ్రాయబడిన టెక్స్ట్ ఇన్‌పుట్ నుండి టెక్స్ట్ ఉత్పత్తి చేయడం ఎలా అనేదానిని ప్రదర్శిస్తుంది. ఈ నోట్‌బుక్ కొన్ని ఫంక్షన్లను నిర్వచిస్తుంది:

1. `tts_file_name(text)`: ఈ ఫంక్షన్ ఇన్‌పుట్ టెక్స్ట్ ఆధారంగా ఉత్పత్తి చేయబడే ఆడియో ఫైల్ సేవ్ చేయడానికి ఫైల్ పేరు ఉత్పత్తి చేస్తుంది.
2. `edge_free_tts(chunks_list,speed,voice_name,save_path)`: ఈ ఫంక్షన్ ఇన్‌పుట్ టెక్స్ట్ భాగాల జాబితా నుంచి Edge TTS API ద్వారా ఆడియో ఫైల్ ను ఉత్పత్తి చేస్తుంది. ఇన్‌పుట్ పరామితులు భాగాల జాబితా, స్పీచ్ రేట్, వాయిస్ పేరు మరియు ఆడియో సేవ్ చేయడానికి అవుట్‌పుట్ మార్గం.
3. `talk(input_text)`: ఈ ఫంక్షన్ Edge TTS API ఉపయోగించి ఆడియో ఫైల్ ను ఉత్పత్తి చేసి దాన్ని /content/audio డైరెక్టరీలో రాండమ్ ఫైల్ పేరుతో సేవ్ చేస్తుంది. ఇన్‌పుట్ పారామీటర్ స్పీచ్‌కి మార్చాల్సిన టెక్స్ట్.
4. `run_text_prompt(message, chat_history)`: ఈ ఫంక్షన్ Microsoft Phi 3 Mini 4K ఇన్‌స్ట్రక్ట్ డెమోను ఉపయోగించి సందేశ ఇన్‌పుట్ నుంచి ఆడియో ఫైల్ ఉత్పత్తి చేసి దాన్ని చాట్ చరిత్రలో జోడిస్తుంది.
5. `run_audio_prompt(audio, chat_history)`: ఈ ఫంక్షన్ ఆడియో ఫైల్‌ను విస్పర్ మోడల్ API ఉపయోగించి టెక్స్ట్‌గా మార్చి `run_text_prompt()` ఫంక్షన్‌కు పంపుతుంది.
6. కోడ్ ఒక గ్రాడియో యాప్‌ను ప్రారంభించి Phi 3 Mini 4K ఇన్‌స్ట్రక్ట్ డెమోతో సందేశాలు టైప్ చేయడం లేదా ఆడియో ఫైళ్ళను అప్లోడ్ చేయడం ద్వారా వినియోగదారులు ఇంటరాక్ట్ అయ్యేందుకు అనుమతిస్తుంది. అవుట్‌పుట్ యాప్ లో టెక్స్ట్ సందేశంగా ప్రదర్శించబడుతుంది.

## సమస్య పరిష్కారం

Cuda GPU డ్రైవర్‌ల ఇన్‌స్టాలేషన్

1. మీ లినక్స్ అప్లికేషన్‌లు తాజా స్థాయిలో ఉన్నాయని నిర్ధారించుకోండి

    ```bash
    sudo apt update
    ```
  
1. Cuda డ్రైవర్లను ఇన్‌స్టాల్ చేయండి

    ```bash
    sudo apt install nvidia-cuda-toolkit
    ```
  
1. Cuda డ్రైవర్ స్థానం రిజిస్టర్ చేయండి

    ```bash
    echo /usr/lib64-nvidia/ >/etc/ld.so.conf.d/libcuda.conf; ldconfig
    ```
  
1. Nvidia GPU మెమరీ పరిమాణం తనిఖీ చేయండి (12GB GPU మెమరీ అవసరం)

    ```bash
    nvidia-smi
    ```
  
1. ఖాళీ కాష్: మీరు PyTorch ఉపయోగిస్తే, ఇతర GPU అప్లికేషన్లు ఉపయోగించడానికి torch.cuda.empty_cache() ని పిలిచి అన్ని ఉపయోగించని క్యాష్డ్ మెమరీ విడదీయవచ్చు

    ```python
    torch.cuda.empty_cache() 
    ```
  
1. Nvidia Cuda తనిఖీ చేయండి

    ```bash
    nvcc --version
    ```
  
1. Hugging Face టోకెన్ క్రియేట్ చేయడానికి క్రింది దశలు చేయండి:

    - [Hugging Face టోకెన్ సెట్టింగ్ పేజీ](https://huggingface.co/settings/tokens?WT.mc_id=aiml-137032-kinfeylo)కు వెళ్లండి  
    - **New token** ని ఎంచుకోండి  
    - మీరు ఉపయోగించదలచుకున్న ప్రాజెక్ట్ **Name**ని నమోదు చేయండి  
    - **Type**గా **Write** ఎంచుకోండి

> [!NOTE]
>
> మీరు ఈ క్రింది లోపం కలిగితే:
>
> ```bash
> /sbin/ldconfig.real: Can't create temporary cache file /etc/ld.so.cache~: Permission denied 
> ```
>
> దీన్ని పరిష్కరించడానికి మీ టెర్మినల్లో క్రింది ఆజ్ఞను టైప్ చేయండి.
>
> ```bash
> sudo ldconfig
> ```
