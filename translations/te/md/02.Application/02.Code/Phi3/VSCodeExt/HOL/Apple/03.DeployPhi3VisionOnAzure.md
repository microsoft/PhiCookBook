# **ల్యాబ్ 3 - Azure Machine Learning Serviceలో Phi-3-vision‌ను డిప్లాయ్ చేయడం**

మేము NPUని ఉపయోగించి స్థానిక కోడ్ యొక్క ప్రొడక్షన్ డిప్లాయ్‌మెంట్‌ను పూర్తి చేయడానికి ఉపయోగిస్తాము, మరియు దాని ద్వారా PHI-3-VISIONని పరిచయం చేసి చిత్రాల ద్వారా కోడ్‌ను ఉత్పత్తి చేయగల సామర్ధ్యాన్ని పొందాలనుకుంటున్నాం.

ఈ పరిచయంలో, Azure Machine Learning Serviceలో Model As Service గా Phi-3 Vision సర్వీస్‌ను త్వరగా నిర్మించవచ్చు.

***Note***： Phi-3 Vision ధ్రుతంగా కంటెంట్ ఉత్పత్తి చేయడానికి గణన సామర్థ్యాన్ని కోరుకుంటుంది. దీన్ని సాధించడానికి మేము క్లౌడ్ కంప్యూటింగ్ శక్తిని అవసరం పడతాం。


### **1. Azure Machine Learning Service సృష్టించడం**

మాకు Azure Portalలో ఒక Azure Machine Learning Service సృష్టించాల్సి ఉంటుంది. ఇది ఎలా చేయాలో తెలుసుకోవాలనుకుంటే, దయచేసి ఈ లింక్‌ను సందర్శించండి [https://learn.microsoft.com/azure/machine-learning/quickstart-create-resources?view=azureml-api-2](https://learn.microsoft.com/azure/machine-learning/quickstart-create-resources?view=azureml-api-2)

***గమనిక***

1. పంపాల్సిన పరామితుల్లో Authorization, azureml-model-deployment, మరియు Content-Type ఉండాలి. వాటిని పొందడానికి మీరు డిప్లాయ్‌మెంట్ సమాచారం తనిఖీ చేయాలి.

2. పరామితులను పంపడానికి, Phi-3-Visionకి ఒక చిత్రం లింక్ పంపాల్సి ఉంటుంది. పరామితులను ఎలా పంపాలో చూడటానికి GPT-4-Vision పద్ధతిని సూచించండి, ఉదాహరణకి

```json

{
  "input_data":{
    "input_string":[
      {
        "role":"user",
        "content":[ 
          {
            "type": "text",
            "text": "You are a Python coding assistant.Please create Python code for image "
          },
          {
              "type": "image_url",
              "image_url": {
                "url": "https://ajaytech.co/wp-content/uploads/2019/09/index.png"
              }
          }
        ]
      }
    ],
    "parameters":{
          "temperature": 0.6,
          "top_p": 0.9,
          "do_sample": false,
          "max_new_tokens": 2048
    }
  }
}

```

3. Post పద్ధతిని ఉపయోగించి **/score** ని కాల్ చేయండి

**అభినందనలు**！మీరు వేగవంతమైన PHI-3-VISION డిప్లాయ్మెంట్‌ను పూర్తి చేసి చిత్రాలను ఉపయోగించి కోడ్‌ను ఎలా ఉత్పత్తి చేయాలో ప్రయత్నించారు. తరువాత, మేము NPUలు మరియు క్లౌడ్లతో కలిపి అప్లికేషన్లను నిర్మించవచ్చు
