# **Azure Machine Learning సేవ పరిచయం**

[Azure Machine Learning](https://ml.azure.com?WT.mc_id=aiml-138114-kinfeylo) ఒక క్లౌడ్ సేవ, యంత్ర అభ్యాసం (ML) ప్రాజెక్టు జీవితచక్రాన్ని వేగవంతం చేయడానికి మరియు నిర్వహించడానికి.

ML నిపుణులు, డేటా శాస్త్రజ్ఞులు, మరియు ఇంజనీర్లు తమ దైనందిన పని ప్రవాహాల్లో దీన్ని ఉపయోగించవచ్చు:

- మోడళ్లను శిక్షణచేయడం మరియు డిప్లాయ్ చేయడం. మెషీన్ లెర్నింగ్ ఆపరేషన్లను (MLOps) నిర్వహించడం.
- మీరు Azure Machine Learningలో ఒక మోడల్ సృష్టించవచ్చు లేదా PyTorch, TensorFlow, లేదా scikit-learn వంటి ఓపెన్-సోర్స్ ప్లాట్‌ఫారమ్‌లో నిర్మించబడిన మోడల్‌ను ఉపయోగించవచ్చు.
- MLOps సాధనాలు మోడళ్లను మానిటర్ చేయడంలో, పునఃశిక్షణ ఇవ్వడంలో మరియు తిరిగి డిప్లాయ్ చేయడంలో సహాయపడతాయి.

## Azure Machine Learning ఎవరుకోసం?

**డేటా శాస్త్రజ్ఞులు మరియు ML ఇంజనీర్లు**

వారు తమ దైనందిన పని ప్రవాహాలను వేగవంతం చేయడానికి మరియు ఆటోమేట్ చేసుకోవడానికి సాధనాలను ఉపయోగించవచ్చు.
Azure ML సమానత్వం, వివరణాత్మకత, ట్రాకింగ్, మరియు ఆడిటబిలిటీకి సంబంధించిన ఫీచర్లను అందిస్తుంది.
Application Developers:
వారు మోడళ్లను అప్లికేషన్ల లేదా సేవల్లో సులభంగా ఇంటిగ్రేట్ చేయవచ్చు.

**Platform Developers**

వారికి స్థిరమైన Azure Resource Manager APIs ద్వారా బలమైన సాధనాల సముదాయం అందుబాటులో ఉంటుంది.
ఈ సాధనాలు అభివృద్ధి చెందిన ML టూలింగ్ నిర్మించడానికి సహాయపడతాయి.

**Enterprises**

Microsoft Azure క్లౌడ్లో పనిచేస్తున్నప్పుడు, ఎంటర్‌ప్రైజ్‌లు పరిచయమైన భద్రత మరియు పాత్ర ఆధారిత యాక్సెస్ నియంత్రణ ప్రయోజనాలను పొందుతాయి.
సంరక్షిత డేటా మరియు నిర్దిష్ట ఆపరేషన్లకు యాక్సెస్ ను నియంత్రించడానికి ప్రాజెక్టులను ఏర్పాటు చేయవచ్చు.

## జట్టులో ప్రతి ఒక్కరి ఉత్పాదకత
ML ప్రాజెక్టులు తరచుగా నిర్మించడానికి మరియు నిర్వహించడానికి విభిన్న నైపుణ్యాలున్న ఒక జట్టును అవసరం చేస్తాయి.

Azure ML మీకు వీలుగా చేసే సాధనాలను అందిస్తుంది:
- పంచుకున్న నోట్‌బుక్స్, కంప్యూట్ వనరులు, సర్వర్‌లెస్ కంప్యూట్, డేటా, మరియు ఎన్విరాన్‌మెంట్స్ ద్వారా మీ జట్టుతో సహకరించడం.
- లినియేజ్ మరియు ఆడిట్ అనుగుణ్యత అవసరాలను తీర్చడానికి సమానత్వం, వివరణాత్మకత, ట్రాకింగ్ మరియు ఆడిటబిలిటీతో మోడళ్లను అభివృద్ది చేయడం.
- ML మోడళ్లను వేగంగా మరియు సులభంగా స్కేలుపై డిప్లాయ్ చేయడం, మరియు వాటిని MLOps తో సమర్ధవంతంగా నిర్వహించడం మరియు పాలన చేయడం.
- బిల్ట్-ఇన్ పాలన, భద్రత, మరియు అనుగుణ్యతతో ఎక్కడైనా మెషీన్ లెర్నింగ్ పనులు అమలు చేయడం.

## పరస్పర అనుకూలమైన ప్లాట్‌ఫాం సాధనాలు

ML జట్టులోని ఎవరో తమ ఇష్టమైన సాధనాలను ఉపయోగించి పని పూర్తి చేయవచ్చు.
మీరు వేగవంతమైన ప్రయోగాలు, హైపర్‌పారామెటర్ ట్యూనింగ్, పైప్లైన్‌లు నిర్మించడం లేదా ఇన్ఫరెన్సులను నిర్వహించడం చేస్తున్నా, మీరు పరిచయమైన ఇంటర్‌ఫేస్‌లను ఉపయోగించవచ్చు, వాటిలో ఉన్నాయి:
- Azure Machine Learning Studio
- Python SDK (v2)
- Azure CLI (v2)
- Azure Resource Manager REST APIs

మీరు మోడళ్లను సుదృఢం చెయ్యగా మరియు అభివృద్ధి చక్రం మొత్తం సహకరించగా, మీరు Azure Machine Learning స్టూడియో UIలో ఆస్తులు, వనరులు మరియు మెట్రిక్స్‌ను పంచుకోవచ్చు మరియు కనుగొనవచ్చు.

## **LLM/SLM in Azure ML**

Azure ML అనేక LLM/SLM-సంబంధిత ఫంక్షన్లను జోడించింది, LLMOps మరియు SLMOps ను కలిపి పూర్తి ఎంటర్‌ప్రైజ్-స్థాయి జనరేటివ్ కృత్రిమ బుద్ధి సాంకేతిక ప్లాట్‌ఫాం సృష్టిస్తుంది.

### **మోడల్ క్యాటలాగ్**

ఎంటర్‌ప్రైజ్ వినియోగదారులు Model Catalog ద్వారా వివిధ వ్యాపార సందర్భాల అనుసారం వేరే వేరే మోడళ్లను డిప్లాయ్ చేయవచ్చు, మరియు ఎంటర్‌ప్రైజ్ డెవలపర్లు లేదా వినియోగదారులు అందుకు యాక్సెస్ చేసుకునేలా Model as Service రూపంలో సేవలు అందించవచ్చు.

![మోడళ్లు](../../../../translated_images/models.e6c7ff50a51806fd.te.png)

Azure Machine Learning స్టూడియోలోని మోడల్ క్యాటలాగ్ జనరేటివ్ AI అప్లికేషన్లు తయారు చేయడానికి ఉపయోగపడే విస్తృత శ్రేణి మోడళ్లను కనుగొనడం మరియు ఉపయోగించడం కోసం ఒక హబ్‌గా ఉంటుంది. మోడల్ క్యాటలాగ్‌లో Azure OpenAI service, Mistral, Meta, Cohere, Nvidia, Hugging Face వంటి మోడల్ ప్రొవైడర్ల ద్వారా వందలకు చేరువగా మోడళ్లు ఉన్నాయి, వాటిలో Microsoft శిక్షకులు తయారుచేసిన మోడళ్ళు కూడా ఉంటాయి. Microsoft కాకుండా ఇతర ప్రొవైడర్ల నుండి వచ్చిన మోడళ్లు Microsoft's Product Terms లో నిర్వచించినట్లుగా Non-Microsoft Products గా పరిగణింపబడుతాయి మరియు మోడల్‌కు సంబంధించిన నిబంధనలకు లోబడి ఉంటాయి.

### **జాబ్ పైప్లైన్**

మెషీన్ లెర్నింగ్ పైప్లైన్ యొక్క కోర్ ఒక పూర్తి మెషీన్ లెర్నింగ్ పనిని బహుళ అడుగుల వర్క్‌ఫ్లోగా విభజించడం. ప్రతి దశ ఒక నిర్వహించదగిన భాగం, ఇది వ్యక్తిగతంగా అభివృద్ధి చేయబడవచ్చు, ఆప్టిమైజ్ చేయబడవచ్చు, కాన్ఫిగర్ చేయబడవచ్చు మరియు ఆటోమేట్ చేయబడవచ్చు. దశలు బాగా నిర్వచించబడిన ఇంటర్‌ఫేస్‌ల ద్వారా కనెక్ట్ చేయబడతాయి. Azure Machine Learning పైప్లైన్ సర్వీస్ సమస్త పైప్లైన్ దశల మధ్య ఉన్న డిపెండెన్సీలు ఆటోమేటిగ్గా ఒర్కెస్ట్రేట్ చేస్తుంది.

SLM / LLM ను ఫైన్-ట్యూనింగ్ చేస్తున్నప్పుడు, మేము పైప్‌లైన్ ద్వారా మా డేటా, శిక్షణ మరియు జనరేషన్ ప్రక్రియలను నిర్వహించవచ్చు

### **ప్రాంప్ట్ ఫ్లో**


Azure Machine Learning prompt flow ఉపయోగించడంలోని ప్రయోజనాలు
Azure Machine Learning prompt flow వినియోగదారులను ఆలోచన నుంచి ప్రయోగం వరకూ, మరియు చ 마지막 గా ప్రొడక్షన్-రెడీ LLM-ఆధారిత అప్లికేషన్లు తయారుచేసే దాకా మారడానికి సహాయపడే వివిధ ప్రయోజనాలను అందిస్తుంది:

**ప్రాంప్ట్ ఇంజనీరింగ్ చురుకుదనం**

ఇంటరాక్టివ్ ఆథరింగ్ అనుభవం: Azure Machine Learning prompt flow ఫ్లో నిర్మాణానికి దృశ్య ప్రదర్శనను అందిస్తుంది, వినియోగదారులు తమ ప్రాజెక్టులను సులభంగా అర్థం చేసుకుని నావిగేట్ చేయగలరు. ఇది ఫ్లో అభివృద్ధి మరియు డీబగ్గింగ్ కోసం నోట్‌బుక్-టైప్ కోడింగ్ అనుభవాన్ని కూడా అందిస్తుంది.
ప్రాంప్ట్ ట్యూనింగ్ కోసం వేరియంట్లు: వినియోగదారులు బహుళ ప్రాంప్ట్ వేరియంట్లను సృష్టించి సరిపోల్చవచ్చు, పునరావృత మెరుగుదల ప్రక్రియను సులభతరం చేస్తుంది.

ఇవాల్యుయేషన్: బిల్ట్-ఇన్ ఎవాల్యుయేషన్ ఫ్లోలు వినియోగదారులకు వారి ప్రాంప్ట్స్ మరియు ఫ్లోస్ యొక్క నాణ్యత మరియు ప్రభావాన్ని అంచనా వేయడానికి అనుమతిస్తాయి.

సంపూర్ణ వనరులు: Azure Machine Learning prompt flow అభివృద్ధికి ఆరంభపాయింట్గా పనిచేసే బిల్ట్-ఇన్ టూల్స్, నమూనాలు మరియు టెంప్లేట్‌ల లైబ్రరీని కలుపుతుంది, ఇది సృజనాత్మకతకు ప్రేరణనిస్తూ ప్రక్రియను వేగవంతం చేస్తుంది.

**LLM-ఆధారిత అప్లికేషన్లకు ఎంటర్‌ప్రైజ్ సిద్ధత**

సహకారం: Azure Machine Learning prompt flow జట్టు సహకారాన్ని మద్దతు చేయడం ద్వారా బహుళ వినియోగదారులు ప్రాంప్ట్ ఇంజనీరింగ్ ప్రాజెక్ట్లపై కలిసి పని చేయగలుగుతారు, జ్ఞానాన్ని పంచుకోవచ్చు మరియు వెర్షన్ నియంత్రణని నిర్వహించవచ్చు.

అన్ని-ఒక ప్లాట్‌ఫాం: Azure Machine Learning prompt flow అభివృద్ధి మరియు ఎవాల్యుయేషన్ నుండి డిప్లాయ్‌మెంట్ మరియు మానిటరింగ్ వరకు మొత్తం ప్రాంప్ట్ ఇంజనీరింగ్ ప్రక్రియను సరళీకృతం చేస్తుంది. వినియోగదారులు తమ ఫ్లోలను Azure Machine Learning endpoints గా సులభంగా డిప్లాయ్ చేసి వాటి పనితీరును రియల్-టైమ్‌లో మానిటర్ చేయవచ్చు, తద్వారా ఉత్తమ ఆపరేషన్ మరియు నిరంతర మెరుగుదలను నిర్ధారించవచ్చు.

Azure Machine Learning Enterprise Readiness Solutions: Prompt flow Azure Machine Learning యొక్క బలమైన ఎంటర్‌ప్రైజ్ రెడినెస్ పరిష్కారాలను ఉపయోగిస్తుంది, ఫ్లోల అభివృద్ధి, ప్రయోగం మరియు డిప్లాయ్‌మెంట్‌కు భద్రత, స్కేలబిలిటీ మరియు విశ్వసనీయత కలిగిన పునాదిని అందిస్తుంది.

Azure Machine Learning prompt flow తో, వినియోగదారులు తమ ప్రాంప్ట్ ఇంజనీరింగ్ చురుకుదనాన్ని ఉపయోగించగలరు, సమర్థవంతంగా సహకరించగలరు, మరియు విజయవంతమైన LLM-ఆధారిత అప్లికేషన్ అభివృద్ధి మరియు డిప్లాయ్‌మెంట్ కోసం ఎంటర్‌ప్రైజ్-గ్రేడ్ పరిష్కారాలను అన్వయించగలరు.
