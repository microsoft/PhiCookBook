# మీ VS Code ఎక్స్‌టెన్షన్‌కు స్వాగతం

## ఫోల్డర్‌లో ఏముంది

* ఈ ఫోల్డర్ మీ ఎక్స్‌టెన్షన్ కోసం అవసరమైన అన్ని ఫైళ్లను కలిగి ఉంది.
* `package.json` - ఇది మీ ఎక్స్‌టెన్షన్ మరియు కమాండ్‌ను ప్రకటించే మానిఫెస్ట్ ఫైల్.
  * సాంపుల్ ప్లగిన్ ఒక కమాండ్‌ను రిజిస్టర్ చేసి దాని శీర్షిక మరియు కమాండ్ పేరు నిర్వచిస్తుంది. ఈ సమాచారంతో VS Code కమాండ్‌ను కమాండ్ పాలెట్‌లో చూపించగలదు. ఇప్పుడే ప్లగిన్‌ను లోడ్ చేయవలసిన అవసరం లేదు.
* `src/extension.ts` - ఇది మీ కమాండ్ యొక్క అమలు అందించే ప్రధాన ఫైల్.
  * ఫైల్ ఒక ఫంక్షన్‌ను ఎగుమతి చేస్తుంది, `activate`, ఇది మీ ఎక్స్‌టెన్షన్ very first timeactivated (ఈ సందర్భంలో కమాండ్‌ను అమలు చేయడం ద్వారా) పిలవబడుతుంది. `activate` ఫంక్షన్ లోపల మేము `registerCommand` ను పిలుస్తాము.
  * కమాండ్ అమలును కలిగించే ఫంక్షన్‌ను `registerCommand` కు రెండవ పరామితిగా పంపిస్తాము.

## సెటప్

* సిఫార్సు చేసిన ఎక్స్‌టెన్షన్లను ఇన్‌స్టాల్ చేయండి (amodio.tsl-problem-matcher, ms-vscode.extension-test-runner, and dbaeumer.vscode-eslint)


## వెంటనే ప్రారంభించండి

* మీ ఎక్స్‌టెన్షన్ లోడ్ అయిన కొత్త విండో తెరవడానికి `F5` నొక్కండి.
* (`Ctrl+Shift+P` లేదా Mac పై `Cmd+Shift+P`) నొక్కి కమాండ్ పాలెట్ నుండి మీ కమాండ్‌ను అమలు చేయండి మరియు `Hello World` టైప్ చేయండి.
* మీ ఎక్స్‌టెన్షన్‌ను డీబగ్ చేయడానికి `src/extension.ts` లో మీ కోడ్‌లో బ్రేక్‌పాయింట్లు సెట్ చేయండి.
* మీ ఎక్స్‌టెన్షన్ నుంచి అవుట్‌పుట్‌ను డీబగ్ కన్సోల్‌లో కనుగొనండి.

## మార్పులు చేయండి

* `src/extension.ts` లో కోడ్ మార్చిన తర్వాత డీబగ్ టూల్‌బార్ నుండి ఎక్స్‌టెన్షన్‌ను మళ్లీ ప్రారంభించవచ్చు.
* మీ మార్పులను లోడ్ చేయడానికిగాను మీ ఎక్స్‌టెన్షన్ ఉన్న VS Code విండోను రీలోడ్ చేయవచ్చు (`Ctrl+R` లేదా Mac లో `Cmd+R`).


## API ను అన్వేషించండి

* ఫైల్ `node_modules/@types/vscode/index.d.ts` ను తెరిచినప్పుడు మీరు మా API యొక్క పూర్తి సెట్‌ని చూడవచ్చు.

## టెస్టులు నడపండి

* [ఎక్స్‌టెన్షన్ టెస్ట్ రన్నర్](https://marketplace.visualstudio.com/items?itemName=ms-vscode.extension-test-runner) ను ఇన్‌స్టాల్ చేయండి
* **Tasks: Run Task** కమాండ్ ద్వారా "watch" టాస్క్‌ను నడపండి. ఇది రన్ అవుతుందో లేదో ఖచ్చితంగా చూసుకోండి, లేకపోతే టెస్టులు కనుగొనబడకపోవచ్చు.
* ఆక్టివిటీ బార్ నుండి పరీక్షల వీక్షణను తెరిచి Run Test" బటన్‌ను క్లిక్ చేయండి, లేదా హాట్‌కీ `Ctrl/Cmd + ; A` ఉపయోగించండి
* పరీక్ష ఫలితాల అవుట్‌పుట్‌ను Test Results వీక్షణలో చూడండి.
* `src/test/extension.test.ts` లో మార్పులు చేయండి లేదా `test` ఫోల్డర్ లో కొత్త టెస్ట్ ఫైళ్ళను సృష్టించండి.
  * ఇవ్వబడిన టెస్ట్ రన్నర్ కేవలం `**.test.ts` పేరు ప్యాటర్న్‌కు డగ్గు పడే ఫైళ్ళను మాత్రమే పరిగణిస్తుంది.
  * మీ టెస్టులను నిర్మించడానికి మీరు `test` ఫోల్డర్ లో ఫోల్డర్లు సృష్టించవచ్చు.

## మరింత ముందుకు వెళ్లండి

* మీ ఎక్స్‌టెన్షన్ పరిమాణాన్ని తగ్గించి స్టార్ట్-అప్ సమయాన్ని మెరుగుపరచడానికి [మీ ఎక్స్‌టెన్షన్‌ను బండిల్ చేయడం](https://code.visualstudio.com/api/working-with-extensions/bundling-extension).
* VS Code ఎక్స్‌టెన్షన్ మార్కెట్‌ప్లేస్‌లో [మీ ఎక్స్‌టెన్షన్‌ను ప్రచురించండి](https://code.visualstudio.com/api/working-with-extensions/publishing-extension).
* బిల్డ్లను ఆటోమేట్ చేయడానికి [సతత ఇంటిగ్రేషన్](https://code.visualstudio.com/api/working-with-extensions/continuous-integration) సెటప్ చేయండి.
