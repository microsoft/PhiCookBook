# మీ VS Code ఎక్స్‌టెన్షన్‌కు స్వాగతం

## ఫోల్డర్‌లో ఏముంది

* ఈ ఫోల్డర్‌లో మీ ఎక్స్‌టెన్షన్‌కు అవసరమైన అన్ని ఫైళ్లు ఉన్నాయి.
* `package.json` - ఇది మానిఫెస్ట్ ఫైల్, ఇందులో మీరు మీ ఎక్స్‌టెన్షన్ మరియు కమాండ్‌ను నిర్వచిస్తారు.
  * ఈ నమూనా ప్లగిన్ ఒక కమాండ్‌ను రిజిస్టర్ చేసి దాని శీర్షికను మరియు కమాండ్ పేరు ను నిర్వచిస్తుంది. ఈ సమాచారంతో VS Code కమాండ్ ప్యాలెట్‌లో కమాండ్‌ను చూపించగలదు. ఇది ఇప్పటిదాకా ప్లగిన్‌ను లోడ్ చేయాల్సిన అవసరం లేదు.
* `src/extension.ts` - ఇది మీ కమాండ్ అమలును అందించే ప్రధాన ఫైల్.
  * ఆ ఫైల్ ఒక ఫంక్షన్‌ను ఎగుమతి చేస్తుంది, `activate`, ఇది మీ ఎక్స్‌టెన్షన్ మొదటి సారిగా యాక్టివేట్ అయినప్పుడు (ఈ సందర్భంలో కమాండ్‌ను అమలు చేయడంవల్ల) పిలవబడుతుంది. `activate` ఫంక్షన్ యొక్క లోపల మేము `registerCommand`ను పిలుస్తాము.
  * మేము కమాండ్ అమలును కలిగివున్న ఫంక్షన్‌ను `registerCommand`కి రెండవ పారామీటర్‌గా పంపుతాము.

## Setup

* సిఫార్సు చేయబడిన ఎక్స్‌టెన్షన్లను ఇన్‌స్టాల్ చేయండి (amodio.tsl-problem-matcher, ms-vscode.extension-test-runner, and dbaeumer.vscode-eslint)


## వెంటనే ప్రారంభించండి

* మీ ఎక్స్‌టెన్షన్ లోడ్ అయిన కొత్త విండోను తెరవడానికి `F5` నొక్కండి.
* (`Ctrl+Shift+P` లేదా Macలో `Cmd+Shift+P`) నొక్కి కమాండ్ ప్యాలెట్ నుండి మీ కమాండ్‌ను నడిపించి `Hello World` టైప్ చేయండి.
* మీ ఎక్స్‌టెన్షన్‌ను డీబాగ్ చేయడానికి `src/extension.ts` లోని మీ కోడ్‌లో బ్రేక్‌పాయింట్‌లు సెట్ చేయండి.
* డీబగ్ కన్సోల్‌లో మీ ఎక్స్‌టెన్షన్ నుంచి అవుట్పుట్‌ను కనుగొనండి.

## మార్పులు చేయండి

* `src/extension.ts`లో కోడ్ మార్చిన తర్వాత మీరు డీబగ్ టూల్‌బార్ నుండి ఎక్స్‌టెన్షన్‌ను మళ్లీ ప్రారంభించవచ్చు.
* మీ మార్పులను లోడ్ చేయడానికి VS Code విండోను కూడా రీలోడ్ చేయవచ్చు (`Ctrl+R` లేదా Macలో `Cmd+R`) మీ ఎక్స్‌టెన్షన్‌తో కలిసి.

## APIని అన్వేషించండి

* మీరు ఫైల్ `node_modules/@types/vscode/index.d.ts` తెరిచేప్పుడు మా API యొక్క పూర్తి సెట్‌ను చూడవచ్చు.

## టెస్టులు నడపండి

* ఇన్‌స్టాల్ చేయండి [ఎక్స్‌టెన్షన్ టెస్ట్ రన్నర్](https://marketplace.visualstudio.com/items?itemName=ms-vscode.extension-test-runner)
* "watch" టాస్కును **Tasks: Run Task** కమాండ్ ద్వారా నడపండి. ఇది నడుస్తున్నదని నిర్ధారించుకోండి, ఉండకపోతే టెస్టులు కనబడకపోవచ్చు.
* activity bar నుండి Testing వీక్షణను తెరిచి Run Test" బటన్‌పై క్లిక్ చేయండి, లేదా హాట్‌కీ `Ctrl/Cmd + ; A` ఉపయోగించండి
* టెస్ట్ ఫలితాలు Test Results వీక్షణలో అవుట్పుట్‌ను చూడండి.
* `src/test/extension.test.ts`లో మార్పులు చేయండి లేకపోతే `test` ఫోల్డర్‌లో కొత్త టెస్ట్ ఫైళ్లను సృష్టించండి.
  * అందించిన టెస్ట్ రన్నర్ పేరు నమూనా `**.test.ts`కి సరిపోయే ఫైళ్లను మాత్రమే పరిగణిస్తుంది.
  * మీరు మీ టెస్టులను ఇష్టపడిన విధంగా నిర్మించడానికి `test` ఫోల్డర్‌లో ఫోల్డర్లను సృష్టించవచ్చు.

## ముందుకు వెళ్లండి

* మీ ఎక్స్‌టెన్షన్ పరిమాణాన్ని తగ్గించి స్టార్టప్ సమయాన్ని మెరుగుపరచడానికి [మీ ఎక్స్‌టెన్షన్‌ను బండిల్ చేయడం](https://code.visualstudio.com/api/working-with-extensions/bundling-extension) ద్వారా చేయండి.
* [మీ ఎక్స్‌టెన్షన్‌ను ప్రచురించండి](https://code.visualstudio.com/api/working-with-extensions/publishing-extension) VS Code ఎక్స్‌టెన్షన్ మార్కెట్‌ప్లేస్‌లో.
* [సతత ఇంటిగ్రేషన్](https://code.visualstudio.com/api/working-with-extensions/continuous-integration) సెట్‌అప్ చేయడం ద్వారా బిల్డ్స్‌ను ఆటోమెేట్ చేయండి.
