## Ollama 的 Phi 系列


[Ollama](https://ollama.com) 讓更多人能夠透過簡單腳本直接部署開源 LLM 或 SLM，並且也能建立 API 以協助本地 Copilot 應用情境。

## **1. 安裝**

Ollama 支援在 Windows、macOS 與 Linux 上執行。您可以透過此連結安裝 Ollama（[https://ollama.com/download](https://ollama.com/download)）。安裝成功後，您可以直接在終端機視窗使用 Ollama 指令來呼叫 Phi-3。您可以查看 [Ollama 中的所有可用模型庫](https://ollama.com/library)。如果您在 Codespace 中打開此儲存庫，Ollama 已會事先安裝。

```bash

ollama run phi4

```

> [!NOTE]
> 模型在您首次執行時會先下載。當然您也可以直接指定已下載的 Phi-4 模型。我們以 WSL 作為範例來執行該指令。模型下載完成後，您就可以直接在終端機上互動。

![執行](../../../../../translated_images/tw/ollama_run.e9755172b162b381.png)

## **2. 從 Ollama 呼叫 phi-4 API**

如果您想呼叫由 Ollama 產生的 Phi-4 API，您可以在終端機中使用這個指令來啟動 Ollama 伺服器。

```bash

ollama serve

```

> [!NOTE]
> 如果在 MacOS 或 Linux 上執行，請注意您可能會遇到以下錯誤 **"Error: listen tcp 127.0.0.1:11434: bind: address already in use"**。在執行指令時您可能會得到此錯誤。您可以忽略該錯誤，因為它通常表示伺服器已在執行，或是您可以停止並重新啟動 Ollama：

**macOS**

```bash

brew services restart ollama

```

**Linux**

```bash

sudo systemctl stop ollama

```

Ollama 支援兩種 API：generate 和 chat。您可以依需求呼叫 Ollama 提供的模型 API，向在 11434 埠執行的本地服務發送請求。

**Chat**

```bash

curl http://127.0.0.1:11434/api/chat -d '{
  "model": "phi3",
  "messages": [
    {
      "role": "system",
      "content": "Your are a python developer."
    },
    {
      "role": "user",
      "content": "Help me generate a bubble algorithm"
    }
  ],
  "stream": false
  
}'
```

## 其他資源

請在 [他們的資源庫](https://ollama.com/library) 查看 Ollama 中可用的模型清單。

使用此指令從 Ollama 伺服器拉取您的模型

```bash
ollama pull phi4
```

使用此指令執行模型

```bash
ollama run phi4
```

***注意：*** 欲了解更多請造訪此連結 [https://github.com/ollama/ollama/blob/main/docs/api.md](https://github.com/ollama/ollama/blob/main/docs/api.md)

## 從 Python 呼叫 Ollama

您可以使用 `requests` 或 `urllib3` 向上述的本地伺服器端點發出請求。然而，在 Python 中使用 Ollama 的一個常見方式是透過 [openai](https://pypi.org/project/openai/) SDK，因為 Ollama 也提供與 OpenAI 相容的伺服器端點。

以下是 phi3-mini 的範例：

```python
import openai

client = openai.OpenAI(
    base_url="http://localhost:11434/v1",
    api_key="nokeyneeded",
)

response = client.chat.completions.create(
    model="phi4",
    temperature=0.7,
    n=1,
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Write a haiku about a hungry cat"},
    ],
)

print("Response:")
print(response.choices[0].message.content)
```

## 從 JavaScript 呼叫 Ollama 
```javascript
// 使用 Phi-4 摘要檔案的範例
script({
    model: "ollama:phi4",
    title: "Summarize with Phi-4",
    system: ["system"],
})

// 摘要的範例
const file = def("FILE", env.files)
$`Summarize ${file} in a single paragraph.`
```

## 從 C# 呼叫 Ollama

建立一個新的 C# Console 應用程式並加入下列 NuGet 套件：

```bash
dotnet add package Microsoft.SemanticKernel --version 1.34.0
```

接著在 `Program.cs` 檔案中替換此程式碼

```csharp
using Microsoft.SemanticKernel;
using Microsoft.SemanticKernel.ChatCompletion;

// add chat completion service using the local ollama server endpoint
#pragma warning disable SKEXP0001, SKEXP0003, SKEXP0010, SKEXP0011, SKEXP0050, SKEXP0052
builder.AddOpenAIChatCompletion(
    modelId: "phi4",
    endpoint: new Uri("http://localhost:11434/"),
    apiKey: "non required");

// invoke a simple prompt to the chat service
string prompt = "Write a joke about kittens";
var response = await kernel.InvokePromptAsync(prompt);
Console.WriteLine(response.GetValue<string>());
```

使用下列指令執行應用程式：

```bash
dotnet run
```
