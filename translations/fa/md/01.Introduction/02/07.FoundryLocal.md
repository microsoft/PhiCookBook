## شروع کار با مدل‌های خانواده فی در Foundry Local

### معرفی Foundry Local

Foundry Local یک راه‌حل قدرتمند استنتاج AI روی دستگاه است که قابلیت‌های AI با سطح سازمانی را مستقیماً به سخت‌افزار محلی شما می‌آورد. این آموزش شما را در تنظیم و استفاده از مدل‌های خانواده فی با Foundry Local راهنمایی می‌کند و کنترل کامل بر روی بارهای کاری AI شما را با حفظ حفظ حریم خصوصی و کاهش هزینه‌ها ارائه می‌دهد.

Foundry Local با اجرای مدل‌های AI به صورت محلی روی دستگاه شما، مزایای عملکرد، حریم خصوصی، سفارشی‌سازی و هزینه را فراهم می‌کند. این سرویس به صورت یکپارچه با گردش‌های کاری و برنامه‌های موجود شما از طریق CLI، SDK و REST API با رابط کاربری شهودی ادغام می‌شود.


![arch](../../../../../translated_images/fa/foundry-local-arch.8823e321dd8258d7.webp)

### چرا Foundry Local را انتخاب کنیم؟

درک مزایای Foundry Local به شما کمک می‌کند تا تصمیمات آگاهانه‌تری در مورد استراتژی استقرار AI خود اتخاذ کنید:

- **استنتاج روی دستگاه:** مدل‌ها را به صورت محلی روی سخت‌افزار خود اجرا کنید، هزینه‌ها را کاهش دهید و داده‌های خود را به طور کامل روی دستگاه نگه دارید.

- **سفارشی‌سازی مدل:** از مدل‌های از پیش‌تنظیم شده انتخاب کنید یا مدل‌های خود را برای برآورده کردن نیازها و کیس‌های استفاده خاص به کار ببرید.

- **صرفه‌جویی در هزینه:** با استفاده از سخت‌افزار موجود خود هزینه‌های مکرر خدمات ابری را حذف کنید و AI را در دسترس‌تر کنید.

- **ادغام بی‌وقفه:** از طریق SDK، نقاط پایان API یا CLI با برنامه‌های خود ارتباط برقرار کنید و در صورت نیاز به Microsoft Foundry مقیاس‌پذیری آسان داشته باشید.

> **یادداشت شروع:** این آموزش بر استفاده از Foundry Local از طریق رابط‌های CLI و SDK تمرکز دارد. شما هر دو روش را یاد خواهید گرفت تا بهترین روش را برای کیس استفاده خود انتخاب کنید.

## بخش ۱: راه‌اندازی Foundry Local CLI

### گام ۱: نصب

CLI Foundry Local دروازه مدیریت و اجرای مدل‌های AI به صورت محلی است. بیایید با نصب آن روی سیستم خود شروع کنیم.

**پلتفرم‌های پشتیبانی شده:** ویندوز و مک

برای دستورالعمل‌های تفصیلی نصب، لطفاً به [مستندات رسمی Foundry Local](https://github.com/microsoft/Foundry-Local/blob/main/README.md) مراجعه کنید.

### گام ۲: کاوش مدل‌های موجود

پس از نصب Foundry Local CLI، می‌توانید مدل‌های موجود برای کیس استفاده خود را بیابید. این دستور همه مدل‌های پشتیبانی شده را به شما نشان می‌دهد:


```bash
foundry model list
```

### گام ۳: آشنایی با مدل‌های خانواده فی

خانواده فی مجموعه‌ای از مدل‌ها را با بهینه‌سازی برای کیس‌های استفاده و پیکربندی‌های سخت‌افزاری مختلف ارائه می‌دهد. در اینجا مدل‌های فی موجود در Foundry Local آمده است:

**مدل‌های فی موجود:** 

- **phi-3.5-mini** - مدل جمع‌وجور برای وظایف پایه
- **phi-3-mini-128k** - نسخه با زمینه گسترده‌تر برای مکالمات طولانی‌تر
- **phi-3-mini-4k** - مدل با زمینه استاندارد برای استفاده عمومی
- **phi-4** - مدل پیشرفته با قابلیت‌های بهبود یافته
- **phi-4-mini** - نسخه سبک وزن ازPhi-4
- **phi-4-mini-reasoning** - مخصوص وظایف استدلال پیچیده

> **سازگاری سخت‌افزاری:** هر مدل می‌تواند بر اساس توانایی‌های سیستم شما برای تسریع سخت‌افزاری مختلف (CPU، GPU) پیکربندی شود.

### گام ۴: اجرای اولین مدل فی خود

بیایید با یک مثال عملی شروع کنیم. مدل `phi-4-mini-reasoning` را اجرا می‌کنیم، که در حل مسائل پیچیده به صورت گام به گام عالی است.


**دستور اجرای مدل:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **راه‌اندازی اولین بار:** هنگام اجرای مدل برای نخستین بار، Foundry Local به طور خودکار آن را به دستگاه محلی شما دانلود می‌کند. زمان دانلود بسته به سرعت شبکه شما متفاوت است، پس لطفاً در هنگام راه‌اندازی اولیه صبور باشید.

### گام ۵: آزمایش مدل با یک مساله واقعی

اکنون بیایید مدل خود را با یک مسأله منطقی کلاسیک امتحان کنیم تا ببینیم چگونه استدلال گام به گام انجام می‌دهد:

**مسأله نمونه:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**رفتار مورد انتظار:** مدل باید این مسأله را به گام‌های منطقی تقسیم کند و با استفاده از این واقعیت که قرقاول‌ها دو پا و خرگوش‌ها چهار پا دارند، سیستم معادلات را حل کند.

## بخش ۲: ساخت برنامه‌ها با SDK Foundry Local

### چرا از SDK استفاده کنیم؟

در حالی که CLI برای تست و تعاملات سریع عالی است، SDK به شما امکان می‌دهد تا Foundry Local را به صورت برنامه‌نویسی در برنامه‌های خود ادغام کنید. این امر امکانات زیر را فراهم می‌کند:

- ساخت برنامه‌های سفارشی با قدرت AI
- ایجاد گردش‌های کاری خودکار
- ادغام قابلیت‌های AI در سیستم‌های موجود
- توسعه چت‌بات‌ها و ابزارهای تعاملی

### زبان‌های برنامه‌نویسی پشتیبانی شده

Foundry Local پشتیبانی SDK را برای چندین زبان برنامه‌نویسی برای سازگاری با ترجیحات توسعه شما فراهم می‌کند:

**📦 SDKهای موجود:**

- **C# (.NET):** [مستندات و مثال‌های SDK](https://github.com/microsoft/Foundry-Local/tree/main/sdk/cs)
- **Python:** [مستندات و مثال‌های SDK](https://github.com/microsoft/Foundry-Local/tree/main/sdk/python)
- **JavaScript:** [مستندات و مثال‌های SDK](https://github.com/microsoft/Foundry-Local/tree/main/sdk/js)
- **Rust:** [مستندات و مثال‌های SDK](https://github.com/microsoft/Foundry-Local/tree/main/sdk/rust)

### مراحل بعدی

1. **SDK مورد علاقه خود را** بر اساس محیط توسعه انتخاب کنید
2. **مستندات مخصوص SDK** را برای راهنمایی‌های دقیق پیاده‌سازی دنبال کنید
3. **با مثال‌های ساده شروع کنید** قبل از ساخت برنامه‌های پیچیده‌تر
4. **کد نمونه ارائه شده را** در هر مخزن SDK کاوش کنید
