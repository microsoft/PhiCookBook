# இன்டராக்டிவ் பை 3 மினி 4K இன்ஸ்ட்ரக்ட்ச் சாட்பாட் வித் விச்பர்

## மேலோட்டம்

இன்டராக்டிவ் பை 3 மினி 4K இன்ஸ்ட்ரக்ட் சாட்பாட் என்பது பயனர்கள் மைக்ரோசாஃப்ட் பை 3 மினி 4K இன்ஸ்ட்ரக்ட் டெமோவை உரை அல்லது ஆடியோ உள்ளீட்டைப் பயன்படுத்தி தொடர்பு கொள்ள அனுமதிக்கும் ஒரு கருவி ஆகும். இந்த சாட்பாட் மொழி மாற்றம், வானிலை புதுப்பிப்புகள் மற்றும் பொது தகவல் சேகரிப்பு போன்ற பல வேலைகளுக்கு பயன்படுத்தப்படலாம்.

### தொடங்கு

இந்த சாட்பாட் பயன்படுத்த, இவைகளைப் பின்தொடரவும்:

1. புதிய [E2E_Phi-3-mini-4k-instruct-Whispser_Demo.ipynb](https://github.com/microsoft/Phi-3CookBook/blob/main/code/06.E2E/E2E_Phi-3-mini-4k-instruct-Whispser_Demo.ipynb) கோப்பை திறக்கவும்
2. நோட்புக் பிரதான சாளரத்தில், உரை உள்ளீட்டுப் பெட்டி மற்றும் "Send" பொத்தானுடன் ஒரு சாட்பாக்ஸ் இடைமுகம் காணலாம்.
3. உரை அடிப்படையிலான சாட்பாட் பயன்படுத்த, உங்கள் செய்தியை உரை உள்ளீட்டுப் பெட்டியில் টাইப் செய்து "Send" பொத்தானை அழுத்தவும். சாட்பாட் நேரடியாக நோட்புக் உள்ளேயிருந்து இயக்கக்கூடிய ஒரு ஆடியோ கோப்புடன் பதிலளிக்கும்.

**கவனம்**: இந்த கருவிக்கு GPU மற்றும் மைக்ரோசாஃப்ட் பை-3 மற்றும் OpenAI விச்பர் மாதிரிகள் பயன்பாட்டிற்கான அணுகல் தேவை, இது பேச்சு அங்கீகாரம் மற்றும் மொழி மாற்றத்திற்கு பயன்படுத்தப்படுகிறது.

### GPU தேவைகள்

இந்த டெமோவை இயக்க 12கிபி GPU நினைவுகள் தேவையுள்ளது.

**Microsoft-Phi-3-Mini-4K instruct** டெமோவை GPU-யில் இயக்க நினைவக தேவைகள் audio அல்லது உரை உள்ளீட்டின் அளவு, மொழி மாற்ற மொழி, மாதிரி வேகம் மற்றும் GPU-யின் கிடைக்கும் நினைவகத்தைப் பொறுத்தது.

பொதுவாக, விச்பர் மாதிரி GPU-களில் இயங்க வடிவமைக்கப்பட்டுள்ளது. விச்பர் மாதிரியை இயக்கக் குறைந்தபட்ச GPU நினைவக பரிந்துரைக்கப்படும் அளவு 8 GB, ஆனால் தேவையானால் அதிக நினைவகத்தையும் கையாள முடியும்.

பெரிய அளவிலான தரவு அல்லது அதிக எண்ணிக்கையிலான கோரிக்கைகளை மாதிரியில் இயக்குவது அதிக GPU நினைவகத்தையும்/அல்லது செயல்திறன் சிக்கல்களையும் உருவாக்கக்கூடும். உங்கள் பயன்பாட்டைக் கண்ணோட்டமிட்டு, வெவ்வேறு அமைப்புகளில் சோதனை செய்து நினைவக பயன்பாட்டை கண்காணிக்க பரிந்துரைக்கப்படுகிறது.

## விச்பருடன் இன்டராக்டிவ் பை 3 மினி 4K இன்ஸ்ட்ரக்ட் சாட்பாடுக்கான E2E மாதிரி

[Interactive Phi 3 Mini 4K Instruct Chatbot with Whisper](https://github.com/microsoft/Phi-3CookBook/blob/main/code/06.E2E/E2E_Phi-3-mini-4k-instruct-Whispser_Demo.ipynb) என்ற பெயரில் உள்ள ஜூபிட்டர் நோட்புக் எவ்வாறு மைக்ரோசாஃப்ட் பை 3 மினி 4K இன்ஸ்ட்ரக்ட் டெமோவை ஆடியோ அல்லது எழுத்து உள்ளீட்டிலிருந்து உரையை உருவாக்க பயன்படுத்துவது என்பதைக் காட்டுகிறது. இந்த நோட்புக் சில செயல்முறைகளை வரையறுக்கிறது:

1. `tts_file_name(text)`: இந்த செயல்முறை உருவாக்கப்பட்ட ஆடியோ கோப்பை சேமிக்க உள்ளீட்டு உரையை அடிப்படையாகக் கொண்டு கோப்பு பெயரை உருவாக்கும்.
1. `edge_free_tts(chunks_list,speed,voice_name,save_path)`: இந்த செயல்முறை Edge TTS API-ஐ பயன்படுத்தி உள்ளீட்டு உரையின் துண்டுகள் பட்டியலிலிருந்து ஒரு ஆடியோ கோப்பை உருவாக்கும். உள்ளீட்டு அளவுருக்கள் துண்டுகளின் பட்டியல், பேச்சு வேகம், குரல் பெயர் மற்றும் உருவாக்கப்பட்ட ஆடியோவை சேமிக்க வெளியீட்டு பாதை ஆகும்.
1. `talk(input_text)`: இந்த செயல்முறை Edge TTS API-ஐ பயன்படுத்தி ஆடியோ கோப்பை உருவாக்கி, /content/audio அடைவருக்குள் ஒரு சீர்திருத்தமற்ற கோப்பு பெயருடன் சேமிக்கும். உள்ளீடு உரையே பேச்சுக்கு மாற்றப்படும்.
1. `run_text_prompt(message, chat_history)`: இந்த செயல்முறை மைக்ரோசாஃப்ட் பை 3 மினி 4K இன்ஸ்ட்ரக்ட் டெமோவைப் பயன்படுத்தி செய்தி உள்ளீட்டிலிருந்து ஆடியோ கோப்பை உருவாக்கி, அதை சாட் வரலாற்றில் இணைக்கிறது.
1. `run_audio_prompt(audio, chat_history)`: இந்த செயல்முறை ஆடியோ கோப்பை விச்பர் மாதிரி API மூலம் உரையாக மாற்றி, `run_text_prompt()` செயல்முறைக்கு அனுப்புகிறது.
1. கோடு மைக்ரோசாஃப்ட் பை 3 மினி 4K இன்ஸ்ட்ரக்ட் டெமோவுடன் பயனர்கள் உரையாட 메시ஜ்களை تایப் செய்வதோ அல்லது ஆடியோ கோப்புகளை பதிவேற்றுவதோ மூலம் தொடர்பு கொள்ள Gradio செயலியை வெளியிடுகிறது. வெளியீடு ஆப்பில் உரை செய்தியாகக் காட்டப்படுகிறது.

##சிக்கல் தீர்க்கும் வழிகள்

Cuda GPU டிரைவர்களை நிறுவுதல்

1. உங்கள் லினக்ஸ் பயன்பாடுகள் புதுப்பிக்கப்பட்டுள்ளதா என்பதை உறுதிப்படுத்துக

    ```bash
    sudo apt update
    ```

1. Cuda டிரைவர்களை நிறுவுக

    ```bash
    sudo apt install nvidia-cuda-toolkit
    ```

1. cuda டிரைவர் இடத்தை பதிவு செய்யவும்

    ```bash
    echo /usr/lib64-nvidia/ >/etc/ld.so.conf.d/libcuda.conf; ldconfig
    ```

1. Nvidia GPU நினைவக அளவைச் சரிபார்க்கவும் (தேவை 12GB GPU நினைவகம்)

    ```bash
    nvidia-smi
    ```

1. கேஷை காலி செய்க: நீங்கள் PyTorch பயன்படுத்தினால், torch.cuda.empty_cache()ஐ அழைப்பதன் மூலம் பயன்படாத அனைத்து கேஷ் நினைவகத்தையும் விட்டு வைக்கலாம். பிற GPU பயன்பாடுகள் அதனைப் பயன்படுத்தலாம்.

    ```python
    torch.cuda.empty_cache() 
    ```

1. Nvidia Cuda-வைச் சரிபார்க்கவும்

    ```bash
    nvcc --version
    ```

1. ஒரு Hugging Face டோக்கன் உருவாக்க பின்வரும் பணிகளை செய்யவும்.

    - [Hugging Face Token Settings page](https://huggingface.co/settings/tokens?WT.mc_id=aiml-137032-kinfeylo) செல்லவும்.
    - **New token** ஐ தேர்வு செய்யவும்.
    - நீங்கள் பயன்படுத்த விரும்பும் திட்டத்தின் **Name**யை உள்ளிடவும்.
    - **Type**-ஐ **Write** ஆக தேர்ந்தெடுக்கவும்.

> [!NOTE]
>
> நீங்கள் பின்வரும் தவறை சந்தித்தால்:
>
> ```bash
> /sbin/ldconfig.real: Can't create temporary cache file /etc/ld.so.cache~: Permission denied 
> ```
>
> இதனை தீர்க்க, உங்கள் டெர்மினலில் கீழ்காணும் கட்டளையை டைப் செய்யவும்.
>
> ```bash
> sudo ldconfig
> ```
