## ഫൈൻ ടിουνിംഗ് സീനാറിയോകൾ

![FineTuning with MS Services](../../../../translated_images/ml/FinetuningwithMS.3d0cec8ae693e094.webp)

ഈ വിഭാഗം Microsoft Foundry ഉം Azure പരിസ്ഥിതികളിലുള്ള ഫൈൻ-റ്റ്യൂണിംഗ് സേനാറിയോകൾക്ക് ഒരു അവലോകനം നൽകുന്നു, Déployment മോഡലുകൾ, ഇൻഫ്രാസ്ട്രക്ചർ ലയറുകൾ, കൂടാതെ സാധാരണയായി ഉപയോഗിക്കുന്ന ഒപ്റ്റിമൈസേഷൻ സാങ്കേതിക വിദ്യകൾ ഉൾപ്പെടുത്തുന്നു.

**പ്ലാറ്റ്ഫോം**  
ഇത് Microsoft Foundry (ഇതിനകം Azure AI Foundry ആയി അറിയപ്പെട്ടത്)യും Azure Machine Learning ഉം അടങ്ങിയ മാനേജഡ് സർവീസുകൾ ഉൾക്കൊള്ളുന്നു, ഇവ മോഡൽ മാനേജ്‌മെന്റ്, ഓർക്കസ്ട്രേഷൻ, പരീക്ഷണ ട്രാക്കിംഗ്, ഡെപ്ലോയ്മെന്റ് വർക്ക്‌ഫ്ളോസുകൾ എന്നിവ നൽകി നൽകുന്നു.

**ഇൻഫ്രാസ്ട്രക്ചർ**  
ഫൈൻ-റ്റ്യൂണിംഗ് സ്‌കേബിള്‍ കംപ്യൂട്ട് റിസോഴ്‌സുകൾ ആവശ്യമാണ്. Azure പരിസ്ഥിതികളിൽ ഇത് സാധാരണയായി GPU അടിസ്ഥാനത്തിലുള്ള വെർച്വൽ മെഷീനുകളും ലഘുഭാരമുള്ള വർക്ക്‌ലോഡുകൾക്കായി CPU വിഭവങ്ങളും, ഡാറ്റാസെറ്റുകൾക്കും ചെക്ക്പോയിന്റുകൾക്കും സ്കേബിള്‍ സ്റ്റോറേജും ഉൾപ്പെടുന്നു.

**ടൂളുകളും ഫ്രെയിംവർക്കുകളും**  
ഫൈൻ-റ്റ്യൂണിംഗ് വർക്ക്‌ഫ്ളോയുകൾ സാധാരണയായി Hugging Face Transformers, DeepSpeed, PEFT (Parameter-Efficient Fine-Tuning) പോലുള്ള ഫ്രെയിംവർക്കുകളും ഒപ്റ്റിമൈസേഷൻ ലൈബ്രറികളും ആശ്രയിക്കുന്നു.

Microsoft സാങ്കേതിക വിദ്യകളോടെ ഫൈൻ-റ്റ്യൂണിംഗ് പ്രക്രിയ പ്ലാറ്റ്ഫോം സർവീസുകൾ, കംപ്യൂട്ട് ഇൻഫ്രാസ്ട്രക്ചർ, പരിശീലന ഫ്രെയിംവർക്കുകൾ എന്നിവ വ്യാപിപ്പിക്കുന്നു. ഈ ഘടകങ്ങൾ എങ്ങനെ പ്രവർത്തിക്കുന്നുവെന്ന് മനസിലാക്കി, ഡെവലപ്പർമാർ ഫൗണ്ടേഷൻ മോഡലുകളെ പ്രത്യേക ടാസ്കുകളിലും ഉൽ‌പാദന സേനാറിയോകളിലും കാര്യക്ഷമമായി ഇന്റർഡാപ്റ്റ് ചെയ്യാം.

## മോഡൽ ആസ് സർവീസ്

Hosted fine-tuning ഉപയോഗിച്ച്, കംപ്യൂട്ട് സജ്ജമാക്കാനും മാനേജ്മെന്റ് വേണ്ടാതെ മോഡൽ ഫൈൻ-റ്റ്യൂൺ ചെയ്യുക.

![MaaS Fine Tuning](../../../../translated_images/ml/MaaSfinetune.3eee4630607aff0d.webp)

Serverless fine-tuning Phi-3, Phi-3.5, Phi-4 മോഡൽ ഫാമിലികൾക്കായി ഇനി ലഭ്യമാണ്, ഇത് ഡെവലപ്പർമാർക്ക് കംപ്യൂട്ട് ഏറെയ്ക്കം ക്രമീകരിക്കാതെ ക്ലൗഡ്, എഞ്ച് സേനാറിയോകൾക്ക് മോഡലുകൾ എളുപ്പത്തിൽ കസ്റ്റമൈസ് ചെയ്യാൻ കഴിയും.

## മോഡൽ ആസ് എ പ്ലാറ്റ്ഫോം

ഉപയോക്താക്കൾക്ക് അവരുടെ തന്നെ കംപ്യൂട്ട് മാനേജു ചെയ്തുകൊണ്ട് മോഡലുകൾ ഫൈൻ-റ്റ്യൂൺ ചെയ്യുന്നതിന്.
