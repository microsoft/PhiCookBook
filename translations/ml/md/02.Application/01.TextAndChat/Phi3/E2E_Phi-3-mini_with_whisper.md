# ഇൻററാക്ടീവ് ഫി 3 മിനി 4K ഇൻസ്ട്രക്റ്റ് ചാറ്റ്ബോട്ട് വിത്ത് വിസ്പർ

## അവലോകനം

ഇൻററാക്ടീവ് ഫി 3 മിനി 4K ഇൻസ്ട്രക്റ്റ് ചാറ്റ്ബോട്ട് ഒരു ഉപകരണം ആണ്, ഇത് ഉപയോക്താക്കളെ മൈക്രോസോഫ்ட் ഫി 3 മിനി 4K ഇൻസ്ട്രക്റ്റ് ഡെമോയുമായി ടെക്സ്റ്റ് അല്ലെങ്കിൽ ഓഡിയോ ഇൻപുട്ടിലൂടെ ഇന്ററാക്റ്റ് ചെയ്യാൻ അനുവദിക്കുന്നു. ഈ ചാറ്റ്ബോട്ട് വിവർത്തനം, കാലാവസ്ഥാ അപ്‌ഡേറ്റുകൾ, പൊതുവായ വിവരം ശേഖരണം പോലുള്ള വിവിധ ജോലികളിന് ഉപയോഗിക്കാം.

### തുടങ്ങുന്നത്

ഈ ചാറ്റ്ബോട്ട് ഉപയോഗിക്കാൻ, താഴെ പറയുന്ന നിർദ്ദേശങ്ങൾ പാലിക്കുക:

1. ഒരു പുതിയ [E2E_Phi-3-mini-4k-instruct-Whispser_Demo.ipynb](https://github.com/microsoft/Phi-3CookBook/blob/main/code/06.E2E/E2E_Phi-3-mini-4k-instruct-Whispser_Demo.ipynb) തുറക്കുക
2. നോട്ട്‌ബുക്കിന്റെ പ്രധാന വിൻഡോയിൽ, ടെക്സ്റ്റ് ഇൻപുട്ട് ബോക്സ് ഉളള ഒരു ചാറ്റ്ബോക്സ് ഇന്റർഫേസ് കാണും, കൂടാതെ "Send" ബട്ടണും ആയിരിക്കും.
3. ടെക്സ്റ്റ് അധിഷ്ഠിത ചാറ്റ്ബോട്ട് ഉപയോഗിക്കാൻ, താങ്കളുടെ സന്ദേശം ടെക്സ്റ്റ് ഇൻപുട്ട് ബോക്സിൽ ടൈപ് ചെയ്ത് "Send" ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക. ചാറ്റ്ബോട്ട്, നോട്ട്‌ബുക്കിനുള്ളിൽ നിന്നായി നേരിട്ടു പ്ലേ ചെയ്യാവുന്ന ഒരു ഓഡിയോ ഫയൽ പ്രതിസന്ധിക്കുന്നതാണ്.

**ഗവേഷിക്കുക**: ഈ ഉപകരണം GPU-വും മൈക്രോസോഫ്റ്റ് ഫി-3 മോഡലും OpenAI Whisper മോഡലും ഉപയോഗിക്കുന്നതിന് അനുമതിയും ആവശ്യമാണ്, ഇത് ശബ്ദം തിരിച്ചറിയാനും വിവർത്തനത്തിനും ഉപയോഗിക്കുന്നു.

### GPU ആവശ്യകതകൾ

ഈ ഡെമോ നടത്താൻ 12GB GPU മെമ്മറി ആവശ്യമുണ്ട്.

**Microsoft-Phi-3-Mini-4K instruct** ഡെമോ GPUയിൽ പ്രവർത്തിപ്പിക്കുന്നതിനുള്ള മെമ്മറി ആവശ്യങ്ങൾ നിർണയിക്കുന്നത് പല ഘടകങ്ങളെ ആശ്രയിച്ചിരിക്കുന്നു, ഉദാഹരണത്തിന്, ഇൻപുട്ട് ഡാറ്റയുടെ വലുപ്പം (ഓഡിയോ അല്ലെങ്കിൽ ടെക്സ്റ്റ്), വിവർത്തനത്തിനുപയോഗിക്കുന്ന ഭാഷ, മോഡലിന്റെ വേഗത, GPUയിൽ ലഭ്യമായ മെമ്മറി എന്നിവ.

പൊതു രീതിയിൽ, Whisper മോഡൽ GPUകളിൽ പ്രവർത്തിക്കുന്നതിന് രൂപകൽപ്പന ചെയ്തതാണ്. Whisper മോഡൽ പ്രവർത്തിപ്പിക്കാൻ നിർദേശിക്കുന്ന കുറഞ്ഞ GPU മെമ്മറി 8 GB ആണ്, എന്നാൽ അത് കൂടുതൽ മെമ്മറി കൈകാര്യം ചെയ്യാനും കഴിയുന്നു.

വലിയ തോതിലുള്ള ഡാറ്റ അല്ലെങ്കിൽ ഉയർന്ന തോതിലുള്ള അഭ്യർത്ഥനകൾ മോഡലിൽ പ്രവർത്തിപ്പിക്കുന്നത് കൂടുതൽ GPU മെമ്മറി ആവശ്യപ്പെടാം അല്ലെങ്കിൽ പ്രകടന പ്രശ്നങ്ങൾ സൃഷ്ടിക്കാം. നിങ്ങളുടെ ഉപയോഗം വിവിധ ക്രമീകരണങ്ങളോടെ പരീക്ഷിച്ച് മെമ്മറി ഉപയോഗം നിരീക്ഷിച്ച് താങ്കളുടെ പ്രത്യേക ആവശ്യങ്ങൾക്കായി മികച്ച ക്രമീകരണങ്ങൾ കണ്ടെത്താൻ ശുപാർശ ചെയ്യുന്നു.

## ഇൻററാക്ടീവ് ഫി 3 മിനി 4K ഇൻസ്ട്രക്റ്റ് ചാറ്റ്ബോട്ടിന് വവ്വേർിൽ E2E സാമ്പിൾ

[ഇൻററാക്ടീവ് ഫി 3 മിനി 4K ഇൻസ്ട്രക്റ്റ് ചാറ്റ്ബോട്ട് വിത്ത് വിസ്പർ](https://github.com/microsoft/Phi-3CookBook/blob/main/code/06.E2E/E2E_Phi-3-mini-4k-instruct-Whispser_Demo.ipynb) എന്ന ശീർഷകമുള്ള ജൂപ്പിറ്റർ നോട്ട്‌ബുക്ക് മൈക്രോസോഫ്റ്റ് ഫി 3 മിനി 4K ഇൻസ്ട്രക്റ്റ് ഡെമോ ഉപയോഗിച്ച് എങ്ങനെ ഓഡിയോ അല്ലെങ്കിൽ എഴுதிய ടെക്സ്റ്റ് ഇൻപുട്ടിൽ നിന്നു ടെക്സ്റ്റ് ജനറേറ്റ് ചെയ്യാമെന്ന് കാണിക്കുന്നു. നോട്ട്‌ബുക്ക് താഴെ പറയുന്ന ചില ഫംഗ്ഷനുകൾ നിർവചിക്കുന്നു:

1. `tts_file_name(text)`: ഈ ഫംഗ്ഷൻ, നിർമ്മിച്ച ഓഡിയോ ഫയൽ സംരക്ഷിക്കുന്നതിന് ഇൻപുട്ട് ടെക്സ്റ്റിനെ അടിസ്ഥാനമാക്കി ഒരു ഫയൽ നാമം നിർവർത്ഥിക്കുന്നു.
1. `edge_free_tts(chunks_list,speed,voice_name,save_path)`: ഈ ഫംഗ്ഷൻ Edge TTS API ഉപയോഗിച്ച് ഇൻപുട്ട് ടെക്സ്റ്റിന്റെ ചങ്കുകളുടെ പട്ടികയിൽ നിന്നു ഓഡിയോ ഫയൽ നിർമ്മിക്കുന്നു. ഇൻപുട്ട് പാരാമീറ്ററുകൾ ചങ്കുകളുടെ പട്ടിക, സംസാര വേഗം, വോയിസ് നാമം, നിർമിച്ച ഓഡിയോ ഫയൽ സംരക്ഷിക്കുന്ന പാത എന്നിവയാണ്.
1. `talk(input_text)`: ഈ ഫംഗ്ഷൻ Edge TTS API ഉപയോഗിച്ച് ഒരു ഓഡിയോ ഫയൽ നിർമ്മിച്ച് അത് /content/audio ഡയറക്ടറിയിൽ ഒരു രാൻഡം ഫയൽ നാമത്തിൽ സംരക്ഷിക്കുന്നു. ഇൻപുട്ട് പാരാമീറ്റർ ആണ് സംസാരത്തിന് പരിവർത്തനം ചെയ്യേണ്ട ടെക്സ്റ്റ്.
1. `run_text_prompt(message, chat_history)`: ഈ ഫംഗ്ഷൻ മൈക്രോസോഫ്റ്റ് ഫി 3 മിനി 4K ഇൻസ്ട്രക്റ്റ് ഡെമോ ഉപയോഗിച്ച് സന്ദേശ ഇൻപുട്ടിൽ നിന്നു ഓഡിയോ ഫയൽ നിർമ്മിക്കുകയും അത് ചാറ്റ് ഹിസ്റ്ററിയിൽ ചേർക്കുകയും ചെയ്യുന്നു.
1. `run_audio_prompt(audio, chat_history)`: ഈ ഫംഗ്ഷൻ ഓഡിയോ ഫയൽ ഒരു ടെക്സ്റ്റായി Whisper മോഡൽ API ഉപയോഗിച്ച് മാറ്റുകയും അത് `run_text_prompt()` ഫംഗ്ഷനിൽ പാസ്സുചെയ്യുകയും ചെയ്യുന്നു.
1. കോഡ് ഒരു Gradio ആപ്പ് ആരംഭിക്കുന്നു, ഇതിലൂടെ ഉപയോക്താക്കൾ ഫി 3 മിനി 4K ഇൻസ്ട്രക്റ്റ് ഡെമോയുമായി സന്ദേശങ്ങൾ ടൈപ് ചെയ്യുകയോ ഓഡിയോ ഫയലുകൾ അപ്‌ലോഡ് ചെയ്യുകയോ ചെയ്തുകൊണ്ട് സംവദിക്കാം. ഔട്ട്പുട്ട് ആപ്പിൽ ടെക്സ്റ്റ് സന്ദേശമായി പ്രദർശിപ്പിക്കുന്നു.

## പ്രശ്നപരിഹാരം

Cuda GPU ഡ്രൈവർ ഇൻസ്റ്റാൾ කිරීම

1. നിങ്ങളുടെ ലിനക്സ് അപ്ലിക്കേഷനുകൾ അപ്‌ഡേറ്റ് ചെയ്തിട്ടുണ്ടെന്ന് ഉറപ്പാക്കുക

    ```bash
    sudo apt update
    ```

1. Cuda ഡ്രൈവർ ഇൻസ്റ്റാൾ ചെയ്യുക

    ```bash
    sudo apt install nvidia-cuda-toolkit
    ```

1.Cuda ഡ്രൈവർ സ്ഥലം രജിസ്റ്റർ ചെയ്യുക

    ```bash
    echo /usr/lib64-nvidia/ >/etc/ld.so.conf.d/libcuda.conf; ldconfig
    ```

1. Nvidia GPU മെമ്മറി വലുപ്പം പരിശോധിക്കുക (ആവശ്യമാണ് 12GB GPU മെമ്മറി)

    ```bash
    nvidia-smi
    ```

1. കാഷെ കാലിയാക്കുക: നിങ്ങൾ PyTorch ഉപയോഗിക്കുന്നെങ്കിൽ, torch.cuda.empty_cache() വിളിച്ച് ഉപയോഗിക്കാത്ത എല്ലാ കാഷെ ചെയ്ത മെമ്മറിയും റിലീസ് ചെയ്യാം, ഇതുവഴി അത് മറ്റ് GPU ആപ്ലിക്കേഷനുകൾക്ക് ഉപയോഗിക്കാം

    ```python
    torch.cuda.empty_cache() 
    ```

1. Nvidia Cuda പരിശോധന

    ```bash
    nvcc --version
    ```

1. Hugging Face ടോക്കൺ സൃഷ്ടിക്കാൻ താഴെ പറയുന്ന പ്രവർത്തനങ്ങൾ ചെയ്യുക.

    - [Hugging Face Token Settings page](https://huggingface.co/settings/tokens?WT.mc_id=aiml-137032-kinfeylo) സന്ദർശിക്കുക.
    - **New token** തിരഞ്ഞെടുക്കുക.
    - ഉപയോഗിക്കാൻ ആഗ്രഹിക്കുന്ന പ്രോജക്ട് **പേര്** നൽകുക.
    - **Type** ആയി **Write** തിരഞ്ഞെടുക്കുക.

> [!NOTE]
>
> താഴെ കാണിച്ച പിശക് നിങ്ങൾ അനുഭവപ്പെട്ടാൽ:
>
> ```bash
> /sbin/ldconfig.real: Can't create temporary cache file /etc/ld.so.cache~: Permission denied 
> ```
>
> ഇത് പരിഹരിക്കാൻ, നിങ്ങളുടെ ടെർമിനലിൽ താഴെ കാണിക്കുന്ന കമാൻഡ് ടൈപ്പ് ചെയ്യുക.
>
> ```bash
> sudo ldconfig
> ```
