# Phi-3.5-Instruct WebGPU RAG ചാറ്റ്ബോട്ട്

## WebGPUയും RAG മാതൃകയും പ്രദർശിപ്പിക്കാൻ ഡെമോ

Phi-3.5 Onnx ഹോസ്റ്റ് ചെയ്ത മോഡലോടുകൂടിയ RAG മാതൃക Retrieval-Augmented Generation സമീപനം ഉപയോഗിക്കുന്നു, Phi-3.5 മോഡലുകളുടെ ശേഷിയും ONNX ഹോസ്റ്റിങ്ങിന്റെ പ്രയോജനങ്ങളും സംയോജിപ്പിച്ച് കാര്യക്ഷമമായ AI വിന്യസനങ്ങൾക്ക് സഹായകമാണ്. ഡൊമൈൻ-നിർദിഷ്ട കാര്യങ്ങൾക്ക് മോഡലുകൾ ഫൈൻ-ട്യൂൺ ചെയ്യുന്നതിൽ ഈ മാതൃക പ്രധാനമാണ്, ഇത് ഗുണമേൻമ, ചിലവു- ഫലപ്രതിത്തി, ദൈർഘ്യമുള്ള കോൺടെക്സ്റ്റ് ബോധ്യം എന്നിവയുടെ സമന്വയം നൽകുന്നു. പല വ്യവസായങ്ങളുടെയും കസ്റ്റമൈസേഷൻ ആവശ്യങ്ങൾക്കനുസരിച്ച് കണ്ടെത്താനും, പരീക്ഷിക്കാനും, ഉപയോഗിക്കാനും എളുപ്പമുള്ള വ്യാപക മോഡൽ തിരഞ്ഞെടുപ്പുമായി ഇത് Azure AI-യുടെ സ്യൂട്ടിന്റെ ഭാഗമാണ്. 

## WebGPU എന്താണ്
WebGPU ഒരു ആധുനിക വെബ് ഗ്രാഫിക്സ് API ആണ്, വെബ് ബ്രൗസറുകളിൽ നിന്ന് ഡിവൈസിന്റെ ഗ്രാഫിക്സ് പ്രോസസ്സിംഗ് യൂണിറ്റിലേക്കുള്ള (GPU) കാര്യക്ഷമ പ്രവേശനം നൽകാൻ രൂപകൽപ്പന ചെയ്‌തത്. ഇത് WebGL-ന്റെ പൈതൃകനെ ആയാണ് ലക്ഷ്യമിടുന്നത്, താഴെ കുറിച്ചിരിക്കുന്ന ചില പ്രധാന മെച്ചപ്പെടുത്തലുകൾ നൽകുന്നു:

1. **ആധുനിക GPU കളുമായി അനുയോജ്യത**: WebGPU ആധുനിക GPU ശില്പശാസ്ത്രങ്ങളുമായി സ്ത്രീത്രമായി പ്രവർത്തിക്കാൻ നിർമ്മിച്ചതാണ്, Vulkan, Metal, Direct3D 12 പോലുള്ള സിസ്റ്റം API കളെ ഉപയോഗപ്പെടുത്തി.
2. ** വിപുലമായ പ്രകടനം**: ഇത് പൊതുമേഖലാ-ഉദ്ദേശ്യ GPU കണക്കുകൂട്ടലുകൾക്കും വേഗമുള്ള പ്രവർത്തനങ്ങൾക്ക് പിന്തുണ നൽകുന്നു, ഗ്രാഫിക്സ് റെൻഡറിംഗിനും മെഷിൻ ലേണിംഗ് ടാസ്കുകൾക്കും അനുയോജ്യമാണ്.
3. **ഉന്നതമായ ഫീച്ചറുകൾ**: WebGPU കൂടുതൽ സങ്കീർണവും ഡൈനാമിക് ഗ്രാഫിക്സ്, കണക്കുകൂട്ടൽ വർക്ക്‌ലോഡുകൾക്കും സമ്മതിക്കുന്ന കൂടുതൽ അതുല്യ GPU ശേഷികളെ ആക്സസ് ചെയ്യാൻ അനുവദിക്കുന്നു.
4. **JavaScript-ന്റെ ജോലിഭാരം കുറയ്ക്കൽ**: കൂടുതൽ ജോലികൾ GPUയ്ക്ക് മൊഴിമാറ്റം ചെയ്‌തുകൊണ്ട്, WebGPU JavaScript-ൽ ഉള്ള ജോലിഭാരം aanzienlijk കുറയ്ക്കുന്നു, അതേസമയം മികച്ച പ്രകടനവും സുഗമമായ അനുഭവങ്ങളും നൽകുന്നു.

WebGPU നിലവിൽ Google Chrome പോലുള്ള ബ്രൗസറുകളിൽ പിന്തുണയ്ക്കപ്പെടുന്നു, മറ്റ് പ്ലാറ്റ്ഫോത്തിലേക്കുള്ള പിന്തുണ വികസിപ്പിക്കുന്ന പ്രവൃത്തി തുടരുകയാണ്.

### 03.WebGPU
Required Environment:

**Supported browsers:** 
- Google Chrome 113+
- Microsoft Edge 113+
- Safari 18 (macOS 15)
- Firefox Nightly.

### WebGPU സജ്ജമാക്കുക:

- Chrome/Microsoft Edge ൽ

Enable the `chrome://flags/#enable-unsafe-webgpu` flag.

#### നിങ്ങളുടെ ബ്രൗസർ തുറക്കുക:
Google Chrome അല്ലെങ്കിൽ Microsoft Edge തുറക്കുക.

#### Flags പേജ് തുറക്കുക:
അഡ്രസ് ബാറിൽ `chrome://flags` ടൈപ്പ് ചെയ്ത് Enter അമർത്തുക.

#### ഫ്ലാഗ് തിരയുക:
പേജിന്റെ മുകളിൽ ഉള്ള സെർച്ച് ബോക്സിൽ 'enable-unsafe-webgpu' ടൈപ്പ് ചെയ്യുക

#### ഫ്ലാഗ് സജീവമാക്കുക:
പരिणാമങ്ങളുടെ പട്ടികയിൽ #enable-unsafe-webgpu ഫ്ലാഗ് കണ്ടെത്തുക.

ഇതിന്റെ അടുത്തുള്ള ഡ്രോപ്പ്ഡൗൺ മെനുവിൽ ക്ലിക് ചെയ്ത് Enabled തിരഞ്ഞെടുക്കുക.

#### നിങ്ങളുടെ ബ്രൗസർ പുനരാരംഭിക്കുക:
ഫ്ലാഗ് സജീവമാക്കിയതിനു ശേഷം മാറ്റങ്ങൾ പ്രാബല്യത്തിൽ വരുന്നതിന് നിങ്ങളുടെ ബ്രൗസർ പുനരാരംഭിക്കേണ്ടതുണ്ട്. പേജിന്റെ അടിയിൽ പ്രത്യക്ഷപ്പെടുന്ന Relaunch ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക.

- Linux үшін, ബ്രൗസർ `--enable-features=Vulkan` എന്ന ഓപ്ഷനോടൊപ്പം ആരംഭിക്കുക.
- Safari 18 (macOS 15) ൽ WebGPU ഡിഫോൾട്ടായി സജീവമാണ്.
- Firefox Nightly ൽ, അഡ്രസ് ബാറിലേക്ക് about:config ടൈപ്പ് ചെയ്ത് `set dom.webgpu.enabled to true` ചെയ്യുക.

### Microsoft Edge-നായി GPU ക്രമീകരിക്കൽ

Windows-ൽ Microsoft Edge-നായി ഹൈ-പെർഫോമൻസ് GPU സജ്ജമാക്കാനുള്ള ചുവടുകൾ ഇവയാണ്:

- **Open Settings:** Start മെനുവിൽ ക്ലിക്ക് ചെയ്ത് Settings തിരഞ്ഞെടുക്കുക.
- **System Settings:** System എന്നതിലേക്ക് പോകുകയായിരുന്നും പിന്നെ Display തിരഞ്ഞെടുക്കുക.
- **Graphics Settings:** താഴേക്ക് സ്‌ക്രോൾ ചെയ്യുകയും Graphics settings ക്ലിക്ക് ചെയ്യുകയും ചെയ്യുക.
- **Choose App:** “Choose an app to set preference” എന്നതിന്റെ അടിയിൽ Desktop app തിരഞ്ഞെടുക്കുകയും പിന്നീട് Browse ചെയ്യുക.
- **Select Edge:** സാധാരണയായി ഉള്ള Edge ഇൻസ്റ്റലേഷൻ ഫോൾഡറിലേക്ക് (`C:\Program Files (x86)\Microsoft\Edge\Application`) നാവിഗേറ്റ് ചെയ്ത് `msedge.exe` തിരഞ്ഞെടുക്കുക.
- **Set Preference:** Options ക്ലിക്ക് ചെയ്യുക, High performance തിരഞ്ഞെടുക്കുക, പിന്നെ Save ക്ലിക്ക് ചെയ്യുക.
ഇത് Microsoft Edge നിങ്ങളുടെ ഹൈ-പെർഫോമൻസ് GPU ഉപയോഗിക്കുന്നതായും മികച്ച പ്രകടനത്തിന് ഇത് സഹായിക്കുന്നതുമാണ് ഉറപ്പാക്കുക. 
- **Restart** നിങ്ങളുടെ മെഷീൻ ഈ ക്രമീകരണങ്ങൾ പ്രാബല്യത്തിൽ വരാൻ പുനരാരംഭിക്കുക
