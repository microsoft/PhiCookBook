ഈ ഡെമോ ഒരു pretrained മോഡൽ ഉപയോഗിച്ച് ഒരു ചിത്രം և ഒരു ടെക്സ്റ്റ് പ്രോംപ്റ്റിന്റെ അടിസ്ഥാനത്തിൽ Python കോഡ് സൃഷ്ടിക്കാൻ എങ്ങനെ ഉപയോഗിക്കാമെന്ന് കാണിക്കുന്നു. 

[ഉദാഹരണ കോഡ്](../../../../code/06.E2E/E2E_OpenVino_Phi3-vision.ipynb)

ഇതാ ഘട്ടങ്ങളായി വിശദീകരണം:

1. **ഇംപോർട്ടുകൾ ಮತ್ತು ക്രമീകരണം**:
   - ആവശ്യമായ ലൈബ്രറികളും മൊഡ്യൂളുകളും ഇറക്കുമതി ചെയ്യപ്പെടുന്നു, അതിൽ `requests`, ചിത്രപ്രോസസ്സിംഗിന് `PIL`, മോഡൽ കൈകാര്യം ചെയ്യാനും പ്രോസസ്സിംഗിനും `transformers` എന്നിവ ഉൾപ്പെടുന്നു.

2. **ചിത്രം ലോഡ് ചെയ്യൽ һәм പ്രദർശിപ്പിക്കൽ**:
   - ഒരു ചിത്രം ഫയൽ (`demo.png`) `PIL` ലൈബ്രറി ഉപയോഗിച്ച് തുറന്ന് പ്രദർശിപ്പിക്കുന്നു.

3. **പ്രോംപ്റ്റ് നിർവചിക്കൽ**:
   - ചിത്രം ഉൾപ്പെടുന്നും ആ ചിത്രം പ്രോസസ് ചെയ്ത് `plt` (matplotlib) ഉപയോഗിച്ച് സേവ് ചെയ്യാൻ ആവശ്യപ്പെടുന്ന ഒരു സന്ദേശം സൃഷ്ടിക്കപ്പെടുന്നു.

4. **പ്രോസസർ ലോഡ് ചെയ്യൽ**:
   - `out_dir` ഡയറക്ടറിയാൽ നിർദ്ദിഷ്ടമായ പ്രീട്രെയിൻഡ് മോഡലിൽ നിന്നാണ് `AutoProcessor` ലോഡ് ചെയ്യപ്പെടുന്നത്. ഈ പ്രോസസറാണ് ടെക്സ്റ്റ് மற்றும் ചിത്രം ഇൻപുട്ടുകൾ കൈകാര്യം کرنا.

5. **പ്രോംപ്റ്റ് സൃഷ്ടിക്കൽ**:
   - മോഡലിന് അനുയോജ്യമായ പ്രോംപ്റ്റായി സന്ദേശം ഫോർമാറ്റ് ചെയ്യാൻ `apply_chat_template` മെതഡ് ഉപയോഗിക്കുന്നു.

6. **ഇൻപുട്ടുകൾ പ്രോസസ്സ് ചെയ്യൽ**:
   - പ്രോംപ്റ്റും ചിത്രം മോഡൽക്ക് മനസ്സിലാകുന്ന ടെൻസറുകളായി പ്രോസസ് ചെയ്യപ്പെടുന്നു.

7. **ജനറേഷൻ آرഗ്യുമെന്റുകൾ സജ്ജീകരിക്കൽ**:
   - മോഡലിന്റെ ജനറേഷൻ പ്രക്രിയയ്ക്ക് അനുബന്ധമായ ആർഗ്യുമെന്റുകൾ നിർവചിക്കപ്പെടുന്നു, ഉദാഹരണത്തിന് സൃഷ്ടിക്കേണ്ട ഏറ്റവും പരമാവധി പുതിയ ടോക്കണുകളുടെ എണ്ണം এবং ഔട്ട്‌പുട്ട് സാമ്പിൾ ചെയ്യണോ എന്നത് ഉൾപ്പെടുന്നു.

8. **കോഡ് ജനറേറ്റ് ചെയ്യൽ**:
   - ഇൻപുട്ടുകളും ജനറേഷൻ آرഗ്യുമെന്റുകളും അടിസ്ഥാനമാക്കി മോഡൽ Python കോഡ് നിർമ്മിക്കുന്നു. ഔട്ട്‌പുട്ട് കൈകാര്യം ചെയ്യാൻ `TextStreamer` ഉപയോഗിക്കുന്നു, പ്രോംപ്റ്റും പ്രത്യേക ടോക്കണുകളും ഒഴിവാക്കിയാണ്.

9. **ഔട്ട്‌പുട്ട്**:
   - സൃഷ്ടിച്ച കോഡ് പ്രിന്റ് ചെയ്യപ്പെടുന്നു, ഇതിൽ ചിത്രം പ്രോസസ് ചെയ്ത് പ്രോംപ്റ്റിൽ നിർദ്ദേശിച്ചതുപോലെ സേവ് ചെയ്യുന്നതിനുള്ള Python കോഡ് ഉൾപ്പെടുകയും ചെയ്യും.

ഈ ഡെമോ OpenVino ഉപയോഗിച്ച് pretrained മോഡൽ പ്രയോജനപ്പെടുത്തി ഉപയോക്തൃ ഇൻപുട്ട് மற்றும் ചിത്രങ്ങളുടെ അടിസ്ഥാനത്തിൽ ഡൈനാമിക്കായി കോഡ് എങ്ങനെ ജനറേറ്റ് ചെയ്യാമെന്ന് കാണിക്കുന്നു.
