### Guidance-AI and Phi Models as a Service (MaaS)
ನಾವು [Guidance](https://github.com/guidance-ai/guidance) ಅನ್ನು Azure AI Foundry ನಲ್ಲಿ Phi-3.5-mini ಸರ್ವರ್‌ಲೆೆಸ್ ಎಂಡ್ಪಾಯಿಂಟ್ ಆఫರಿಂಗ್‌ಗೆ ತರುತ್ತಿದ್ದೇವೆ, ಇದು ಆפּלಿಕೇಶನ್‌ಗೆ ಹೊಂದಾಣಿಕೆಯ ساختಿಯನ್ನುนิirmaಿಸಿದೆ ಮೂಲಕ ಔಟ್ಪುಟ್‌ಗಳನ್ನು ಹೆಚ್ಚು ಭವಿಷ್ಯವಾಣಿ ಮಾಡುವದಕ್ಕಾಗಿ. Guidance ಮೂಲಕ, ನೀವು ದುಬಾರಿಯಾಗುವ ಪುನರ್‌ಪ್ರಯತ್ನಗಳನ್ನು ತೆಗೆದುಹಾಕಬಹುದು, ಉದಾಹರಣೆಗೆ ಮಾದರಿಯನ್ನು ಪೂರ್ವನಿರ್ಧರಿತ ಪಟ್ಟಿಗಳಿಂದ (ಉದಾ., ವೈದ್ಯಕೀಯ ಕೋಡ್ಗಳು) ಆಯ್ಕೆಮಾಡಲು, ಒದಗಿಸಲಾದ ಸಂದರ್ಭದಿಂದ ನೇರ ಉಲ್ಲೇಖಗಳಿಗೆ ಔಟ್ಪುಟ್‌ಗಳನ್ನು ನಿರ್ಬಂಧಿಸಲು, ಅಥವಾ ಯಾವುದೇ regex ಅನ್ನು ಅನುಸರಿಸಲು ನಿರ್ಬಂಧಿಸಬಹುದು. Guidance ಇನ್ಫರೆನ್ಸ್ ಸ್ಟ್ಯಾಕ್‌ನಲ್ಲಿ token ಮೀರಿ token ಮಾದರಿಯನ್ನು ಮಾರ್ಗದರ್ಶನ ಮಾಡುತ್ತದೆ, ವೆಚ್ಚ ಮತ್ತು ಸಡಿಲಿಕೆಯನ್ನು 30-50% ಕಡಿಮೆ ಮಾಡುತ್ತದೆ, ಇದು [Phi-3-mini serverless endpoint](https://aka.ms/try-phi3.5mini) ಗೆ ಒಂದು ವಿಶಿಷ್ಟ ಮತ್ತು ಮೌಲ್ಯವಂತವಾದ ಅಡ್ಒನ್ ಅನ್ನು ಮಾಡಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ.

## [**Guidance-AI**](https://github.com/guidance-ai/guidance) is a framework designed to help developers create and deploy AI models efficiently. It focuses on providing tools and best practices for building robust AI applications. 

When combined with **Phi Models as a Service (MaaS)**, it offers a powerful solution for deploying small language models (SLMs) that are both cost-effective and high-performing.

**Guidance-AI** ಒಂದು ಪ್ರೋಗ್ರಾಮಿಂಗ್ ಫ್ರೇಮ್ವರ್ಕ್ ಆಗಿದ್ದು, ಡೆವಲಪರ್‌ರಿಗೆ ದೊಡ್ಡ ಭಾಷಾ ಮಾದರಿಗಳನ್ನು (LLMs) ಹೆಚ್ಚು ಪರಿಣಾಮಕಾರಿಯಾಗಿ ನಿಯಂತ್ರಿಸಲು ಮತ್ತು ಮಾರ್ಗದರ್ಶನ ಮಾಡಲು ಸಹಾಯ ಮಾಡುವಂತೆ ವಿನ್ಯಾಸಗೊಳಿಸಲಾಗಿದೆ. ಇದು ಔಟ್ಪುಟ್‌ಗಳನ್ನು ನಿಖರವಾಗಿ ರಚಿಸಲು ಅನುಮತಿಸುತ್ತೇ, ಪಾರಂಪರಿಕ ಪ್ರಾಂಪ್ಟಿಂಗ್ ಅಥವಾ ಫೈನ್‌ಟ್ಯೂನಿಂಗ್ ವಿಧಾನಗಳಿಗಿಂತ ಕಡಿಮೆ ಸಡಿಲಿಕೆ ಮತ್ತು ವೆಚ್ಚವನ್ನು ಒದಗಿಸುತ್ತದೆ.

### Key Features of Guidance-AI:
- **Efficient Control**: ಡೆವಲಪರ್‌ಗಳು ಭಾಷಾ ಮಾದರಿ ಪಠ್ಯವನ್ನು ಹೇಗೆ ಉತ್ಪಾದಿಸುತ್ತದೆ ಎಂದು ನಿಯಂತ್ರಿಸಲು ಸಾಧ್ಯಮಾಡುತ್ತದೆ, ಉತ್ತಮ ಗುಣಮಟ್ಟದ ಮತ್ತು ಸಂಬಂಧಿತ ಔಟ್ಪುಟ್‌ಗಳನ್ನು ಖಾತ್ರಿಪಡಿಸುತ್ತದೆ.
- **Cost and Latency Reduction**: ಉತ್ಪಾದನಾ ಪ್ರಕ್ರಿಯನ್ನು ಹೆಚ್ಚು ವೆಲವಂತ ಮತ್ತು ವೇಗವಾಗಿ ಮಾಡುವಂತೆ ಗಟ್ಟಿಯಾಗಿಸುತ್ತದೆ.
- **Flexible Integration**: Transformers, llama.cpp, AzureAI, VertexAI ಮತ್ತು OpenAI ಸೇರಿದಂತೆ ವಿವಿಧ ಬ್ಯಾಕೆಂಡ್‌ಗಳೊಂದಿಗೆ ಕೆಲಸ ಮಾಡುತ್ತದೆ.
- **Rich Output Structures**: ಶರತಾ, ಲೂಪ್‌ಗಳು ಮತ್ತು ಉಪಕರಣ ಬಳಕೆದಂತಹ ಸಂಕೀರ್ಣ ಔಟ್ಪುಟ್ ರಚನೆಗಳನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ, ಇದರಿಂದ ಸ್ಪಷ್ಟ ಮತ್ತು ಪಾರ್ಸೇಬಲ್ ಫಲಿತಾಂಶಗಳನ್ನು ಸೃಷ್ಟಿಸುವುದು ಸುಲಭವಾಗುತ್ತದೆ.
- **Compatibility**: ಒಬ್ಬ Guidance ಪ್ರೋಗ್ರಾಂ ಅನ್ನು ಹಲವು ಬ್ಯಾಕೆಂಡ್‌ಗಳಲ್ಲಿ ನಿರ್ವಹಿಸಬಹುದಾಗಿದೆ, ಇದು ಲವಚಿಕತೆ ಮತ್ತು ಬಳಕೆಯ ಸುಲಭತೆಯನ್ನು ಹೆಚ್ಚಿಸುತ್ತದೆ.

### Example Use Cases:
- **Constrained Generation**: ಮಾದರಿಯ ಔಟ್ಪುಟ್ ಅನ್ನು ಮಾರ್ಗದರ್ಶನ ಮಾಡಲು регуляр выражения (regular expressions) ಮತ್ತು ಸಾಂದರ್ಭಿಕ-ರಹಿತ ವ್ಯಾಕರಣಗಳನ್ನು ಬಳಸುವುದು.
- **Tool Integration**: ಪಠ್ಯ ಉತ್ಪಾದನಾ ಕೆಲಸದೊಳಗೆ ಕ್ಯಾಲ್ಕುಲೇಟರ್‌ ಹಾಗು ಇತರ ಉಪಕರಣಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಮಧ್ಯೆ ಕೆಲಸಗೊಳಿಸುವಂತೆ ಒದಗಿಸುವುದು.

For more detailed information and examples, you can check out the [Guidance-AI GitHub repository](https://github.com/guidance-ai/guidance).

### Key Features of Phi Models:
1. **Cost-Effective**: ಕಡಿಮೆ ವೆಚ್ಚದಲ್ಲಿ ಉತ್ತಮ ಪ್ರದರ್ಶನವನ್ನು ಕಾಪಾಡುವಂತೆ ವಿನ್ಯಾಸಗೊಳಿಸಲಾಗಿದೆ.
2. **Low Latency**: ತ್ವರಿತ ಪ್ರತ್ಯುತ್ತರಗಳನ್ನು ಅವಶ್ಯಕಕೆಾದ ರಿಯಲ್-ಟೈಮ್ ಅಪ್ಲಿಕೇಶನ್‌ಗಳಿಗೆ ಐಡಿಯಲ್.
3. **Flexibility**: ಕ್ಲೌಡ್, ಎಡ್ಜ್ ಮತ್ತು ಆಫ್‌ಲೈನ್ ಪರಿಸರಗಳಲ್ಲಿ ನಿಯೋಜಿಸಲು ಸಾಧ್ಯ.
4. **Customization**: ಡೊಮೇನ್-ನಿರ್ದಿಷ್ಟ ಡೇಟಾ ಮೂಲಕ ಮಾದರಿಗಳನ್ನು ಫೈನ್-ಟ್ಯೂನ್ ಮಾಡಿ ಕಾರ್ಯಕ್ಷಮತೆಯನ್ನು ಹೆಚ್ಚಿಸಬಹುದು.
5. **Security and Compliance**: ಜವಾಬ್ದಾರಿ, ಪಾರದರ್ಶಕತೆ, ನ್ಯಾಯಾಚರಣೆ, ನಂಬಕತೆ, ಸುರಕ್ಷತೆ, ಗೌಪ್ಯತೆ ಮತ್ತು ಒಳಗೊಂಡಿರುವಿಕೆಯನ್ನು ಖಾತ್ರಿ ಪಡಿಸುವ Microsoft's AI ತತ್ತ್ವಗಳೊಂದಿಗೆ ನಿರ್ಮಿಸಲಾಗಿದೆ.

### Phi Models as a Service (MaaS):
Phi ಮಾದರಿ ಗಳು ಇನ್ಫರೆನ್ಸ್ API ಗಳ ಮೂಲಕ pay-as-you-go બિલಿಂಗ್ ವ್ಯವಸ್ಥೆಯಲ್ಲಿ ಲಭ್ಯವಿದ್ದು, ಪ್ರಾಥಮಿಕ ದೊಡ್ಡ ವೆಚ್ಚಗಳಿಲ್ಲದೆ ನಿಮ್ಮ ಅಪ್ಲಿಕೇಶನ್‌ಗಳೊಂದಿಗೆ ಅವುಗಳನ್ನು ಸರಳವಾಗಿ ಏಕತ್ರಗೊಳಿಸಲು ಸಹಾಯ ಮಾಡುತ್ತವೆ.

### Getting Started with Phi-3:
Phi ಮಾದರಿಗಳನ್ನು ಬಳಸಲು ಪ್ರಾರಂಭಿಸಲು, ನೀವು [Azure AI model catalog](https://ai.azure.com/explore/models) ಅಥವಾ the GitHub Marketplace Models](https://github.com/marketplace/models) ಅನ್ನು ಪರಿಶೀಲಿಸಬಹುದು, ಇದು ಪೂರ್ವನಿರ್ಮಿತ ಮತ್ತು ಕಸ್ಟಮೈಸ್ ಮಾಡಲು ಸಾದ್ಯವಾದ ಮಾದರಿಗಳನ್ನು ಒದಗಿಸುತ್ತದೆ. ಹೆಚ್ಚಾಗಿ, [Azure AI Foundry](https://ai.azure.com) ಎಂಬ ಟೂಲ್ಗಳನ್ನು ಉಪಯೋಗಿಸಿ ನಿಮ್ಮ AI ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ಅಭಿವೃದ್ಧಿಪಡಿಸಿ ಮತ್ತು ನಿಯೋಜಿಸಬಹುದು.
