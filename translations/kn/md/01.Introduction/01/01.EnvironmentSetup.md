# Phi-3 ಅನ್ನು ಸ್ಥಳೀಯವಾಗಿ ಪ್ರಾರಂಭಿಸಿ

ಈ ಮಾರ್ಗದರ್ಶಕವು Ollama ಬಳಸಿ Phi-3 ಮಾದರಿಯನ್ನು ರನ್ ಮಾಡಲು ನಿಮ್ಮ ಸ್ಥಳೀಯ ಪರಿಸರವನ್ನು ಸಂರಚಿಸಲು ಸಹಾಯ ಮಾಡುವುದು. GitHub Codespaces, VS Code Dev Containers ಅಥವಾ ನಿಮ್ಮ ಸ್ಥಳೀಯ ಪರಿಸರವನ್ನು ಬಳಸಿಕೊಂಡು ನೀವು ಮಾದರಿಯನ್ನು ಹಲವು ವಿಧಗಳಲ್ಲಿ ರನ್ ಮಾಡಬಹುದು.

## ಪರಿಸರ ಸ್ಥಾಪನೆ

### GitHub Codespaces

GitHub Codespaces ಅನ್ನು ಬಳಸಿ ನೀವು ಈ ಟೆಂಪ್ಲೇಟ್ ಅನ್ನು ವರ್ಚುವಲ್ ಆಗಿ ರನ್ ಮಾಡಬಹುದು. ಬಟನ್ ನಿಮ್ಮ ಬ್ರೌಸರ್‌ನಲ್ಲಿ ವೆಬ್ ಆಧಾರಿತ VS Code ಇನ್ಸ್‌ಟನ್ಸ್ ಅನ್ನು ತೆರೆಯುತ್ತದೆ:

1. ಟೆಂಪ್ಲೇಟ್ ಅನ್ನು ತೆರೆಯಿ (ಇದು ಕೆಲವು ನಿಮಿಷಗಳು ತೆಗೆದುಕೊಳ್ಳಬಹುದು):

    [![GitHub Codespaces ನಲ್ಲಿ ತೆರೆಯಿರಿ](https://github.com/codespaces/badge.svg)](https://codespaces.new/microsoft/phi-3cookbook)

2. ಟರ್ಮಿನಲ್ ವಿಂಡೋವನ್ನು ತೆರೆಯಿರಿ

### VS Code Dev Containers

⚠️ ಈ ಆಯ್ಕೆಯನ್ನು ಕಾರ್ಯಗತಮಾಡಲು ನಿಮ್ಮ Docker Desktop ಗೆ ಕನಿಷ್ಠ 16 GB RAM ಹಂಚಿಕೆಯಿರಬೇಕು. ನಿಮಗೆ 16 GB ಕ್ಕಿಂತ ಕಡಿಮೆ RAM ಇದ್ದರೆ, ನೀವು [GitHub Codespaces option](../../../../../md/01.Introduction/01) ಪ್ರಯತ್ನಿಸಬಹುದು ಅಥವಾ [ಸ್ಥಳೀಯವಾಗಿ ಸೆಟ್‌ಅಪ್ ಮಾಡಿ](../../../../../md/01.Introduction/01).

ಸಂಬಂಧಿತ ಆಯ್ಕೆ VS Code Dev Containers ಆಗಿದ್ದು, ಇದು ಡೆವ್ ಕಂಟೈನರ್‍ಸ್ ವಿಸ್ತರಣೆ ([Dev Containers ವಿಸ್ತರಣೆ](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers)) ಬಳಸಿ ಪ್ರಾಜೆಕ್ಟ್ ಅನ್ನು ನಿಮ್ಮ ಸ್ಥಳೀಯ VS Code ನಲ್ಲಿ ತೆರೆಯುತ್ತದೆ:

1. Docker Desktop ಪ್ರಾರಂಭಿಸಿ (ಇದ್ದಿಲ್ಲದಿದ್ದರೆ ಅದನ್ನು ಸ್ಥಾಪಿಸಿ)
2. ಪ್ರಾಜೆಕ್ಟ್ ಅನ್ನು ತೆರೆಯಿರಿ:

    [![Dev Containers ನಲ್ಲಿ ತೆರೆಯಿರಿ](https://img.shields.io/static/v1?style=for-the-badge&label=Dev%20Containers&message=Open&color=blue&logo=visualstudiocode)](https://vscode.dev/redirect?url=vscode://ms-vscode-remote.remote-containers/cloneInVolume?url=https://github.com/microsoft/phi-3cookbook)

3. ತೆರೆಯುವ VS Code ವಿಂಡೋದಲ್ಲಿ, ಪ್ರಾಜೆಕ್ಟ್ ಫೈಲ್‌ಗಳು ಕಾಣಿಸಿಕೊಂಡ ನಂತರ (ಇದು ಕೆಲವು ನಿಮಿಷಗಳು ತೆಗೆದುಕೊಳ್ಳಬಹುದು), ಟರ್ಮಿನಲ್ ವಿಂಡೋವನ್ನು ತೆರೆಯಿರಿ.
4. [deployment steps](../../../../../md/01.Introduction/01)ೊಂದಿಗೆ ಮುಂದುವರಿಯಿರಿ

### ಸ್ಥಳೀಯ ಪರಿಸರ

1. ಕೆಳಗಿನ ಸಾಧನಗಳು ಇನ್‌ಸ್ಟಾಲ್ ಆಗಿರುವುದನ್ನು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ:

    * [Ollama](https://ollama.com/)
    * [Python 3.10+](https://www.python.org/downloads/)
    * [OpenAI Python SDK](https://pypi.org/project/openai/)

## ಮಾದರಿಯನ್ನು ಪರೀಕ್ಷಿಸಿ

1. Ollama ನಿಂದ phi3:mini ಮಾದರಿಯನ್ನು ಡೌನ್‌ಲೋಡ್ ಮಾಡಿ ಮತ್ತು ರನ್ ಮಾಡಲು ಕೇಳಿ:

    ```shell
    ollama run phi3:mini
    ```

    ಮಾದರಿಯನ್ನು ಡೌನ್‌ಲೋಡ್ ಮಾಡಲು ಅದು ಕೆಲವು ನಿಮಿಷಗಳು ತೆಗೆದುಕೊಳ್ಳುತ್ತದೆ.

2. ಔಟ್‌ಪುಟ್‌ನಲ್ಲಿ "success" ಕಾಣಿಸಿದಾಗ, ನೀವು ಆ ಮಾದರಿಗೆ ಪ್ರಾಂಪ್ಟ್‌ನಿಂದ ಸಂದೇಶವನ್ನು ಕಳುಹಿಸಬಹುದು.

    ```shell
    >>> Write a haiku about hungry hippos
    ```

3. ಕೆಲವು ಸೆಕೆಂಡುಗಳ ನಂತರ, ಮಾದರಿಯಿಂದ ಪ್ರತಿಕ್ರಿಯೆಯ ಸ್ಟ್ರೀಮ್ ಕಾಣಿಸಬೇಕು.

4. ಭಾಷಾ ಮಾದರಿಗಳೊಂದಿಗೆ ಬಳಸಲಾಗುವ ವಿಭಿನ್ನ ತಂತ್ರಗಳ ಬಗ್ಗೆ ತಿಳಿಯಲು, Python ನೋಟ್‌ಬುಕ್ [ollama.ipynb](../../../code/01.Introduce/ollama.ipynb) ಅನ್ನು ತೆರೆಯಿರಿ ಮತ್ತು ಪ್ರತಿಯೊಂದು ಸೆಲ್ ಅನ್ನು ರನ್ ಮಾಡಿ . ನೀವು 'phi3:mini' ಹೊರತು ಮತ್ತೊಂದು ಮಾದರಿಯನ್ನು ಬಳಸಿದ್ದರೆ, ಮೊದಲ ಸೆಲ್‌ನಲ್ಲಿ `MODEL_NAME` ಅನ್ನು ಬದಲಾಯಿಸಿ.

5. Python ನಲ್ಲಿ phi3:mini ಮಾದರಿಯೊಂದಿಗೆ ಸಂಭಾಷಣೆ ನಡೆಸಲು, Python ಫೈಲ್ [chat.py](../../../../../code/01.Introduce/chat.py) ಅನ್ನು ತೆರೆಯಿರಿ ಮತ್ತು ಅದನ್ನು ರನ್ ಮಾಡಿ. ಅಗತ್ಯವಿದ್ದಲ್ಲಿ ಫೈಲ್ ಮೇಲ್ಭಾಗದಲ್ಲಿರುವ `MODEL_NAME` ಅನ್ನು ನೀವು ಬದಲಾಯಿಸಬಹುದು, ಮತ್ತು ಬೇಕಾದರೆ ಸಿಸ್ಟಂ ಮೆಸೇಜ್ ಅನ್ನು ಬದಲಾಯಿಸಬಹುದು ಅಥವಾ few-shot ಉದಾಹರಣೆಗಳನ್ನು ಸೇರಿಸಬಹುದು.
