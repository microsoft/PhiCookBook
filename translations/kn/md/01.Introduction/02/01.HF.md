# **Hugging Face ನಲ್ಲಿ Phi ಕುಟುಂಬವನ್ನು ಬಳಸುವುದು**


[Hugging Face](https://huggingface.co/) ಒಂದು ಬಹುಪ್ರಸಿದ್ಧ AI ಸಮುದಾಯವಾಗಿದೆ, ಸಮೃದ್ಧ ಡೇಟಾ ಮತ್ತು ಓಪನ್ ಸೋರ್ಸ್ ಮಾದರಿ ಸಂಪನ್ಮೂಲಗಳೊಂದಿಗೆ. ವಿಭಿನ್ನ ನಿರ್ಮಾಪಕರು Hugging Face ಮೂಲಕ ಓಪನ್-ಸೋರ್ಸ್ LLM ಮತ್ತು SLM ಬಿಡುಗಡೆ ಮಾಡುತ್ತಾರೆ, ಉದಾಹರಣೆಗೆ Microsoft, Meta, Mistral, Apple, Google ಇತ್ಯಾದಿ.

Microsoft Phi Family ಅನ್ನು Hugging Face ನಲ್ಲಿ ಬಿಡುಗಡೆ ಮಾಡಲಾಗಿದೆ. ಡೆವೆಲಪರ್‌ಗಳು ತಮ್ಮ ಸನ್ನಿವೇಶಗಳು ಮತ್ತು ವ್ಯಾಪಾರದ ಅವಶ್ಯಕತೆಗಳ ಆಧಾರದ ಮೇಲೆ ಸಂಬಂಧಿತ Phi ಫ್ಯಾಮಿಲಿಯ ಮಾದರಿಗಳನ್ನು ಡೌನ್‌ಲೋಡ್ ಮಾಡಬಹುದು. Hugging Face ನಲ್ಲಿ Phi PyTorch ಮಾದರಿಗಳನ್ನು ನಿಯೋಜಿಸುವದೊಡನೆ ಜೊತೆಗೆ, ನಾವು ಪ್ರಮಾಣೀಕೃತ (quantized) ಮಾದರಿಗಳನ್ನು ಕೂಡ విడుదల ಮಾಡಿದ್ದೇವೆ, GGUF ಮತ್ತು ONNX ಫಾರ್ಮ್ಯಾಟ್‌ಗಳನ್ನು ಬಳಸಿಕೊಂಡು ಅಂತಿಮ ಬಳಕೆದಾರರಿಗೆ ಆಯ್ಕೆಯನ್ನು ನೀಡಲು.


## **Hugging Face ನಲ್ಲಿ ಮಾದರಿಗಳನ್ನು ಡೌನ್‌ಲೋಡ್ ಮಾಡುವುದು**

ಈ ಲಿಂಕ್ ಮೂಲಕ ನೀವು Phi ಫ್ಯಾಮಿಲಿ ಮಾದರಿಯನ್ನು ಡೌನ್‌ಲೋಡ್ ಮಾಡಬಹುದು

[Hugging Face ನಲ್ಲಿ Microsoft ಮಾದರಿಗಳು](https://huggingface.co/microsoft)

-  **Phi-1 / 1.5** https://huggingface.co/collections/microsoft/phi-1-6626e29134744e94e222d572

-  **Phi-3 / 3.5** https://huggingface.co/collections/microsoft/phi-3-6626e15e9585a200d2d761e3

-  **Phi-4** https://huggingface.co/collections/microsoft/phi-4-677e9380e514feb5577a40e4

- **Phi-4-reasoning** https://huggingface.co/microsoft/Phi-4-reasoning

- **Phi-4-reasoning Plus** https://huggingface.co/microsoft/Phi-4-reasoning-plus 

- **Phi-4-mini-reasoning** https://huggingface.co/microsoft/Phi-4-mini-reasoning

ನೀವು ಮಾದರಿಯನ್ನು ವಿಭಿನ್ನ ಮಾರ್ಗಗಳಲ್ಲಿ ಡೌನ್‌ಲೋಡ್ ಮಾಡಬಹುದು, ಉದಾಹರಣೆಗೆ ***Hugging Face CLI SDK*** ಅನ್ನು ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡಿ ಅಥವಾ ***git clone*** ಅನ್ನು ಬಳಸಿರಿ.

### **Phi ಫ್ಯಾಮಿಲಿ ಮಾದರಿಯನ್ನು ಡೌನ್‌ಲೋಡ್ ಮಾಡಲು Hugging Face CLI ಬಳಸುವುದು**

- Hugging Face CLI ಅನ್ನು ಸ್ಥಾಪಿಸಿ

```bash

pip install -U "huggingface_hub[cli]"

```

- huggingface-cli ಬಳಸಿ ಲಾಗಿನ್ ಮಾಡುವುದು

ನಿಮ್ಮ [Settings page](https://huggingface.co/settings/tokens) ನಿಂದ [User Access Token](https://huggingface.co/docs/hub/security-tokens) ಬಳಸಿ Hugging Face ಗೆ ಲಾಗಿನ್ ಮಾಡಿ


```bash

huggingface-cli login --token $HF_TOKEN --add-to-git-credential

```

- ಡೌನ್‌ಲೋಡ್ ಮಾಡಿ 


ನೀವು ಮಾದರಿಯನ್ನು ಡೌನ್‌ಲೋಡ್ ಮಾಡಿ ಕ್ಯಾಶ್‌ಗೆ ಉಳಿಸಬಹುದು

```bash

huggingface-cli download microsoft/phi-4

```

ನೀವು ಸ್ಥಳವನ್ನು ನಿಮ್ಮ ನಿರ್ದಿಷ್ಟ ಸ್ಥಳಕ್ಕೆ ಸೆಟ್ ಮಾಡಬಹುದು


```bash

huggingface-cli download microsoft/phi-4 --local-dir $YOUR_PATH

```


### **Phi ಫ್ಯಾಮಿಲಿ ಮಾದರಿಯನ್ನು ಡೌನ್‌ಲೋಡ್ ಮಾಡಲು git clone ಬಳಸುವುದು**

ಮಾದರಿಯನ್ನು ಡೌನ್‌ಲೋಡ್ ಮಾಡಲು ನೀವು ***git clone*** ಅನ್ನು ಕೂಡ ಬಳಸಬಹುದು

```bash

git lfs install

git clone https://huggingface.co/microsoft/phi-4

```

## **ಉದಾಹರಣೆಗಳು - Microsoft Phi-4 ನಲ್ಲಿ ಇನ್ಫರೆನ್ಸ್**

- **transformers ಗ್ರಂಥಾಲಯವನ್ನು ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡುವುದು**

```bash

pip install transformers -U

```

- **ಈ ಕೋಡ್ ಅನ್ನು VSCode ನಲ್ಲಿ ರನ್ ಮಾಡುವುದು**

```python

import transformers

pipeline = transformers.pipeline(
    "text-generation",
    model="microsoft/phi-4",
    model_kwargs={"torch_dtype": "auto"},
    device_map="auto",
)

messages = [
    {"role": "user", "content": "I have $20,000 in my savings account, where I receive a 4% profit per year and payments twice a year. Can you please tell me how long it will take for me to become a millionaire? Also, can you please explain the math step by step as if you were explaining it to an uneducated person?"},
]

outputs = pipeline(messages, max_new_tokens=2048)
print(outputs[0]["generated_text"][-1])

```
