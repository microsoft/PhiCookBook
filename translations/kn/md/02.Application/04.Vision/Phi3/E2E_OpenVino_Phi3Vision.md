ಈ ಡೆಮೊವು ಚಿತ್ರ ಮತ್ತು ಪಠ್ಯ ಸೂಚನೆ ಆಧರಿಸಿ ಪೂರ್ವಪ್ರಶಿಕ್ಷಿತ ಮಾದರಿಯನ್ನು ಬಳಸಿಕೊಂಡು Python ಕೋಡ್ ಅನ್ನು ರಚಿಸಲು ಹೇಗೆ ಬಳಸಬೇಕೆಂದು ತೋರಿಸುತ್ತದೆ. 

[ನಮೂನಾ ಕೋಡ್](../../../../code/06.E2E/E2E_OpenVino_Phi3-vision.ipynb)

ಇದಕ್ಕೆ ಹಂತಹಂತಗಿನ ವಿವರ ಇಲ್ಲಿದೆ:

1. **ಆಮದುಗಳು ಮತ್ತು ಸಿದ್ಧತೆ**:
   - ಅಗತ್ಯ ಗ್ರಂಥಾಲಯಗಳು ಮತ್ತು ಮೋಡ್ಯೂಲ್‌ಗಳನ್ನು ಆಮದು ಮಾಡಲಾಗುತ್ತದೆ, ಅದರೊಳಗೆ `requests`, ಚಿತ್ರ ಪ್ರಕ್ರಿಯೆಯಿಗಾಗಿ `PIL`, ಮತ್ತು ಮಾದರಿ ಮತ್ತು ಪ್ರೊಸೆಸಿಂಗ್ ನಿರ್ವಹಣೆಗೆ `transformers` ಸೇರಿವೆ.

2. **ಚಿತ್ರವನ್ನು ಲೋಡ್ ಮಾಡಿ ಪ್ರದರ್ಶಿಸುವುದು**:
   - `PIL` ಗ್ರಂಥಾಲಯ ಬಳಸಿ ಒಂದು ಚಿತ್ರ ಫೈಲ್ (`demo.png`) ತೆರೆಯಲಾಗುತ್ತದೆ ಮತ್ತು ಪ್ರದರ್ಶಿಸಲಾಗುತ್ತದೆ.

3. **ಪ್ರಾಂಪ್ಟ್ ವ್ಯಾಖ್ಯಾನ**:
   - ಚಿತ್ರ ಮತ್ತು `plt` (matplotlib) ಬಳಸಿ ಚಿತ್ರವನ್ನು ಪ್ರಕ್ರಿಯೆಗೊಳಿಸಿ ಸಂರಕ್ಷಿಸುವ ಪೈಥನ್ ಕೋಡ್ ರಚಿಸಲು ವಿನಂತಿಯನ್ನು ಒಳಗೊಂಡ ಒಂದು ಸಂದೇಶ ರಚಿಸಲಾಗುತ್ತದೆ.

4. **ಪ್ರೊಸೆಸರ್ ಅನ್ನು ಲೋಡ್ ಮಾಡುವುದು**:
   - `AutoProcessor` ಅನ್ನು `out_dir` ಡೈರೆಕ್ಟರಿಯಲ್ಲಿ ಸೂಚಿಸಿದ ಪೂರ್ವಪ್ರಶಿಕ್ಷಿತ ಮಾದರಿಯಿಂದ ಲೋಡ್ ಮಾಡಲಾಗಿದೆ. ಈ ಪ್ರೊಸೆಸರ್ ಪಠ್ಯ ಮತ್ತು ಚಿತ್ರ ಇನ್‌ಪುಟ್‌ಗಳನ್ನು ನಿರ್ವಹಿಸುತ್ತದೆ.

5. **ಪ್ರಾಂಪ್ಟ್ ಸೃಷ್ಟಿಸುವುದು**:
   - ಸಂದೇಶವನ್ನು ಮಾದರಿಗೆ ಸೂಕ್ತವಾದ ಪ್ರಾಂಪ್ಟ್ ಆಗಿ ಸ್ವರೂಪಗೊಳಿಸಲು `apply_chat_template` ವಿಧಾನ ಬಳಸಲಾಗುತ್ತದೆ.

6. **ಇನ್‌ಪುಟ್‌ಗಳನ್ನು ಪ್ರಕ್ರಿಯೆಗೊಳಿಸುವುದು**:
   - ಪ್ರಾಂಪ್ಟ್ ಮತ್ತು ಚಿತ್ರವನ್ನು ಮಾದರಿ ಅರ್ಥಮಾಡಿಕೊಳ್ಳಬಹುದಾದ ಟೆನ್ಸರ್‌ಗಳಾಗಿ پروಸೆಸ್ ಮಾಡಲಾಗುತ್ತದೆ.

7. **ಜನರೇಶನ್ ಆರ್ಗ್ಯುಮೆಂಟ್‌ಗಳನ್ನು ಹೊಂದಿಸುವುದು**:
   - ಮಾದರಿಯ ಜನರೇಶನ್ ಪ್ರಕ್ರಿಯೆಯ ಆರ್ಗ್ಯುಮೆಂಟ್‌ಗಳು ನಿರ್ಧರಿಸಲಾಗುತ್ತವೆ, ಇದರಲ್ಲಿ ರಚಿಸುವ ಗರಿಷ್ಠ ಹೊಸ ಟೋಕನ್‌ಗಳ ಸಂಖ್ಯೆ ಮತ್ತು ಔಟ್‌ಪುಟ್ ಅನ್ನು ಸ್ಯಾಂಪಲ್ ಮಾಡಬೇಕೇ ಎಂಬುದು ಸೇರಿದೆ.

8. **ಕೋಡ್ ರಚನೆ**:
   - ಇನ್‌ಪುಟ್‌ಗಳು ಮತ್ತು ಜನರೇಶನ್ ಆರ್ಗ್ಯುಮೆಂಟ್‌ಗಳ ಆಧಾರವಾಗಿ ಮಾದರಿ Python ಕೋಡ್ ಅನ್ನು ರಚಿಸುತ್ತದೆ. ಔಟ್‌ಪುಟ್‌ನ್ನು ನಿರ್ವಹಿಸಲು `TextStreamer` ಬಳಸಲಾಗುತ್ತದೆ, ಇದು ಪ್ರಾಂಪ್ಟ್ ಮತ್ತು ವಿಶೇಷ ಟೋಕನ್‌ಗಳನ್ನು ಬಿಟ್ಟುಹಾಕುತ್ತದೆ.

9. **ಔಟ್‌ಪುಟ್**:
   - ರಚಿಸಿದ ಕೋಡ್ ಮುದ್ರಣಗೊಳ್ಳುತ್ತದೆ, ಅದು ಪ್ರಾಂಪ್ಟ್‌ನಲ್ಲಿ ಸೂಚಿಸಿದಂತೆ ಚಿತ್ರವನ್ನು ಪ್ರಕ್ರಿಯೆಗೊಳಿಸಿ ಸಂರಕ್ಷಿಸುವ Python ಕೋಡ್ ಅನ್ನು ಒಳಗೊಂಡಿರಬೇಕು.

ಈ ಡೆಮೊವು OpenVino ಬಳಸಿ ಪೂರ್ವಪ್ರಶಿಕ್ಷಿತ ಮಾದರಿಯನ್ನು ಉಪಯೋಗಿಸಿ ಬಳಕೆದಾರನ ಇನ್‌ಪುಟ್ ಮತ್ತು ಚಿತ್ರಗಳ ಆಧಾರದ ಮೇಲೆ ಡೈನಾಮಿಕ್ ಆಗಿ ಕೋಡ್ ರಚಿಸುವುದನ್ನು ಹೇಗೆ ಸಾಧಿಸಬಹುದು ಎಂದು示ಿಸಿದೆ.
