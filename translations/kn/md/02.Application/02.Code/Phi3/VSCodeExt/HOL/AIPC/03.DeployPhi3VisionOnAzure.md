# **Lab 3 - Azure Machine Learning Service ನಲ್ಲಿ Phi-3-vision ಅನ್ನು ನಿಯೋಜಿಸುವುದು**

ನಾವು ಸ್ಥಳೀಯ ಕೋಡ್‌ನ ಉತ್ಪಾದನಾ ನಿಯೋಜನೆಯನ್ನು ಪೂರ್ಣಗೊಳಿಸಲು NPU ಅನ್ನು ಬಳಸುತ್ತೇವೆ, ಮತ್ತು ನಂತರ ಇದರ ಮೂಲಕ PHI-3-VISION ಅನ್ನು ಪರಿಚಯಿಸಿ ಚಿತ್ರಗಳಿಂದ ಕೋಡ್ ರಚಿಸುವ ಸಾಮರ್ಥ್ಯವನ್ನು ಸೇರಿಸಲು ಬಯಸುತ್ತೇವೆ.

ಈ ಪರಿಚಯದಲ್ಲಿ, ನಾವು Azure Machine Learning Service ನಲ್ಲಿ ಒಂದು Model As Service ಆಗಿರುವ Phi-3 Vision ಸೇವೆಯನ್ನು ತ್ವರಿತವಾಗಿ ರಚಿಸಬಹುದು.

***ಗಮನಿಸಿ***： Phi-3 Vision ಗೆ ವಿಷಯವನ್ನು ವೇಗವಾಗಿ ರಚಿಸಲು ಗಣನಾ ಶಕ್ತಿ ಅಗತ್ಯವಿದೆ. ಇದನ್ನು ಸಾಧಿಸಲು ನಮಗೆ ಕ್ಲೌಡ್ ಗಣನಾಶಕ್ತಿ ಬೇಕಾಗುತ್ತದೆ.


### **1. Azure Machine Learning Service ರಚಿಸಿ**

ನಾವು Azure Portal ನಲ್ಲಿ ಒಂದು Azure Machine Learning Service ರಚಿಸಬೇಕು. ಹೇಗೆ ಎಂದು ತಿಳಿದುಕೊಳ್ಳಲು, ದಯವಿಟ್ಟು ಈ ಲಿಂಕ್ ಅನ್ನು ನೋಡಿ [https://learn.microsoft.com/azure/machine-learning/quickstart-create-resources?view=azureml-api-2](https://learn.microsoft.com/azure/machine-learning/quickstart-create-resources?view=azureml-api-2)

***ಗಮನಿಸಿ***

1. ಪ್ರಸಾರ ಮಾಡಲು ಅಗತ್ಯವಿರುವ ಪರಾಮೀಟರ್‌ಗಳಲ್ಲಿ 반드시 Authorization, azureml-model-deployment ಮತ್ತು Content-Type ಸೇರಿರಬೇಕು. ಅವುಗಳನ್ನು ಪಡೆಯಲು ನೀವು ನಿಯೋಜನೆ ಮಾಹಿತಿಯನ್ನು ಪರಿಶೀಲಿಸಬೇಕು.

2. ಪರಾಮೀಟರ್‌ಗಳನ್ನು ಪ್ರಸಾರ ಮಾಡಲು, Phi-3-Vision ಗೆ ಚಿತ್ರ ಲಿಂಕ್ ಅನ್ನು ಪ್ರಸಾರ ಮಾಡಬೇಕಾಗುತ್ತದೆ. ದಯವಿಟ್ಟು ಪರಾಮೀಟರ್‌ಗಳನ್ನು ಪ್ರಸಾರ ಮಾಡುವುದಕ್ಕೆ GPT-4-Vision ವಿಧಾನವನ್ನು ನೋಡಿ, ಉದಾಹರಣೆಗೆ

```json

{
  "input_data":{
    "input_string":[
      {
        "role":"user",
        "content":[ 
          {
            "type": "text",
            "text": "You are a Python coding assistant.Please create Python code for image "
          },
          {
              "type": "image_url",
              "image_url": {
                "url": "https://ajaytech.co/wp-content/uploads/2019/09/index.png"
              }
          }
        ]
      }
    ],
    "parameters":{
          "temperature": 0.6,
          "top_p": 0.9,
          "do_sample": false,
          "max_new_tokens": 2048
    }
  }
}

```

3. Post ವಿಧಾನವನ್ನು ಬಳಸಿಕೊಂಡು **/score** ಅನ್ನು ಕರೆಮಾಡಿ

**ಅಭಿನಂದನೆಗಳು** ！ನೀವು ವೇಗದ PHI-3-VISION ನಿಯೋಜನೆಯನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದ್ದೀರಿ ಮತ್ತು ಚಿತ್ರಗಳನ್ನು ಬಳಸಿಕೊಂಡು ಕೋಡ್ ರಚಿಸುವ ವಿಧಾನವನ್ನು ಪ್ರಯತ್ನಿಸಿದ್ದೀರಿ. ಮುಂದಾಗಿ, ನಾವು NPUs ಮತ್ತು ಕ್ಲೌಡ್‌ಗಳೊಂದಿಗೆ ಸಂಯೋಜಿಸಿ ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ರಚಿಸಬಹುದು
