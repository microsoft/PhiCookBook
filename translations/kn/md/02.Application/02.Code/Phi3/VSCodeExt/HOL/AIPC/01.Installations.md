# **ಲ್ಯಾಬ್ 0 - ಸ್ಥಾಪನೆ**

When we enter the Lab, we need to configure the relevant environment :


### **1. Python 3.11+**

It is recommended to use miniforge to configure your Python environment 

To configure miniforge, please refer to [https://github.com/conda-forge/miniforge](https://github.com/conda-forge/miniforge)

After configuring miniforge, run the following command in Power Shell

```bash

conda create -n pyenv python==3.11.8 -y

conda activate pyenv

```


### **2. Prompt flow SDK ಸ್ಥಾಪನೆ**

ಲ್ಯಾಬ್ 1 ನಲ್ಲಿ ನಾವು Prompt flow ಅನ್ನು ಬಳಸುತ್ತೇವೆ, ಆದ್ದರಿಂದ ನಿಮಗೆ Prompt flow SDK ಅನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಬೇಕಿದೆ.

```bash

pip install promptflow --upgrade

```

ನೀವು ಈ ಆಜ್ಞೆಯಿಂದ promptflow SDK ಅನ್ನು ಪರಿಶೀಲಿಸಬಹುದು


```bash

pf --version

```

### **3. Visual Studio Code Prompt flow ವಿಸ್ತರಣೆ ಸ್ಥಾಪನೆ**

### **4. Intel NPU Acceleration Library**

Intel ನ ಹೊಸ ತಲೆಮಾರು ಪ್ರೊಸೆಸರ್‌ಗಳು NPU ಅನ್ನು ಬೆಂಬಲಿಸುತ್ತವೆ. NPU ಅನ್ನು ಬಳಸಿಕೊಂಡು LLMs / SLMs ಅನ್ನು ಲೋಕಲ್‌ನಲ್ಲಿ ರನ್ ಮಾಡಲು ಇಚ್ಛಿಸಿದರೆ, ನೀವು ***Intel NPU Acceleration Library*** ಅನ್ನು ಬಳಸಬಹುದು. ಹೆಚ್ಚಿನ ಮಾಹಿತಿಗಾಗಿ ನೀವು ಓದಲು ಸಾಧ್ಯವಿದೆ [https://github.com/microsoft/PhiCookBook/blob/main/md/01.Introduction/03/AIPC_Inference.md](https://github.com/microsoft/PhiCookBook/blob/main/md/01.Introduction/03/AIPC_Inference.md).

bash ನಲ್ಲಿ Intel NPU Acceleration Library ಅನ್ನು ಸ್ಥಾಪಿಸಿ


```bash

pip install intel-npu-acceleration-library

```

***ಗಮನಿಸಿ***: ದಯವಿಟ್ಟು ಗಮನಿಸಿ ಈ ಲೈಬ್ರರಿ transformers ***4.40.2*** ಅನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ, ದಯವಿಟ್ಟು ಆ ಆವೃತ್ತಿಯನ್ನು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ


### **5. ಇತರೆ Python ಲೈಬ্রರಿ**


requirements.txt ಫೈಲ್ ಅನ್ನು ರಚಿಸಿ ಮತ್ತು ಈ ವಿಷಯವನ್ನು ಸೇರಿಸಿ

```txt

notebook
numpy 
scipy 
scikit-learn 
matplotlib 
pandas 
pillow 
graphviz

```


### **6. NVM ಸ್ಥಾಪನೆ**

Powershell ನಲ್ಲಿ nvm ಅನ್ನು ಸ್ಥಾಪಿಸಿ 


```bash

winget install -e --id CoreyButler.NVMforWindows

```

nodejs 18.20 ಅನ್ನು ಸ್ಥಾಪಿಸಿ


```bash

nvm install 18.20.0

nvm use 18.20.0

```

### **7. Visual Studio Code ಅಭಿವೃದ್ಧಿ ಬೆಂಬಲ ಸ್ಥಾಪನೆ**


```bash

npm install --global yo generator-code

```

ಅಭಿನಂದನೆ! ನೀವು ಯಶಸ್ವಿಯಾಗಿ SDK ಅನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಿದ್ದೀರಿ. ಮುಂದಾಗಿ, ಪ್ರಾಯೋಗಿಕ ಹಂತಗಳಿಗೆ ಮುಂದಾಗಿರಿ.
