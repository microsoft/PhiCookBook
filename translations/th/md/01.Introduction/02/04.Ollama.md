## ตระกูล Phi ใน Ollama


[Ollama](https://ollama.com) ช่วยให้คนจำนวนมากสามารถปรับใช้ LLM หรือ SLM แบบโอเพนซอร์สโดยตรงผ่านสคริปต์ง่ายๆ และยังสามารถสร้าง APIs เพื่อช่วยกรณีการใช้งาน Copilot ท้องถิ่นได้ด้วย

## **1. การติดตั้ง**

Ollama รองรับการรันบน Windows, macOS, และ Linux คุณสามารถติดตั้ง Ollama ผ่านลิงก์นี้ ([https://ollama.com/download](https://ollama.com/download)) หลังการติดตั้งเสร็จสิ้น คุณสามารถใช้สคริปต์ Ollama เรียก Phi-3 โดยตรงผ่านหน้าต่างเทอร์มินัลได้ คุณสามารถดู [ไลบรารีที่มีอยู่ใน Ollama](https://ollama.com/library) ทั้งหมดได้ หากคุณเปิดที่เก็บนี้ใน Codespace จะมี Ollama ติดตั้งไว้แล้ว

```bash

ollama run phi4

```

> [!NOTE]
> โมเดลจะถูกดาวน์โหลดก่อนเมื่อคุณรันครั้งแรก แน่นอนว่าคุณยังสามารถระบุโมเดล Phi-4 ที่ดาวน์โหลดไว้แล้วโดยตรงได้ เราจะยกตัวอย่างการใช้ WSL ในการรันคำสั่ง หลังจากโมเดลดาวน์โหลดเสร็จเรียบร้อยแล้ว คุณสามารถโต้ตอบได้โดยตรงในเทอร์มินัล

![การรัน](../../../../../translated_images/th/ollama_run.e9755172b162b381.png)

## **2. เรียกใช้ phi-4 API จาก Ollama**

หากคุณต้องการเรียกใช้ Phi-4 API ที่สร้างโดย Ollama คุณสามารถใช้คำสั่งนี้ในเทอร์มินัลเพื่อเริ่มเซิร์ฟเวอร์ Ollama

```bash

ollama serve

```

> [!NOTE]
> หากรันบน macOS หรือ Linux โปรดทราบว่าคุณอาจพบข้อผิดพลาดต่อไปนี้ **"Error: listen tcp 127.0.0.1:11434: bind: address already in use"** ข้อผิดพลาดนี้อาจเกิดขึ้นเมื่อเรียกใช้คำสั่ง คุณสามารถละเลยข้อผิดพลาดนี้ได้ เนื่องจากโดยทั่วไปจะบ่งชี้ว่าเซิร์ฟเวอร์กำลังรันอยู่แล้ว หรือคุณสามารถหยุดแล้วเริ่ม Ollama ใหม่:

**macOS**

```bash

brew services restart ollama

```

**Linux**

```bash

sudo systemctl stop ollama

```

Ollama รองรับ API สองตัว: generate และ chat คุณสามารถเรียกใช้ API ของโมเดลที่ Ollama ให้มาได้ตามต้องการ โดยส่งคำขอไปยังบริการท้องถิ่นที่รันบนพอร์ต 11434

**Chat**

```bash

curl http://127.0.0.1:11434/api/chat -d '{
  "model": "phi3",
  "messages": [
    {
      "role": "system",
      "content": "Your are a python developer."
    },
    {
      "role": "user",
      "content": "Help me generate a bubble algorithm"
    }
  ],
  "stream": false
  
}'
```

นี่คือผลลัพธ์ใน Postman

## ทรัพยากรเพิ่มเติม

ตรวจสอบรายการโมเดลที่มีอยู่ใน Ollama ใน [ไลบรารีของพวกเขา](https://ollama.com/library)

ดึงโมเดลของคุณจากเซิร์ฟเวอร์ Ollama โดยใช้คำสั่งนี้

```bash
ollama pull phi4
```

รันโมเดลโดยใช้คำสั่งนี้

```bash
ollama run phi4
```

***หมายเหตุ:*** เยี่ยมชมลิงก์นี้ [https://github.com/ollama/ollama/blob/main/docs/api.md](https://github.com/ollama/ollama/blob/main/docs/api.md) เพื่อเรียนรู้เพิ่มเติม

## เรียกใช้ Ollama จาก Python

คุณสามารถใช้ `requests` หรือ `urllib3` เพื่อส่งคำขอไปยังเอ็นด์พอยต์ของเซิร์ฟเวอร์ท้องถิ่นที่ใช้ข้างต้น อย่างไรก็ตาม วิธีที่นิยมใช้ Ollama ใน Python คือผ่าน SDK [openai](https://pypi.org/project/openai/) เนื่องจาก Ollama ให้เอ็นด์พอยต์ที่เข้ากันได้กับ OpenAI ด้วย

นี่คือตัวอย่างสำหรับ phi3-mini:

```python
import openai

client = openai.OpenAI(
    base_url="http://localhost:11434/v1",
    api_key="nokeyneeded",
)

response = client.chat.completions.create(
    model="phi4",
    temperature=0.7,
    n=1,
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Write a haiku about a hungry cat"},
    ],
)

print("Response:")
print(response.choices[0].message.content)
```

## เรียกใช้ Ollama จาก JavaScript 

```javascript
// ตัวอย่างการสรุปไฟล์ด้วย Phi-4
script({
    model: "ollama:phi4",
    title: "Summarize with Phi-4",
    system: ["system"],
})

// ตัวอย่างการสรุป
const file = def("FILE", env.files)
$`Summarize ${file} in a single paragraph.`
```

## เรียกใช้ Ollama จาก C#

สร้างแอป C# Console ใหม่และเพิ่มแพ็กเกจ NuGet ต่อไปนี้:

```bash
dotnet add package Microsoft.SemanticKernel --version 1.34.0
```

จากนั้นแทนที่โค้ดนี้ในไฟล์ `Program.cs`

```csharp
using Microsoft.SemanticKernel;
using Microsoft.SemanticKernel.ChatCompletion;

// add chat completion service using the local ollama server endpoint
#pragma warning disable SKEXP0001, SKEXP0003, SKEXP0010, SKEXP0011, SKEXP0050, SKEXP0052
builder.AddOpenAIChatCompletion(
    modelId: "phi4",
    endpoint: new Uri("http://localhost:11434/"),
    apiKey: "non required");

// invoke a simple prompt to the chat service
string prompt = "Write a joke about kittens";
var response = await kernel.InvokePromptAsync(prompt);
Console.WriteLine(response.GetValue<string>());
```

รันแอปด้วยคำสั่ง:

```bash
dotnet run
```
