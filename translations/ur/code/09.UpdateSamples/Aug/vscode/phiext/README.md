مثال کے طور پر:

یہ ایکسٹینشن درج ذیل سیٹنگز فراہم کرتی ہے:

* `myExtension.enable`: اس ایکسٹینشن کو فعال/غیر فعال کریں۔
* `myExtension.thing`: `blah` پر سیٹ کریں تاکہ کچھ کام کریں۔
