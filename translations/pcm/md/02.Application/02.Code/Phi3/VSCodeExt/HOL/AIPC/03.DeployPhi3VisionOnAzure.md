***Note***

1. The parameters wey you go send must include Authorization, azureml-model-deployment, and Content-Type. You go need check the deployment information to get them.

2. To send parameters, Phi-3-Vision go need an image link. Abeg refer to the GPT-4-Vision method to send parameters, for example

```json

{
  "input_data":{
    "input_string":[
      {
        "role":"user",
        "content":[ 
          {
            "type": "text",
            "text": "You are a Python coding assistant.Please create Python code for image "
          },
          {
              "type": "image_url",
              "image_url": {
                "url": "https://ajaytech.co/wp-content/uploads/2019/09/index.png"
              }
          }
        ]
      }
    ],
    "parameters":{
          "temperature": 0.6,
          "top_p": 0.9,
          "do_sample": false,
          "max_new_tokens": 2048
    }
  }
}

```

3. Call **/score** using the Post method
