# **Lab 0 - Instalashun**

When we enter the Lab, we need to configure the relevant environment :


### **1. Python 3.11+**

E dey recommended make you use miniforge to configure your Python environment 

To configure miniforge, abeg refer to [https://github.com/conda-forge/miniforge](https://github.com/conda-forge/miniforge)

After you don configure miniforge, run the following command in Power Shell

```bash

conda create -n pyenv python==3.11.8 -y

conda activate pyenv

```


### **2. Install Prompt flow SDK**

For Lab 1, we go use Prompt flow, so you need to configure di Prompt flow SDK.

```bash

pip install promptflow --upgrade

```

You fit check promptflow sdk with dis command


```bash

pf --version

```

### **3. Install Visual Studio Code Prompt flow Extension**

![pf](../../../../../../../../../translated_images/pf_ext.8cf76b5846e9b856.pcm.png)


### **4. Intel NPU Acceleration Library**

Intel new generation processors dey support NPU. If you wan use NPU to run LLMs / SLMs locally, you fit use ***Intel NPU Acceleration Library*** . If you want know more, abeg read [https://github.com/microsoft/PhiCookBook/blob/main/md/01.Introduction/03/AIPC_Inference.md](https://github.com/microsoft/PhiCookBook/blob/main/md/01.Introduction/03/AIPC_Inference.md).

Install Intel NPU Acceleration Library in bash


```bash

pip install intel-npu-acceleration-library

```

***Note***: Abeg note say dis library dey support transformers ***4.40.2***, abeg confirm di version


### **5. Other Python Library**


make requirements.txt and add dis content

```txt

notebook
numpy 
scipy 
scikit-learn 
matplotlib 
pandas 
pillow 
graphviz

```


### **6. Install NVM**

install nvm for Powershell 


```bash

winget install -e --id CoreyButler.NVMforWindows

```

install nodejs 18.20


```bash

nvm install 18.20.0

nvm use 18.20.0

```

### **7. Install Visual Studio Code Development Support**


```bash

npm install --global yo generator-code

```

Congratulations! You don successfully configure di SDK. Next, proceed to di hands-on steps.
