## Οικογένεια Phi στο Ollama


[Ollama](https://ollama.com) επιτρέπει σε περισσότερους ανθρώπους να αναπτύσσουν απευθείας ανοικτού κώδικα LLM ή SLM μέσω απλών scripts, και μπορεί επίσης να δημιουργήσει APIs για να βοηθήσει σενάρια τοπικών εφαρμογών Copilot.

## **1. Εγκατάσταση**

Το Ollama υποστηρίζει εκτέλεση σε Windows, macOS, και Linux. Μπορείτε να εγκαταστήσετε το Ollama μέσω αυτού του συνδέσμου ([https://ollama.com/download](https://ollama.com/download)). Μετά την επιτυχή εγκατάσταση, μπορείτε να χρησιμοποιήσετε απευθείας το script του Ollama για να καλέσετε το Phi-3 μέσω ενός παραθύρου τερματικού. Μπορείτε να δείτε όλες τις [διαθέσιμες βιβλιοθήκες στο Ollama](https://ollama.com/library). Εάν ανοίξετε αυτό το αποθετήριο σε ένα Codespace, θα έχει ήδη εγκατεστημένο το Ollama.

```bash

ollama run phi4

```

> [!NOTE]
> Το μοντέλο θα κατεβεί πρώτα όταν το εκτελέσετε για πρώτη φορά. Φυσικά, μπορείτε επίσης να καθορίσετε απευθείας το ήδη κατεβασμένο μοντέλο Phi-4. Παίρνουμε το WSL ως παράδειγμα για την εκτέλεση της εντολής. Αφού το μοντέλο κατέβει επιτυχώς, μπορείτε να αλληλεπιδράσετε απευθείας στο τερματικό.

![εκτέλεση](../../../../../translated_images/el/ollama_run.e9755172b162b381.png)

## **2. Κλήση του API phi-4 από το Ollama**

Εάν θέλετε να καλέσετε το API Phi-4 που δημιουργείται από το Ollama, μπορείτε να χρησιμοποιήσετε αυτήν την εντολή στο τερματικό για να ξεκινήσετε τον διακομιστή Ollama.

```bash

ollama serve

```

> [!NOTE]
> Εάν εκτελείτε MacOS ή Linux, σημειώστε ότι ενδέχεται να αντιμετωπίσετε το ακόλουθο σφάλμα **"Error: listen tcp 127.0.0.1:11434: bind: address already in use"**. Μπορεί να λάβετε αυτό το σφάλμα κατά την εκτέλεση της εντολής. Μπορείτε είτε να αγνοήσετε αυτό το σφάλμα, καθώς συνήθως υποδεικνύει ότι ο διακομιστής έχει ήδη ξεκινήσει, είτε να σταματήσετε και να επανεκκινήσετε το Ollama:

**macOS**

```bash

brew services restart ollama

```

**Linux**

```bash

sudo systemctl stop ollama

```

Το Ollama υποστηρίζει δύο API: generate και chat. Μπορείτε να καλέσετε το API του μοντέλου που παρέχει το Ollama σύμφωνα με τις ανάγκες σας, στέλνοντας αιτήματα στην τοπική υπηρεσία που τρέχει στην πόρτα 11434.

**Chat**

```bash

curl http://127.0.0.1:11434/api/chat -d '{
  "model": "phi3",
  "messages": [
    {
      "role": "system",
      "content": "Your are a python developer."
    },
    {
      "role": "user",
      "content": "Help me generate a bubble algorithm"
    }
  ],
  "stream": false
  
}'
```

Αυτό είναι το αποτέλεσμα στο Postman

![Στιγμιότυπο οθόνης των αποτελεσμάτων JSON για αίτημα generate](../../../../../translated_images/el/ollama_gen.bda5d4e715366cc9.png)

## Πρόσθετοι Πόροι

Δείτε τη λίστα των διαθέσιμων μοντέλων στο Ollama στη [βιβλιοθήκη τους](https://ollama.com/library).

Κατεβάστε το μοντέλο σας από τον διακομιστή Ollama χρησιμοποιώντας αυτήν την εντολή

```bash
ollama pull phi4
```

Εκτελέστε το μοντέλο χρησιμοποιώντας αυτήν την εντολή

```bash
ollama run phi4
```

***Σημείωση:*** Επισκεφθείτε αυτόν τον σύνδεσμο [https://github.com/ollama/ollama/blob/main/docs/api.md](https://github.com/ollama/ollama/blob/main/docs/api.md) για να μάθετε περισσότερα

## Κλήση του Ollama από Python

Μπορείτε να χρησιμοποιήσετε `requests` ή `urllib3` για να κάνετε αιτήματα στα τοπικά endpoints του διακομιστή που χρησιμοποιούνται παραπάνω. Ωστόσο, ένας δημοφιλής τρόπος χρήσης του Ollama σε Python είναι μέσω του SDK [openai](https://pypi.org/project/openai/), καθώς το Ollama παρέχει επίσης συμβατά με OpenAI endpoints διακομιστή.

Εδώ είναι ένα παράδειγμα για phi3-mini:

```python
import openai

client = openai.OpenAI(
    base_url="http://localhost:11434/v1",
    api_key="nokeyneeded",
)

response = client.chat.completions.create(
    model="phi4",
    temperature=0.7,
    n=1,
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Write a haiku about a hungry cat"},
    ],
)

print("Response:")
print(response.choices[0].message.content)
```

## Κλήση του Ollama από JavaScript 

```javascript
// Παράδειγμα σύνοψης αρχείου με το Phi-4
script({
    model: "ollama:phi4",
    title: "Summarize with Phi-4",
    system: ["system"],
})

// Παράδειγμα σύνοψης
const file = def("FILE", env.files)
$`Summarize ${file} in a single paragraph.`
```

## Κλήση του Ollama από C#

Δημιουργήστε μια νέα εφαρμογή C# Console και προσθέστε το ακόλουθο πακέτο NuGet:

```bash
dotnet add package Microsoft.SemanticKernel --version 1.34.0
```

Στη συνέχεια αντικαταστήστε αυτόν τον κώδικα στο αρχείο `Program.cs`

```csharp
using Microsoft.SemanticKernel;
using Microsoft.SemanticKernel.ChatCompletion;

// add chat completion service using the local ollama server endpoint
#pragma warning disable SKEXP0001, SKEXP0003, SKEXP0010, SKEXP0011, SKEXP0050, SKEXP0052
builder.AddOpenAIChatCompletion(
    modelId: "phi4",
    endpoint: new Uri("http://localhost:11434/"),
    apiKey: "non required");

// invoke a simple prompt to the chat service
string prompt = "Write a joke about kittens";
var response = await kernel.InvokePromptAsync(prompt);
Console.WriteLine(response.GetValue<string>());
```

Εκτελέστε την εφαρμογή με την εντολή:

```bash
dotnet run
```
