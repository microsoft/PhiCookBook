**Repository рђќрђйрђ▓рђирђЁрђірђ║рђИрђЎрђЙрђ»:**

### Local Setup

**рђюрђГрђ»рђАрђЋрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИ:**
- Python 3.12 рђърђГрђ»рђирђЎрђЪрђ»рђљрђ║ рђАрђЉрђђрђ║
- .NET 8.0 SDK (C# рђћрђЎрђ░рђћрђгрђЎрђ╗рђгрђИрђАрђљрђйрђђрђ║)
- Node.js 18+ рђћрђЙрђёрђирђ║ npm (JavaScript рђћрђЎрђ░рђћрђгрђЎрђ╗рђгрђИрђАрђљрђйрђђрђ║)
- рђАрђћрђірђ║рђИрђєрђ»рђХрђИ 16GB RAM рђАрђђрђ╝рђХрђЋрђ╝рђ»рђЉрђгрђИрђърђірђ║

**Installation:**
```bash
git clone https://github.com/microsoft/PhiCookBook.git
cd PhiCookBook
```

**Python рђћрђЎрђ░рђћрђгрђЎрђ╗рђгрђИрђАрђљрђйрђђрђ║:**
рђћрђЎрђ░рђћрђг directory рђАрђЉрђ░рђИрђърђГрђ»рђирђърђйрђгрђИрђЋрђ╝рђ«рђИ dependencies рђЎрђ╗рђгрђИрђђрђГрђ» install рђюрђ»рђЋрђ║рђЋрђФ:
```bash
cd code/<example-directory>
pip install -r requirements.txt  # if requirements.txt exists
```

**.NET рђћрђЎрђ░рђћрђгрђЎрђ╗рђгрђИрђАрђљрђйрђђрђ║:**
```bash
cd md/04.HOL/dotnet/src
dotnet restore LabsPhi.sln
dotnet build LabsPhi.sln
```

**JavaScript/Web рђћрђЎрђ░рђћрђгрђЎрђ╗рђгрђИрђАрђљрђйрђђрђ║:**

### Jupyter Notebooks рђђрђГрђ» run рђюрђ»рђЋрђ║рђЂрђ╝рђёрђ║рђИ

рђћрђЎрђ░рђћрђгрђЎрђ╗рђгрђИрђАрђЎрђ╗рђгрђИрђЁрђ»рђърђірђ║ Jupyter notebooks рђАрђќрђ╝рђЁрђ║рђЋрђ▒рђИрђЉрђгрђИрђърђірђ║:
```bash
pip install jupyter notebook
jupyter notebook  # Opens browser interface
# Navigate to desired .ipynb file
```

### Python Scripts рђђрђГрђ» run рђюрђ»рђЋрђ║рђЂрђ╝рђёрђ║рђИ

```bash

```



### JavaScript/Web рђћрђЎрђ░рђћрђгрђЎрђ╗рђгрђИрђђрђГрђ» run рђюрђ»рђЋрђ║рђЂрђ╝рђёрђ║рђИ

## рђЁрђЎрђ║рђИрђърђЋрђ║рђЂрђ╝рђёрђ║рђИ

рђц repository рђърђірђ║ рђћрђЎрђ░рђћрђг code рђћрђЙрђёрђирђ║ рђърђёрђ║рђЂрђћрђ║рђИрђЁрђгрђЎрђ╗рђгрђИрђЋрђФрђЮрђёрђ║рђЋрђ╝рђ«рђИ рђАрђЂрђ╝рђгрђИ software project рђЎрђ╗рђгрђИрђюрђГрђ» unit tests рђЎрђЋрђФрђЮрђёрђ║рђЋрђФрЂІ Validation рђърђірђ║ рђАрђ▒рђгрђђрђ║рђЋрђФрђћрђірђ║рђИрђюрђЎрђ║рђИрђЎрђ╗рђгрђИрђќрђ╝рђёрђирђ║ рђАрђЎрђ╗рђгрђИрђАрђгрђИрђќрђ╝рђёрђирђ║ рђЋрђ╝рђ»рђюрђ»рђЋрђ║рђърђірђ║:

1. **рђћрђЎрђ░рђћрђгрђЎрђ╗рђгрђИрђђрђГрђ» run рђюрђ»рђЋрђ║рђЂрђ╝рђёрђ║рђИ** - рђћрђЎрђ░рђћрђгрђљрђЁрђ║рђЂрђ»рђЁрђ«рђърђірђ║ error рђЎрђЏрђЙрђГрђўрђ▓ рђАрђюрђ»рђЋрђ║рђюрђ»рђЋрђ║рђърђёрђирђ║рђърђірђ║
2. **output рђЎрђ╗рђгрђИрђђрђГрђ» рђЁрђЁрђ║рђєрђ▒рђИрђЂрђ╝рђёрђ║рђИ** - рђЎрђ▒рђгрђ║рђњрђџрђ║рЂЈ рђАрђђрђ╝рђ▒рђгрђёрђ║рђИрђЋрђ╝рђ»рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИрђърђірђ║ рђърђёрђирђ║рђюрђ╗рђ▒рђгрђ║рђърђ▒рђгрђАрђќрђ╝рђ▒рђЎрђ╗рђгрђИрђќрђ╝рђЁрђ║рђЏрђЎрђірђ║
3. **рђърђёрђ║рђЂрђћрђ║рђИрђЁрђгрђЎрђ╗рђгрђИрђђрђГрђ» рђюрђГрђ»рђђрђ║рђћрђгрђЂрђ╝рђёрђ║рђИ** - рђюрђЎрђ║рђИрђірђйрђЙрђћрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИрђърђірђ║ documentation рђАрђљрђГрђ»рђёрђ║рђИ рђАрђюрђ»рђЋрђ║рђюрђ»рђЋрђ║рђърђёрђирђ║рђърђірђ║

**рђАрђЎрђ╗рђгрђИрђєрђ»рђХрђИ validation рђћрђірђ║рђИрђюрђЎрђ║рђИ:**
- Target рђЋрђљрђ║рђЮрђћрђ║рђИрђђрђ╗рђёрђ║рђљрђйрђёрђ║ рђћрђЎрђ░рђћрђгрђЎрђ╗рђгрђИрђђрђГрђ» рђЁрђЎрђ║рђИрђърђЋрђ║ run рђюрђ»рђЋрђ║рђЋрђФ
- Dependencies рђЎрђ╗рђгрђИрђђрђГрђ» рђЎрђЙрђћрђ║рђђрђћрђ║рђЁрђйрђг install рђюрђ»рђЋрђ║рђћрђГрђ»рђёрђ║рђђрђ╝рђ▒рђгрђёрђ║рђИ рђЁрђЁрђ║рђєрђ▒рђИрђЋрђФ
- рђЎрђ▒рђгрђ║рђњрђџрђ║рђЎрђ╗рђгрђИрђђрђГрђ» download/load рђюрђ»рђЋрђ║рђћрђГрђ»рђёрђ║рђђрђ╝рђ▒рђгрђёрђ║рђИ рђАрђљрђірђ║рђЋрђ╝рђ»рђЋрђФ
- Documentation рђћрђЙрђёрђирђ║ рђђрђГрђ»рђђрђ║рђірђ«рђърђ▒рђг рђАрђЋрђ╝рђ»рђАрђЎрђ░рђђрђГрђ» рђАрђљрђірђ║рђЋрђ╝рђ»рђЋрђФ

### Documentation Standards

**URL Formatting:**
- `[text](../../url)` format рђђрђГрђ» рђАрђърђ»рђХрђИрђЋрђ╝рђ»рђЋрђФрЂі рђАрђЋрђГрђ»рђћрђ▒рђЏрђгрђЎрђ╗рђгрђИрђЎрђЉрђірђирђ║рђЋрђФрђћрђЙрђёрђирђ║
- Relative links: `./` рђђрђГрђ» рђюрђђрђ║рђЏрђЙрђГ directory рђАрђљрђйрђђрђ║рЂі `../` рђђрђГрђ» parent рђАрђљрђйрђђрђ║ рђАрђърђ»рђХрђИрђЋрђ╝рђ»рђЋрђФ
- URL рђЎрђ╗рђгрђИрђљрђйрђёрђ║ рђћрђГрђ»рђёрђ║рђёрђХрђАрђЉрђ░рђИ locales рђЎрђЋрђФрђЮрђёрђ║рђърђёрђирђ║рђЋрђФ (рђЦрђЋрђЎрђг `/en-us/`, `/en/`)

**Images:**
- рђАрђгрђИрђюрђ»рђХрђИ images рђђрђГрђ» `/imgs/` directory рђљрђйрђёрђ║ рђърђГрђЎрђ║рђИрђєрђірђ║рђИрђЋрђФ

- рђЦрђЋрђЎрђг: `phi-3-architecture.png`

**Markdown Files:**
- `/code/` directory рђљрђйрђёрђ║рђЏрђЙрђГрђърђ▒рђг рђАрђюрђ»рђЋрђ║рђюрђ»рђЋрђ║рђћрђ▒рђърђ▒рђг рђћрђЎрђ░рђћрђгрђЎрђ╗рђгрђИрђђрђГрђ» рђЏрђірђ║рђірђйрђЙрђћрђ║рђИрђЋрђФ
- Documentation рђђрђГрђ» code рђЋрђ╝рђ▒рђгрђёрђ║рђИрђюрђ▓рђЎрђЙрђ»рђЎрђ╗рђгрђИрђћрђЙрђёрђирђ║ рђАрђірђ«рђЉрђгрђИрђЋрђФ
- README рђљрђйрђёрђ║ Jupyter notebook links рђђрђГрђ» ­ЪЊЊ emoji рђќрђ╝рђёрђирђ║ рђАрђЎрђЙрђљрђ║рђАрђърђгрђИрђЋрђ╝рђ»рђЋрђФ

### рђќрђГрђ»рђёрђ║рђќрђйрђ▓рђирђЁрђірђ║рђИрђЎрђЙрђ»

## Pull Request рђюрђЎрђ║рђИрђірђйрђЙрђћрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИ

### Submit рђюрђ»рђЋрђ║рђЎрђірђ║рђирђАрђЂрђФ

1. **Repository рђђрђГрђ» Fork рђюрђ»рђЋрђ║рђЋрђФ** рђърђёрђирђ║рђАрђђрђ▒рђгрђёрђирђ║рђърђГрђ»рђи
2. **PR рђЎрђ╗рђгрђИрђђрђГрђ» рђАрђЎрђ╗рђГрђ»рђИрђАрђЁрђгрђИрђАрђюрђГрђ»рђђрђ║ рђЂрђйрђ▓рђЂрђ╝рђгрђИрђЋрђФ:**
   - Bug fixes рђђрђГрђ» PR рђљрђЁрђ║рђЂрђ»рђљрђйрђёрђ║
   - Documentation update рђЎрђ╗рђгрђИрђђрђГрђ» рђљрђЁрђ║рђЂрђ»рђЂрђ╝рђгрђИ PR рђљрђйрђёрђ║
   - рђћрђЎрђ░рђћрђгрђАрђърђЁрђ║рђЎрђ╗рђгрђИрђђрђГрђ» рђЂрђйрђ▓рђЂрђ╝рђгрђИ PR рђљрђйрђёрђ║
   - Typo fixes рђЎрђ╗рђгрђИрђђрђГрђ» рђЋрђ▒рђФрђёрђ║рђИрђЁрђЋрђ║рђћрђГрђ»рђёрђ║рђърђірђ║

3. **Merge conflicts рђђрђГрђ» рђќрђ╝рђ▒рђЏрђЙрђёрђ║рђИрђЋрђФ:**
   - рђЋрђ╝рђ»рђЋрђ╝рђёрђ║рђЎрђЙрђ»рђЎрђ╗рђгрђИрђЋрђ╝рђ»рђюрђ»рђЋрђ║рђЎрђ« рђърђёрђирђ║ local `main` branch рђђрђГрђ» update рђюрђ»рђЋрђ║рђЋрђФ
   - Upstream рђћрђЙрђёрђирђ║ рђЎрђђрђ╝рђгрђЂрђЈ sync рђюрђ»рђЋрђ║рђЋрђФ

4. **Translation PR рђЎрђ╗рђгрђИ:**
   - Folder рђАрђљрђйрђёрђ║рђИрђЏрђЙрђГ рђќрђГрђ»рђёрђ║рђАрђгрђИрђюрђ»рђХрђИрђАрђљрђйрђђрђ║ рђўрђгрђърђгрђЋрђ╝рђћрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИ рђЋрђФрђЮрђёрђ║рђЏрђЎрђірђ║
   - рђЎрђ░рђЏрђёрђ║рђИрђўрђгрђърђгрђЁрђђрђгрђИрђћрђЙрђёрђирђ║ рђќрђйрђ▓рђирђЁрђірђ║рђИрђЎрђЙрђ»рђђрђГрђ» рђљрђ░рђірђ«рђЁрђйрђг рђЉрђГрђћрђ║рђИрђърђГрђЎрђ║рђИрђЋрђФ

### рђюрђГрђ»рђАрђЋрђ║рђърђ▒рђг рђЁрђЁрђ║рђєрђ▒рђИрђЎрђЙрђ»рђЎрђ╗рђгрђИ

PR рђЎрђ╗рђгрђИрђърђірђ║ GitHub workflows рђђрђГрђ» рђАрђюрђГрђ»рђАрђюрђ╗рђ▒рђгрђђрђ║ run рђюрђ»рђЋрђ║рђЋрђ╝рђ«рђИ рђАрђ▒рђгрђђрђ║рђЋрђФрђАрђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИрђђрђГрђ» validate рђюрђ»рђЋрђ║рђърђірђ║:

1. **Relative path validation** - рђАрђљрђйрђёрђ║рђИ link рђЎрђ╗рђгрђИрђАрђгрђИрђюрђ»рђХрђИ рђАрђюрђ»рђЋрђ║рђюрђ»рђЋрђ║рђЏрђЎрђірђ║
   - VS Code рђљрђйрђёрђ║ Ctrl+Click рђќрђ╝рђёрђирђ║ link рђЎрђ╗рђгрђИрђђрђГрђ» locally рђЁрђЎрђ║рђИрђърђЋрђ║рђЋрђФ
   - VS Code рђЎрђЙ path рђАрђђрђ╝рђХрђЋрђ╝рђ»рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИрђђрђГрђ» рђАрђърђ»рђХрђИрђЋрђ╝рђ»рђЋрђФ (`./` рђърђГрђ»рђирђЎрђЪрђ»рђљрђ║ `../`)

2. **URL locale check** - Web URLs рђљрђйрђёрђ║ рђћрђГрђ»рђёрђ║рђёрђХ locales рђЎрђЋрђФрђЮрђёрђ║рђЏрђЎрђірђ║
   - `/en-us/`, `/en/` рђърђГрђ»рђирђЎрђЪрђ»рђљрђ║ рђАрђЂрђ╝рђгрђИрђўрђгрђърђгрђЁрђђрђгрђИ code рђЎрђ╗рђгрђИрђђрђГрђ» рђќрђџрђ║рђЏрђЙрђгрђИрђЋрђФ
   - рђАрђЋрђ╝рђірђ║рђЋрђ╝рђірђ║рђєрђГрђ»рђёрђ║рђЏрђг URLs рђђрђГрђ» рђАрђърђ»рђХрђИрђЋрђ╝рђ»рђЋрђФ

3. **Broken URL check** - URLs рђАрђгрђИрђюрђ»рђХрђИрђърђірђ║ 200 status рђђрђГрђ» return рђЋрђ╝рђ»рђЏрђЎрђірђ║
   - Submit рђЎрђюрђ»рђЋрђ║рђЎрђ« link рђЎрђ╗рђгрђИрђђрђГрђ» access рђюрђ»рђЋрђ║рђћрђГрђ»рђёрђ║рђђрђ╝рђ▒рђгрђёрђ║рђИ рђЁрђЁрђ║рђєрђ▒рђИрђЋрђФ
   - рђЎрђЙрђљрђ║рђЂрђ╗рђђрђ║: рђАрђЂрђ╗рђГрђ»рђирђърђ▒рђг failure рђЎрђ╗рђгрђИрђърђірђ║ network рђђрђћрђирђ║рђърђљрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИрђђрђ╝рђ▒рђгрђёрђирђ║ рђќрђ╝рђЁрђ║рђћрђГрђ»рђёрђ║рђърђірђ║
