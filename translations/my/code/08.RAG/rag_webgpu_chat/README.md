Phi-3-mini WebGPU RAG Chatbot

## WebGPU နှင့် RAG ပုံစံကို ပြသရန် ဒေမို
Phi-3 Onnx Hosted မော်ဒယ်နှင့် တွဲဖက်အသုံးပြုသော RAG ပုံစံသည် Retrieval-Augmented Generation နည်းလမ်းကို အသုံးပြုကာ Phi-3 မော်ဒယ်များ၏ စွမ်းအားနှင့် ONNX hosting ကို ပေါင်းစပ်၍ ထိရောက်သော AI တပ်ဆင်မှုများအတွက် အသုံးပြုသည်။ ဤပုံစံသည် အထူးကဏ္ဍဆိုင်ရာ တာဝန်များအတွက် မော်ဒယ်များကို အတိအကျ ပြင်ဆင်ရန် အထောက်အကူဖြစ်ပြီး အရည်အသွေး၊ စျေးနှုန်းသက်သာမှုနှင့် ရေရှည်အကြောင်းအရာ နားလည်မှုတို့ကို ပေါင်းစပ်ပေးသည်။ Azure AI ၏ စနစ်တစ်ခုအဖြစ် မော်ဒယ်များစွာကို ရှာဖွေ၊ စမ်းသပ်၊ အသုံးပြုရ လွယ်ကူစေပြီး စက်မှုလုပ်ငန်းအမျိုးမျိုး၏ စိတ်ကြိုက်ပြင်ဆင်မှုလိုအပ်ချက်များကို ဖြည့်ဆည်းပေးသည်။ Phi-3 မော်ဒယ်များမှာ Phi-3-mini, Phi-3-small, Phi-3-medium အပါအဝင် Azure AI Model Catalog တွင် ရရှိနိုင်ပြီး ကိုယ်တိုင် စီမံခန့်ခွဲ၍ fine-tune ပြုလုပ်နိုင်သလို HuggingFace နှင့် ONNX ကဲ့သို့သော ပလက်ဖောင်းများမှတဆင့်လည်း တပ်ဆင်နိုင်ပြီး Microsoft ၏ လွယ်ကူ၍ ထိရောက်သော AI ဖြေရှင်းချက်များပေးရန် အာရုံစိုက်မှုကို ပြသသည်။

## WebGPU ဆိုတာဘာလဲ
WebGPU သည် ဝဘ်ဘရောက်ဇာများမှ တိုက်ရိုက် စက်ပစ္စည်း၏ ဂရပ်ဖစ်ဆိုင်ရာ ပရိုဆက်ဆာ (GPU) ကို ထိရောက်စွာ အသုံးပြုနိုင်ရန် ရည်ရွယ်ထားသော ခေတ်မီ ဝဘ်ဂရပ်ဖစ် API ဖြစ်သည်။ WebGL ၏ နောက်ဆက်တွဲအဖြစ် ရည်ရွယ်ထားပြီး အောက်ပါ အချက်များဖြင့် တိုးတက်မှုများ ရရှိထားသည်-

1. **ခေတ်မီ GPU များနှင့် ကိုက်ညီမှု**: WebGPU သည် Vulkan, Metal, Direct3D 12 ကဲ့သို့သော စနစ် API များကို အသုံးပြုကာ ခေတ်မီ GPU ဖွဲ့စည်းမှုများနှင့် အဆင်ပြေစွာ လုပ်ဆောင်နိုင်သည်။
2. **စွမ်းဆောင်ရည် တိုးတက်မှု**: အထွေထွေ GPU တွင် တွက်ချက်မှုများနှင့် လျင်မြန်သော လုပ်ဆောင်ချက်များကို ထောက်ပံ့ကာ ဂရပ်ဖစ် ပုံဆွဲခြင်းနှင့် စက်မှုသင်ယူမှု တို့အတွက် သင့်တော်သည်။
3. **အဆင့်မြင့် လုပ်ဆောင်ချက်များ**: WebGPU သည် ပိုမိုရှုပ်ထွေးပြီး ဒိုင်နမစ်သော ဂရပ်ဖစ်နှင့် တွက်ချက်မှု လုပ်ငန်းများကို လုပ်ဆောင်နိုင်စေသည်။
4. **JavaScript အလုပ်ပမာဏ လျော့နည်းခြင်း**: GPU သို့ ပိုမိုတာဝန်များကို လွှဲပြောင်းပေးခြင်းဖြင့် JavaScript အလုပ်ပမာဏကို လျော့နည်းစေကာ စွမ်းဆောင်ရည်ကောင်းမွန်ပြီး ချောမွေ့သော အသုံးပြုမှုကို ပေးစွမ်းသည်။

WebGPU ကို လက်ရှိတွင် Google Chrome ကဲ့သို့သော ဘရောက်ဇာများတွင် ထောက်ပံ့ထားပြီး အခြားပလက်ဖောင်းများသို့ ထောက်ပံ့မှု တိုးချဲ့ရန် အလုပ်များ ဆက်လက်လုပ်ဆောင်နေသည်။

**ထောက်ပံ့သော ဘရောက်ဇာများ:**

### WebGPU ကို ဖွင့်ရန်-

- Chrome/Microsoft Edge တွင်

`chrome://flags/#enable-unsafe-webgpu` flag ကို ဖွင့်ပါ။

#### ဘရောက်ဇာကို ဖွင့်ပါ
Google Chrome သို့မဟုတ် Microsoft Edge ကို စတင်ဖွင့်ပါ။

#### Flags စာမျက်နှာသို့ ဝင်ရောက်ပါ
လိပ်စာဘားတွင် `chrome://flags` ဟု ရိုက်ထည့်ပြီး Enter နှိပ်ပါ။

#### Flag ကို ရှာဖွေပါ
စာမျက်နှာအပေါ်ဆုံး ရှာဖွေမှုဘောက်စ်တွင် 'enable-unsafe-webgpu' ဟု ရိုက်ထည့်ပါ။

#### Flag ကို ဖွင့်ပါ
ရလဒ်စာရင်းမှ #enable-unsafe-webgpu flag ကို ရှာဖွေပါ။

အနားရှိ dropdown မီနူးကို နှိပ်ပြီး Enabled ကို ရွေးချယ်ပါ။

#### ဘရောက်ဇာကို ပြန်စတင်ပါ
Flag ဖွင့်ပြီးနောက် ပြောင်းလဲမှုများ အကျိုးသက်ရောက်ရန် ဘရောက်ဇာကို ပြန်စတင်ဖွင့်ရမည်။ စာမျက်နှာအောက်ခြေတွင် ပေါ်လာသော Relaunch ခလုတ်ကို နှိပ်ပါ။

- Linux အတွက် `--enable-features=Vulkan` ဖြင့် ဘရောက်ဇာကို စတင်ပါ။
- Safari 18 (macOS 15) တွင် WebGPU ကို ပုံမှန်အားဖြင့် ဖွင့်ထားသည်။
- Firefox Nightly တွင် address bar တွင် about:config ဟု ရိုက်ထည့်ပြီး `dom.webgpu.enabled` ကို true သို့ ပြောင်းပါ။

### Microsoft Edge အတွက် GPU စနစ် ပြင်ဆင်ခြင်း

Windows တွင် Microsoft Edge အတွက် အမြင့်စွမ်းဆောင်ရည် GPU ကို ပြင်ဆင်ရန် အဆင့်များ-

- **Settings ကို ဖွင့်ပါ:** Start မီနူးကို နှိပ်ပြီး Settings ကို ရွေးချယ်ပါ။
- **System Settings:** System သို့ သွားပြီး Display ကို ရွေးပါ။
- **Graphics Settings:** အောက်သို့ ဆွဲချပြီး Graphics settings ကို နှိပ်ပါ။
- **App ရွေးချယ်ခြင်း:** “Choose an app to set preference” အောက်တွင် Desktop app ကို ရွေးပြီး Browse ကို နှိပ်ပါ။
- **Edge ကို ရွေးချယ်ပါ:** Edge တပ်ဆင်ထားသော ဖိုလ်ဒါ (ပုံမှန်အားဖြင့် `C:\Program Files (x86)\Microsoft\Edge\Application`) သို့ သွားပြီး `msedge.exe` ကို ရွေးပါ။
- **စိတ်ကြိုက် သတ်မှတ်ချက်:** Options ကို နှိပ်ပြီး High performance ကို ရွေးချယ်ပြီး Save ကို နှိပ်ပါ။

ဤအဆင့်များဖြင့် Microsoft Edge သည် သင့်အမြင့်စွမ်းဆောင်ရည် GPU ကို အသုံးပြုမည်ဖြစ်ပြီး စွမ်းဆောင်ရည်ကောင်းမွန်စေပါသည်။  
- ပြောင်းလဲမှုများ အကျိုးသက်ရောက်ရန် မိမိကွန်ပျူတာကို ပြန်စတင်ပါ။
