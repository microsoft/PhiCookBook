# ပြောင်းလဲမှုမှတ်တမ်း

ဒီဖိုင်ကို ဘယ်လိုရေးရမလဲဆိုတာအတွက် [Keep a Changelog](http://keepachangelog.com/) ကို ကြည့်ရှုနိုင်ပါတယ်။
