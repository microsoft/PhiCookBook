## הגדרות התוסף

כלול אם התוסף שלך מוסיף הגדרות VS Code דרך נקודת התוסף `contributes.configuration`.

לדוגמה:

תוסף זה תורם את ההגדרות הבאות:

* `myExtension.enable`: הפעל/השבת את התוסף.
* `myExtension.thing`: הגדר ל-`blah` כדי לבצע פעולה מסוימת.
