## **使用 Phi-4-mini-mm 生成代码**

Phi-4-mini 延续了 Phi 家族强大的编码能力。你可以通过 Prompt 提出与编码相关的问题。当然，在加入强大的推理能力后，它具备了更强的编码能力，比如根据需求生成项目。例如，根据需求生成项目，如：

### **需求**

创建一个购物车应用

- 创建一个带有以下方法的 API Rest：
    - 使用页码偏移和限制获取啤酒列表。
    - 通过 id 获取啤酒详情。
    - 根据名称、描述、标语、食物搭配和价格搜索啤酒。
- 在主页创建产品列表。
    - 创建搜索栏以筛选产品。
    - 当用户点击产品时，导航到描述页面。
- （可选）使用滑块按价格筛选产品。
- 创建购物车。
    - 添加产品到购物车。
    - 从购物车中移除产品。
    - 计算购物车中产品的总价。

    model_path,

    torch_dtype='auto',
    _attn_implementation='flash_attention_2',

                    ｜- repositories

                        | - index.html

               ## Backend
                 
                   #### `models/user.py`
                   ```python

                   ```
                   .......
               

               ## Frontend
                 
                   #### `templates/index.html`
                   ```html

                   ```
                   ......."""

prompt = f'{user_prompt}Please create a project with Python and Flask according to the following requirements：\n{requirement}{note}{prompt_suffix}{assistant_prompt}'
