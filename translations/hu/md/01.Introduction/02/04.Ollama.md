## Phi család az Ollamában


[Ollama](https://ollama.com) lehetővé teszi, hogy több ember egyszerű scriptekkel közvetlenül telepítsen nyílt forráskódú LLM- vagy SLM-modelleket, és API-kat is építhet helyi Copilot-alkalmazási forgatókönyvekhez.

## **1. Telepítés**

Ollama támogatja a futtatást Windows-on, macOS-en és Linux-on. Telepítheti az Ollamát ezen a linken ([https://ollama.com/download](https://ollama.com/download)). A sikeres telepítés után közvetlenül használhatja az Ollama parancssorát, hogy Phi-3-at hívjon meg a terminálablakon keresztül. Megtekintheti az [az Ollamában elérhető könyvtárakat](https://ollama.com/library). Ha megnyitja ezt a tárolót egy Codespace-ben, az már tartalmazza az Ollamát.

```bash

ollama run phi4

```

> [!NOTE]
> A modell először lesz letöltve, amikor először futtatja. Természetesen közvetlenül megadhatja a letöltött Phi-4 modellt is. A parancs futtatásához példaként a WSL-t vesszük. Miután a modell sikeresen letöltődött, közvetlenül a terminálon tud vele interakcióba lépni.

![futtatás](../../../../../translated_images/ollama_run.e9755172b162b381.hu.png)

## **2. A phi-4 API meghívása az Ollamából**

Ha meg szeretné hívni az Ollama által létrehozott Phi-4 API-t, a terminálban ezt a parancsot használhatja az Ollama szerver indításához.

```bash

ollama serve

```

> [!NOTE]
> Ha macOS-t vagy Linuxot futtat, kérjük, vegye figyelembe, hogy előfordulhat a következő hiba **"Error: listen tcp 127.0.0.1:11434: bind: address already in use"**. Ezt a hibát megkaphatja a parancs futtatásakor. Ezt a hibát figyelmen kívül hagyhatja, mivel rendszerint azt jelzi, hogy a szerver már fut, vagy leállíthatja, majd újraindíthatja az Ollamát:

**macOS**

```bash

brew services restart ollama

```

**Linux**

```bash

sudo systemctl stop ollama

```

Ollama supports two API: generate and chat. A saját igényeinek megfelelően hívhatja meg az Ollama által biztosított modell-API-t úgy, hogy kéréseket küld a 11434-es porton futó helyi szolgáltatásnak.

**Chat**

```bash

curl http://127.0.0.1:11434/api/chat -d '{
  "model": "phi3",
  "messages": [
    {
      "role": "system",
      "content": "Your are a python developer."
    },
    {
      "role": "user",
      "content": "Help me generate a bubble algorithm"
    }
  ],
  "stream": false
  
}'
```

## További források

Tekintse meg az Ollamában elérhető modellek listáját a [könyvtárukban](https://ollama.com/library).

Húzza le a modelljét az Ollama szerveréről ezzel a paranccsal

```bash
ollama pull phi4
```

Futtassa a modellt ezzel a paranccsal

```bash
ollama run phi4
```

***Megjegyzés:*** Látogassa meg ezt a linket [https://github.com/ollama/ollama/blob/main/docs/api.md](https://github.com/ollama/ollama/blob/main/docs/api.md) a további információkért

## Ollama hívása Pythonból

A fenti helyi szerver-végpontokhoz kéréseket a `requests` vagy a `urllib3` segítségével küldheti. Azonban népszerű módja az Ollama Pythonból történő használatának az [openai](https://pypi.org/project/openai/) SDK használata, mivel az Ollama OpenAI-kompatibilis szervervégpontokat is biztosít.

Itt egy példa a phi3-mini-re:

```python
import openai

client = openai.OpenAI(
    base_url="http://localhost:11434/v1",
    api_key="nokeyneeded",
)

response = client.chat.completions.create(
    model="phi4",
    temperature=0.7,
    n=1,
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Write a haiku about a hungry cat"},
    ],
)

print("Response:")
print(response.choices[0].message.content)
```

## Ollama hívása JavaScriptből 

```javascript
// Példa fájl összefoglalására Phi-4-gyel
script({
    model: "ollama:phi4",
    title: "Summarize with Phi-4",
    system: ["system"],
})

// Példa az összefoglalásra
const file = def("FILE", env.files)
$`Summarize ${file} in a single paragraph.`
```

## Ollama hívása C#-ból

Hozzon létre egy új C# Console alkalmazást, és adja hozzá a következő NuGet csomagot:

```bash
dotnet add package Microsoft.SemanticKernel --version 1.34.0
```

Ezután cserélje le a `Program.cs` fájlban található kódot erre:

```csharp
using Microsoft.SemanticKernel;
using Microsoft.SemanticKernel.ChatCompletion;

// add chat completion service using the local ollama server endpoint
#pragma warning disable SKEXP0001, SKEXP0003, SKEXP0010, SKEXP0011, SKEXP0050, SKEXP0052
builder.AddOpenAIChatCompletion(
    modelId: "phi4",
    endpoint: new Uri("http://localhost:11434/"),
    apiKey: "non required");

// invoke a simple prompt to the chat service
string prompt = "Write a joke about kittens";
var response = await kernel.InvokePromptAsync(prompt);
Console.WriteLine(response.GetValue<string>());
```

Futtassa az alkalmazást ezzel a parancssal:

```bash
dotnet run
```
