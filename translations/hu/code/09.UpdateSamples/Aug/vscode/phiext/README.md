Például:

Ez a bővítmény a következő beállításokat járul hozzá:

* `myExtension.enable`: Engedélyezd vagy tiltsd le ezt a bővítményt.
* `myExtension.thing`: Állítsd `blah` értékre valami elvégzéséhez.
