## Famiglia Phi in Ollama


[Ollama](https://ollama.com) permette a più persone di distribuire direttamente LLM o SLM open source tramite semplici script e può anche creare API per supportare scenari applicativi Copilot locali.

## **1. Installazione**

Ollama supporta l'esecuzione su Windows, macOS e Linux. Puoi installare Ollama tramite questo link ([https://ollama.com/download](https://ollama.com/download)). Dopo l'installazione riuscita, puoi usare direttamente lo script di Ollama per chiamare Phi-3 tramite una finestra del terminale. Puoi vedere tutte le [librerie disponibili in Ollama](https://ollama.com/library). Se apri questo repository in un Codespace, avrà già Ollama installato.

```bash

ollama run phi4

```

> [!NOTE]
> Il modello verrà scaricato la prima volta che lo esegui. Naturalmente, puoi anche specificare direttamente il modello Phi-4 scaricato. Prendiamo WSL come esempio per eseguire il comando. Dopo che il modello è stato scaricato con successo, puoi interagire direttamente nel terminale.

![esecuzione](../../../../../translated_images/it/ollama_run.e9755172b162b381.webp)

## **2. Chiamare l'API phi-4 da Ollama**

Se vuoi chiamare l'API Phi-4 generata da ollama, puoi usare questo comando nel terminale per avviare il server Ollama.

```bash

ollama serve

```

> [!NOTE]
> Se esegui su macOS o Linux, tieni presente che potresti incontrare il seguente errore **"Error: listen tcp 127.0.0.1:11434: bind: address already in use"**. Potresti ricevere questo errore quando esegui il comando. Puoi ignorare quell'errore, poiché solitamente indica che il server è già in esecuzione, oppure puoi fermare e riavviare Ollama:

**macOS**

```bash

brew services restart ollama

```

**Linux**

```bash

sudo systemctl stop ollama

```

Ollama supporta due API: generate e chat. Puoi chiamare l'API del modello fornita da Ollama secondo le tue esigenze, inviando richieste al servizio locale in esecuzione sulla porta 11434.

**Chat**

```bash

curl http://127.0.0.1:11434/api/chat -d '{
  "model": "phi3",
  "messages": [
    {
      "role": "system",
      "content": "Your are a python developer."
    },
    {
      "role": "user",
      "content": "Help me generate a bubble algorithm"
    }
  ],
  "stream": false
  
}'
```

## Risorse aggiuntive

Consulta l'elenco dei modelli disponibili in Ollama nella [loro libreria](https://ollama.com/library).

Scarica il tuo modello dal server Ollama usando questo comando

```bash
ollama pull phi4
```

Esegui il modello usando questo comando

```bash
ollama run phi4
```

***Nota:*** Visita questo link [https://github.com/ollama/ollama/blob/main/docs/api.md](https://github.com/ollama/ollama/blob/main/docs/api.md) per saperne di più

## Chiamare Ollama da Python

Puoi usare `requests` o `urllib3` per effettuare richieste agli endpoint del server locale usati sopra. Tuttavia, un modo popolare per usare Ollama in Python è tramite l'SDK [openai](https://pypi.org/project/openai/), poiché Ollama fornisce anche endpoint server compatibili con OpenAI.

Ecco un esempio per phi3-mini:

```python
import openai

client = openai.OpenAI(
    base_url="http://localhost:11434/v1",
    api_key="nokeyneeded",
)

response = client.chat.completions.create(
    model="phi4",
    temperature=0.7,
    n=1,
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Write a haiku about a hungry cat"},
    ],
)

print("Response:")
print(response.choices[0].message.content)
```

## Chiamare Ollama da JavaScript 

```javascript
// Esempio di come riassumere un file con Phi-4
script({
    model: "ollama:phi4",
    title: "Summarize with Phi-4",
    system: ["system"],
})

// Esempio di come riassumere
const file = def("FILE", env.files)
$`Summarize ${file} in a single paragraph.`
```

## Chiamare Ollama da C#

Crea una nuova applicazione Console C# e aggiungi il seguente pacchetto NuGet:

```bash
dotnet add package Microsoft.SemanticKernel --version 1.34.0
```

Quindi sostituisci questo codice nel file `Program.cs`

```csharp
using Microsoft.SemanticKernel;
using Microsoft.SemanticKernel.ChatCompletion;

// add chat completion service using the local ollama server endpoint
#pragma warning disable SKEXP0001, SKEXP0003, SKEXP0010, SKEXP0011, SKEXP0050, SKEXP0052
builder.AddOpenAIChatCompletion(
    modelId: "phi4",
    endpoint: new Uri("http://localhost:11434/"),
    apiKey: "non required");

// invoke a simple prompt to the chat service
string prompt = "Write a joke about kittens";
var response = await kernel.InvokePromptAsync(prompt);
Console.WriteLine(response.GetValue<string>());
```

Esegui l'app con il comando:

```bash
dotnet run
```
